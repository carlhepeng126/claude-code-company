## Quick Start

1. **For DGW Canva graphics:**
   - Check QUICK-REFERENCE.md for HTML template starter
   - Use CSS color variables from brand palette
   - Follow format-specific size guidelines
   - Run `pnpm render --html ./design.html --format both`

2. **For general design work:**
   - Review color palette and accessibility requirements
   - Use Century Gothic typography exclusively
   - Follow layout and composition guidelines
   - Check quality checklist before publishing

3. **For brand decisions:**
   - Consult Design Decision Framework in SKILL.md
   - Verify WCAG AA compliance
   - Ensure dignity-centered approach
   - Test with colorblindness simulators
