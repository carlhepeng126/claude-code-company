# 团队项目交付 MECE 分类

## 默认一级工作流

| 工作流 | 边界 | 典型产出 |
|---|---|---|
| 项目治理 | 项目目标、节奏、会议、风险、变更 | 项目章程、里程碑表、周报、风险台账 |
| 教学设计 | 课程目标、课程大纲、教学活动、作业、评估 | 课程大纲、讲师手册、学员手册、作业说明 |
| 师资管理 | 讲师筛选、邀约、合同、备课、试讲 | 讲师名单、讲师合同、备课纪要、试讲反馈 |
| 学员管理 | 学员信息、分班、档案、通知、出勤、作业 | 学员档案、分组表、出勤表、作业追踪表 |
| 教学服务 | 开营、班级运营、现场服务、线上群、反馈 | 开营流程、服务 SOP、反馈日报 |
| 客户协同 | 客户资料、访谈、审批、客户侧任务 | 客户协助清单、访谈排期、确认纪要 |
| 商务合同 | 报价、合同、采购流程、变更、发票申请 | 合同文本、报价单、变更单 |
| 行政后勤 | 场地、物料、交通、住宿、餐饮、设备 | 物料清单、场地确认单、设备检查表 |
| 财务结算 | 预算、成本、付款、报销、回款 | 成本表、付款计划、发票/回款记录 |
| 质量评估 | 过程质量、满意度、学习效果、业务结果 | 评估问卷、复盘报告、效果报告 |
| 知识沉淀 | 模板、案例、复盘、可复用资产 | 项目复盘、案例卡、SOP 更新 |

## 阶段拆解

| 阶段 | 核心问题 | 退出标准 |
|---|---|---|
| 0. 移交启动 | 销售承诺、合同范围、方案版本是否清楚？ | 项目章程确认 |
| 1. 开营前 | 人、课、场、钱、客户配合是否到位？ | 开营检查清单全部通过 |
| 2. 交付中 | 每次课程和服务是否按标准执行？ | 每次交付有记录、有反馈、有改进 |
| 3. 阶段复盘 | 客户是否感知到进展？问题是否闭环？ | 阶段报告/客户会完成 |
| 4. 结项交付 | 合同交付物、财务、客户验收是否完成？ | 结项包和回款节点完成 |
| 5. 二次开发 | 是否沉淀案例和后续机会？ | 复盘卡和后续建议形成 |

## 各角色常见职责

| 角色 | 主要负责 |
|---|---|
| 项目负责人 | 目标、边界、节奏、风险、客户关键沟通 |
| 教学负责人 | 教学目标、课程质量、讲师融合、学习效果 |
| 教学服务 | 学员运营、通知、出勤、作业、现场/线上服务 |
| 商务 | 合同、报价、采购流程、变更、回款协同 |
| 行政 | 场地、物料、交通住宿、设备、会务 |
| 财务 | 预算、成本、发票、付款、报销、回款记录 |
| 讲师 | 课程内容、备课、授课、课后建议 |
| 客户 HR | 学员名单、内部通知、场地协调、客户侧审批 |
| 客户业务负责人 | 业务目标确认、学员上级支持、成果验收 |

## 常见工作包示例

### 开营前

- 项目章程确认
- 客户协助事项清单确认
- 学员档案建立
- 学员分班分组
- 讲师排期与备课
- 课程大纲定稿
- 学员手册制作
- 开营流程设计
- 场地/物料/设备确认
- 合同/付款节点确认

### 交付中

- 课前通知发送
- 现场签到与异常记录
- 课堂材料发放
- 作业布置与回收
- 学员反馈收集
- 讲师复盘记录
- 客户日报/周报发送
- 问题闭环跟踪

### 结项后

- 学员过程数据汇总
- 满意度和学习效果分析
- 项目结项报告
- 财务结算与回款
- 客户复盘会
- 二次需求识别
- Wiki 案例沉淀
