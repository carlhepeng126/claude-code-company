# 摩根国际客户痛点归纳 & 课程大纲合并方案

> 数据来源：D:\摩根客户（2024年项目为主），涵盖海工、COSL、HBP、中建土木、海油国际、中信等20+企业客户的培训方案、反馈报告、阶段报告等文档。

## 二、客户类型与痛点分布

### 类型一：海洋工程类（海工、COSL、博迈科、中海油）

**核心特征**：船员为主、工作环境特殊、考证+面试双目标

**痛点标签**：
- 英语水平普遍偏低（船员长期未用英语）
- 出海工作导致学习时间碎片化、网络不稳定
- 需要同时应对 Marlins 考试 + 船东面试两大挑战
- 专业航海英语词汇缺乏
- 岗位覆盖广（高级船员/一般船员目标分差10分）

**典型需求**：Marlins备考课程、船东面试口语、工程英语（海洋石油方向）、海事英语900句

---

### 类型二：能源工程类（HBP、中核、中建土木、中信建设）

**核心特征**：项目部外派人员为主、商务+技术双需求

**痛点标签**：
- 商务英语与专业技术英语双重需求
- 工作调动频繁，学习连续性差
- 词汇量薄弱、听力口语普遍不足
- 缺乏英语学习方法，自学效率低
- 线上学习自律性差，需班主任督促

**典型需求**：商务英语、技术英语（工程/采购方向）、1对1外教口语、学习管理系统（LMS）

---

### 类型三：企业高管/领导层（海油国际、能建国际）

**核心特征**：高层管理者、时间极稀缺、实用导向

**痛点标签**：
- 工作极忙，约课时间不稳定
- 期望快速看到学习效果
- 偏向海外生存/社交等实用场景
- 不愿被复杂系统束缚，追求简便
- 对教学内容私密性有要求

**典型需求**：海外商务社交、海外生存英语、1对1定制化、便捷约课系统

---

### 类型四：汽车/制造业（一汽、重庆长安、潍柴）

**核心特征**：市场营销/海外渠道人员、技术+商务融合

**痛点标签**：
- 需要跨文化沟通能力（对接海外经销商）
- 技术词汇与商务表达双重挑战
- 部分学员英语基础较好但缺乏实战
- 海外出差/展会场景需求明确
- 学习时间碎片化（工学矛盾）

**典型需求**：跨文化商务沟通、海外渠道英语、技术展会英语、产品推介英语

## 四、课程大纲设计建议（合并方案）

### 模块一：英语基础重塑（通用层）

```
【目标】解决85%学员B1-以下、词汇听力双弱的痛点
【对象】全类型客户，入学测评后分级教学

1. 英语学习方法论（4课时）
   - 听力训练方法（精听/泛听结合）
   - 词汇高效记忆法（词根词缀、场景记忆）
   - 自学规划与自律养成

2. 基础能力强化（32-56课时）
   - 听力：日常对话→场景听力→专业听力
   - 词汇：核心词汇→行业词汇→商务词汇
   - 语法：基础→进阶→实用
   - 阅读：技术文档阅读技巧

3. 英语学习工具包
   - APP + 在线课程 + 学习打卡
   - 无网环境学习材料（PDF/音频）
```

### 模块二：场景化专项能力（行业层）

```
【目标】解决"学了不会用"和"网络不稳定"的痛点
【按行业分轨】

A. 海洋工程轨（海工/COSL/中海油）
   - 海事英语900句（按船舶场景：消防/救生/装卸/航行）
   - Marlins备考专项（56小时陆地+50小时登船）
   - 船东面试口语冲刺（3期陆地+3期登船+1对1强化）
   - 石油工程船员英语（在线自学150节）

B. 能源工程轨（HBP/中核/中建土木/中信）
   - 商务英语（邮件/会议/谈判/演示）
   - 技术英语（工程图纸/采购/施工/项目管理）
   - 外派前英语强化（跨文化适应/海外安全）

C. 企业高管轨（海油国际/能建国际）
   - 海外商务社交英语（6个月1对1定制）
   - 海外生存英语（机场/海关/酒店/餐饮/商务活动）
   - 领导力英语（涉外发言/国际会议/演讲）

D. 汽车制造轨（一汽/潍柴/长安）
   - 跨文化商务沟通
   - 海外渠道英语（经销商管理/展会/产品推介）
   - 技术+商务双轨（专业词汇+商务表达）
```

### 模块三：认证/面试冲刺（目标层）

```
【目标】解决"高目标vs低起点"矛盾，确保成果交付

Marlins冲刺路径（海工专属）
  - 阶段1：陆地集中培训（7天×8小时=56小时）
  - 阶段2：登船/在线强化（50小时）
  - 阶段3：考前模拟（多次全真模拟）
  - 辅助：在线自学（42节）+ 学习打卡（15次）

船东面试冲刺路径（海工专属）
  - 陆地集训：3期×7天×8小时（56小时/期）
  - 登船强化：3期×14天×4小时（56小时/期）
  - 关键岗位：1对1重点培训（按需定制）
  - 全程模拟面试（多轮实战）
```

### 模块四：学习管理体系（服务层）

```
【目标】解决"学习动力不足、欠学严重"的痛点

1. 学分管理制度（100分制）
   - 主体课程（占70-80分）：每旷课扣2分
   - 在线自学（占15-20分）：每少1节扣1分
   - 学习打卡（占10-15分）：每少1次扣1分
   - 学分低于80分→考核不通过

2. 班主任服务机制
   - 学习进度实时监控
   - 欠学预警 + 电话/微信督促
   - 定期问卷调研（每周/每月）
   - 1对1学习情况反馈

3. 小组教学 + 因材施教
   - 班级内按水平分组（1对1/1对2/1对3/1对6）
   - 预习→学习→实践闭环
   - 无网环境替代学习方案
```

## 五、关键发现与建议

### 5.1 最需解决的三大痛点

1. **英语基础薄弱** → 课程前置能力测试 + 分级教学
2. **工学矛盾（工作忙、出海）** → 线上+线下混合，碎片化学习内容，无网环境适配
3. **学习动力与自律** → 班主任督促 + 学分化管理 + 实时反馈

### 5.2 最具潜力的课程方向

| 课程方向 | 市场机会 | 核心客户 |
|---------|---------|---------|
| Marlins考证+船东面试双轨 | 海洋工程出海刚需 | 海工、COSL |
| 技术显性化（解决方案设计+讲演） | 差异化竞争，利润高 | COSL、中核 |
| 企业高管1对1定制 | 客单价高，需求明确 | 海油国际、能建国际 |
| 汽车海外渠道英语 | 车企出海趋势强 | 一汽、长安、潍柴 |

### 5.3 课程迭代建议

1. **教材本地化**：针对海工开发海洋石油船员英语专属教材（含视频/音频）
2. **智能化学习路径**：根据学员测评结果自动推荐课程组合
3. **扩大Marlins替考服务**：文档已提及，可作为增值服务
4. **跨客户资源共享**：建立通用素材库，避免重复开发
