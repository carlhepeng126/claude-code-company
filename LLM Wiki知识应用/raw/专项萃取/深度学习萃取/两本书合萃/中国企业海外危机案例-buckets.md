# 中国企业海外危机传播案例研究报告 — 打标分桶素材索引

> 本文档为后续情报虾7大方向洞察提供素材索引，非正式洞察本身。
> 每条素材格式：原文核心内容（一句话概括）→ Carl业务映射（可落地使用方式）
>
> 素材来源：15个危机案例（2003-2016年），涵盖贸易保护主义、政党斗争、利益相关者冲突、国际市场评估不足四大主题。

## ① 企业外语培训

- **案例一 华为遭美封杀**：华为应对美国众议院调查时，需用英语向全球媒体沟通以扭转舆论，最终实现从负面到正面的报道转向。 → 可做课程案例：设计"危机英文沟通"课程——如何用英语向国际媒体/客户/合作伙伴传递企业立场，控制负面舆论。

- **案例二 三一状告奥巴马**：三一用法律武器维权，需要大量的英文法律文书、法庭陈述、媒体沟通。 → 可做课程案例：设计"国际商务法律英语"课程——涵盖英文合同解读、法律条款谈判、法务沟通场景。

- **案例三 国家电网澳大利亚受阻**：国家电网在澳大利亚竞购电力公司时，需要与澳方政府、媒体、社区进行英文沟通，解释并购意图和商业价值。 → 可做课程案例：设计"海外并购英语"课程——覆盖尽职调查英文报告、政府听证会演讲、媒体声明撰写等场景。

- **案例五 墨克高铁竞标**：中国铁建中标后遭墨西哥政府毁约，涉及复杂的英文竞标文件、合同条款、法律诉讼。 → 可做课程案例：设计"国际竞标商务英语"课程——包含英文技术标书编写、竞标演讲、合同谈判场景。

- **案例九 上汽双龙劳资冲突**：上汽与韩国双龙工会的语言和文化沟通障碍导致劳资矛盾升级。 → 可做课程案例：设计"跨文化劳资沟通英语"课程——如何用英语/当地语言与海外工会进行有效的谈判和日常沟通。

- **案例十 振华重工质量门**：振华重工面对美国媒体的质量质疑，需要用英文提供技术证据、回应媒体、维护声誉。 → 可做课程案例：设计"英文危机公关沟通"课程——涵盖英文新闻稿撰写、媒体问答、技术解释性沟通。

- **案例十五 阿里集体诉讼**：阿里巴巴在美遭遇集体诉讼，需使用英文应对SEC调查、律师事务所沟通、投资者关系管理。 → 可做课程案例：设计"上市公司英文投资者关系"课程——涵盖英文财报发布、投资者沟通、集体诉讼应对。

## ② 跨文化

- **案例四 菲律宾北铁项目**：项目因菲律宾政坛更迭（阿罗约→阿基诺三世）而终止，媒体态度180度转变，反映出东道国政治文化与中菲关系变化对项目的直接影响。 → Carl业务映射：用于跨文化政治风险培训——讲解如何识别和分析东道国的政治文化生态（政党轮替、精英博弈、中菲关系周期），避免因政治生态误判导致项目终止。

- **案例五 墨克高铁毁约**：墨西哥政府以"竞标程序不正当"为由毁约，反映出拉美国家独特的政治文化和决策风格。 → Carl业务映射：用于拉美跨文化商务培训——讲解拉美国家的商业文化特点（关系导向、政策随意性、政府干预频繁），帮助客户在拉美开展业务前做充分的文化风险评估。

- **案例六 中远希腊港口**：希腊新政府上任当天叫停中远收购，反映欧洲南部的政治文化特点和"新官不理旧账"的决策风格。 → Carl业务映射：用于欧洲跨文化商务培训——讲解希腊/南欧的商业文化和政治文化特点，以及在欧洲开展大型投资项目时的沟通和关系维护策略。

- **案例七 密松水电站**：缅甸总统叫停项目前，西方媒体大量负面报道聚焦环境破坏、移民安置、安全性，中方应对被动，形象受损严重。 → Carl业务映射：用于东南亚跨文化利益相关者管理培训——讲解在东南亚做项目时如何识别和沟通多元利益相关者（政府、NGO、社区、媒体、宗教团体）。

- **案例八 五矿秘鲁铜矿**：社区抗议导致项目停工，涉及资源民族主义、环保主义、人权保护主义等多元价值诉求冲突。 → Carl业务映射：用于拉美跨文化社区关系培训——讲解秘鲁/拉美社区的文化特点（集体主义、维权意识强、抗议文化），以及建立社区信任的沟通策略。

- **案例九 上汽双龙劳资冲突**：中韩文化差异（中国家长式管理 vs 韩国强势工会文化）导致劳资矛盾不可调和。 → Carl业务映射：用于中韩跨文化培训核心案例——对比中国"企业以老板为中心"和韩国"企业以工会为中心"的文化差异，讲解在韩投资做项目的文化适应策略。

- **案例十一 中铁建麦加轻轨**：在沙特阿拉伯伊斯兰文化环境中做项目，涉及宗教朝觐、本地化用工、合作伙伴关系管理等跨文化挑战。 → Carl业务映射：用于中东跨文化项目培训——讲解沙特/伊斯兰文化特点（宗教主导、关系导向、家族企业模式），帮助客户理解中东市场的独特商业文化。

## ③ 属地化

- **案例四 菲律宾北铁项目**：项目由中菲合资，涉及12年的长期运营，需应对当地政治变化、法律变更、贷款偿还等属地化运营问题。 → Carl业务映射：用于属地化长期运营培训——讲解在东南亚国家开展政府级基建项目时，如何建立属地化团队应对政治周期、法律变更等"慢变量"风险。

- **案例八 五矿秘鲁铜矿**：五矿在秘鲁的邦巴斯铜矿项目通过本地社区关系管理、环境责任承诺等方式建设属地化运营能力。 → Carl业务映射：用于属地化社区关系培训——讲解在拉美资源型国家开展业务的属地化运营策略（本地员工培养、社区投资、环境承诺、利益相关者沟通）。

- **案例九 上汽双龙劳资冲突**：上汽未能在韩国建立有效的本地化人力资源管理体系和工会沟通机制，最终失去控制权。 → Carl业务映射：用于属地化人力资源管理培训——以双龙案例为反面教材，讲解海外子公司属地化人力资源管理的关键要素（工会关系、本地法规、薪酬体系、文化适应）。

- **案例十 振华重工美国项目**：振华重工在美国建造海湾大桥钢结构，需满足美国的工程标准、安全规范、环保要求——属地化合规运营。 → Carl业务映射：用于属地化质量合规培训——讲解在美国/发达市场做项目的属地化合规运营策略（标准认证、质量监管、法律合规）。

- **案例十三 中海外波兰项目**：中海外用低价策略中标波兰高速公路，未充分评估波兰的原材料价格、劳动力成本、法律法规等属地化因素，导致巨亏。 → Carl业务映射：用于属地化市场评估培训——以波兰项目为反面教材，讲解进入东欧/欧盟市场前必须做的属地化调研（法律环境、供应链成本、劳工法规、税务政策）。

- **案例十四 五矿收购伊奎诺克斯**：五矿在加拿大市场的收购尝试因报价不足失败，反映了在成熟市场做属地化并购时需要充分了解当地市场规则和竞争格局。 → Carl业务映射：用于属地化并购策略培训——讲解在发达市场（加拿大/澳洲）做属地化收购时，如何评估当地市场规则、竞争环境、文化因素。

## ④ 涉外演讲谈判

- **案例一 华为应对美封杀**：华为通过海外媒体声明、高管采访、行业会议演讲等方式扭转舆论，从"国家安全威胁"到"公平竞争受害者"。 → Carl业务映射：用于涉外危机演讲培训——训练学员用英文做危机沟通演讲（关键声明、媒体应对、行业会议发言），掌握"从被动到主动"的叙事转换策略。

- **案例二 三一状告奥巴马**：三一需要与美方进行法律谈判、法庭陈述、媒体沟通，以及最终的和解谈判。 → Carl业务映射：用于涉外法律谈判培训——设计"国际法律纠纷谈判"模拟，涵盖证据交换谈判、庭外和解谈判、媒体声明策略。

- **案例十一 中铁建麦加轻轨**：中铁建在预亏41.53亿的情况下，需向沙特、国内公众、政府和媒体进行复杂的沟通和谈判。 → Carl业务映射：用于涉外项目谈判培训——设计"巨亏项目的多方谈判"模拟场景，涵盖与业主的合同谈判、与政府的政策沟通、与媒体的公关声明。

- **案例十二 中建巴哈马项目**：中建面临发展商索赔诉讼和资金链断裂，需要多方谈判（融资方、承包商、政府、法院）。 → Carl业务映射：用于涉外纠纷谈判培训——设计"海外项目烂尾纠纷谈判"模拟，涵盖索赔谈判、融资谈判、政府政策对话等场景。

- **案例十三 中海外波兰项目**：中海外面临业主2.71亿美元索赔，需进行跨文化商业谈判处理纠纷。 → Carl业务映射：用于涉外合同纠纷谈判培训——设计"欧美市场工程索赔谈判"模拟，讲解欧洲商业文化中的合同履约和纠纷处理机制。

- **案例十五 阿里集体诉讼**：阿里巴巴需要与美国律师事务所、投资者团体、SEC进行多轮沟通和谈判。 → Carl业务映射：用于涉外资本市场沟通培训——设计"美股上市公司投资者沟通"模拟，涵盖集体诉讼回应、投资者电话会、媒体声明。

## ⑤ 海外项目

- **案例一 华为遭美封杀**：华为在全球通信市场被美国以"国家安全"为由封杀，虽未直接影响项目运营但严重制约了北美市场拓展。 → Carl业务映射：用于海外项目政治风险培训——讲解通信/科技企业在海外市场面临的国家安全审查风险（CFIUS、DSD等），以及华为的应对策略。

- **案例四 菲律宾北铁项目**：中国援建菲律宾的铁路项目，因菲方政坛更迭被终止，12年投入付之东流。 → Carl业务映射：用于海外基建项目政治风险评估培训——讲解"政治周期风险"概念（大选年、政权更替对项目的影响），以及建立"政治保险"机制的方法。

- **案例五 墨克高铁项目**：中国铁建联合体中标后又遭毁约，是典型的海外基建项目"中标陷阱"。 → Carl业务映射：用于海外项目投标风险评估培训——设计"投标决策评估模型"课程，涵盖东道国政治稳定性评估、合同保护条款设计、毁约风险准备金。

- **案例六 中远希腊港口**：中远在希腊的港口股权投资遭新政府叫停，后通过积极沟通和政府支持得以缓解。 → Carl业务映射：用于海外股权投资项目的政治风险管理培训——讲解"政府更迭风险"的应对策略（与多方政党的关系建立、合同条款中的政府稳定承诺、外交渠道保护）。

- **案例七 密松水电站停建**：中电投在缅甸的水电站项目因环境/社会争议被叫停，是海外能源项目最大的"灰犀牛"事件之一。 → Carl业务映射：用于海外能源项目实施风险管理培训——讲解大坝/矿山/化工等敏感行业的"社会许可证"概念，以及利益相关者管理（NGO、社区、国际媒体）的实操方法。

- **案例十 振华重工海湾大桥**：振华重工为美国海湾大桥制造钢结构，涉及国际工程的严格质量标准和美国媒体的舆论压力。 → Carl业务映射：用于海外工程项目质量与声誉管理培训——讲解在发达国家做工程项目时的质量标准体系和媒体监测机制，提前预警"质量门"类危机。

- **案例十一 中铁建麦加轻轨**：中国铁建在沙特麦加的轻轨项目预亏41.53亿元，但最终成功完成运送朝觐者的任务。 → Carl业务映射：用于海外EPC项目风险管控培训——讲解海外大型EPC项目的典型风险（低价中标、设计变更、不可预见条件）和应对策略（合同索赔、变更管理、风险准备金）。

- **案例十二 中建巴哈马项目**：中建以"投资者+总承包商"双重角色加入巴哈马度假村项目，因资金链断裂陷入烂尾。 → Carl业务映射：用于海外项目投融资模式风险培训——讲解"EPC+F"（融资型总承包）模式的典型风险（融资风险、业主信用风险、资金链断裂），以及合同中的风险分担机制。

- **案例十三 中海外波兰项目**：中海外以不到预算一半的价格中标波兰高速公路，因原材料价格上涨、设计变更等亏损放弃，遭索赔2.71亿美元。 → Carl业务映射：用于海外项目低价中标风险培训——以波兰项目为经典反面教材，详细讲解低价竞标策略的"五大坑"（成本失控、合同陷阱、业主索赔、市场禁入、品牌受损）。

- **案例十四 五矿收购伊奎诺克斯**：五矿试图"蛇吞象"式收购加拿大矿业公司，因报价低于竞购者失败。 → Carl业务映射：用于海外并购项目培训——讲解海外并购的估值方法、竞标策略、尽职调查、文化整合。

## ⑥ 央企动态

- **案例三 国家电网澳大利亚受阻**：国家电网两次在澳大利亚的电力资产竞购被以"国家安全"为由拒绝，反映央企海外并购面临的贸易保护主义壁垒。 → Carl业务映射：用于央企海外投资合规培训——讲解央企在发达国家进行基础设施投资面临的特殊审查（国家安全审查、外商投资审查、国有企业歧视）。

- **案例四 菲律宾北铁项目**：中国援建菲律宾铁路项目（央企承担），因东道国政权更迭受阻。 → Carl业务映射：用于央企海外基建项目政治风险培训——讲解央企在"一带一路"沿线国家的基建项目面临的政治风险特点和应对策略（政府间协议、外交保护、项目保险）。

- **案例五 墨克高铁**：中国铁建在墨西哥的高铁项目中标后被毁约，涉及央企在拉美的基建项目风险。 → Carl业务映射：用于央企拉美市场投资培训——讲解中国铁建在拉美市场的经验和教训，涵盖政治生态评估、合同保护、国际竞标策略。

- **案例六 中远希腊港口**：中远在希腊的港口股权收购叫停后，通过外交渠道和政府支持化解危机。 → Carl业务映射：用于央企海外投资的外交资源运用培训——讲解央企如何运用中国政府的外交资源支持和保护海外投资项目（使馆协助、政府间协议、高层互访推动）。

- **案例七 密松水电站**：中电投在缅甸的密松水电站被叫停，是央企在东南亚最大损失项目之一，至今仍是中国企业海外风险教育的标志性案例。 → Carl业务映射：用于央企海外项目风险评估培训——以密松为标尺案例，讲解央企在东南亚做大型基建项目时必须做好的风险评估（环境、社会、政治、媒体）。

- **案例十一 中铁建麦加轻轨**：中国铁建在沙特麦加轻轨项目巨亏41.53亿元，是央企海外EPC项目亏损的典型案例。 → Carl业务映射：用于央企海外EPC项目投标决策培训——以中铁建麦加轻轨为案例，分析央企低价中标策略的利弊，以及海外项目投标的决策优化模型。

- **案例十三 中海外波兰项目**：中海外（中铁旗下）在波兰公路项目巨亏退出，遭遇3年市场禁入，是央企在欧洲市场最典型的失败案例之一。 → Carl业务映射：用于央企欧洲市场准入策略培训——以波兰案例为反面教材，讲解央企进入欧盟市场的合规要求、竞标规则、社会责任标准。

## ⑦ 行业趋势

- **本书前言综述**：2016年中国对外直接投资约1700亿美元（同比增44%），对外承包工程营业额1594亿美元（同比增3.5%），但中国企业频频遭遇危机，"资源掠夺者""环境破坏者""国家安全威胁者"等负面标签严重阻碍国际化进程。 → Carl业务映射：用于国际化人才培养趋势分析——从中国企业"走出去"规模的快速增长和危机频发的矛盾中，解读企业国际化人才（尤其是危机管理、跨文化沟通、属地化管理人才）的急缺需求。

- **案例一至三的主题趋势**：逆全球化背景下贸易保护主义日益抬头——华为遭美封杀、三一遭总统令、国家电网遭澳政府拒绝，反映出"国家安全"正成为贸易保护主义的新武器。 → Carl业务映射：用于企业国际化战略趋势分享——分析逆全球化趋势对出海企业的影响，帮助客户理解"全球化→区域化"的范式转变，以及对应的组织能力和人才能力调整。

- **案例四至七的主题趋势**：东道国政党斗争风险日益突出——菲律宾北铁、墨克高铁、中远希腊、密松水电站四个案例均受政治博弈影响，反映"政治风险"成为中国企业出海的核心挑战。 → Carl业务映射：用于海外投资政治风险趋势分析——讲解中国企业出海从"经济风险为主"到"政治风险为主"的趋势变化，以及对应的人才培养方向（政治分析、政商关系、政府沟通）。

- **案例八至十的主题趋势**：利益相关者管理成为核心能力——五矿秘鲁社区冲突、上汽双龙劳资矛盾、振华重工质量门，反映项目周边社区、工会、媒体等利益相关者的影响力越来越大。 → Carl业务映射：用于ESG趋势培训——讲解ESG（环境、社会、治理）在全球投资中的重要性日益提升，以及中国企业海外运营必须提升利益相关者管理能力的趋势。

- **案例十一至十五的主题趋势**：国际市场评估不足导致惨痛教训——中铁建麦加轻轨低价中标、中建巴哈马融资模式风险、中海外波兰低价竞标、五矿收购报价不足、阿里集体诉讼——反映中国企业在国际市场规则、合同管理、法律合规方面的系统性短板。 → Carl业务映射：用于企业国际化能力建设趋势分析——从15个案例中概括"五大系统性短板"（市场调研不充分、合同管理薄弱、合规意识不足、跨文化能力欠缺、危机应对被动），以及对应的培训能力建设方向。

- **核心方法论趋势**：本书采用"微观-宏观双情境分析"框架（Coombs情境危机理论+政治经济文化宏观诱因分析），超越单一的危机应对视角，强调"危机是微观情境与宏观环境的互动结果"。 → Carl业务映射：用于企业国际化培训方法论升级——借鉴双情境分析框架，将培训从"技能导向"升级为"情境导向"（微观技能+宏观环境理解），培养学员在复杂国际环境中的系统决策能力。
