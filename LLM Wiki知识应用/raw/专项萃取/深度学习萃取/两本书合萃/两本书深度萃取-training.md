## 一、从两本书萃取的培训产品线

### 产品1：「海外项目全周期风险管理沙盘」🎯

**定位**：面向央企/出海企业的中高层管理者，沉浸式沙盘演练

**适用客户**：中核、中电投、中交、中海油、中铁、中建等有海外工程项目的央企

**设计思路**：
- 基于15个案例提炼5类风险（政治/社会/法律/经营/品牌）
- 每个风险类型配2-3个真实案例做"情境卡"
- 参与者在沙盘中依次面对：投标决策（中海外波兰模式）→ 中期危机（五矿秘鲁社区冲突模式）→ 末期崩盘（中建巴哈马模式）
- 每轮决策后复盘："如果用华为XX方法，会怎样？"

**可嵌入R.I.S.E.体系的位置**：
- R（业务导向）：直接对标客户的海外项目实战需求
- I（沉浸场景）：沙盘演练本身就是沉浸式学习
- S（系统框架）：华为HEROS/GME/5R等模型提供方法论支撑
- E（评估验证）：每轮沙盘后评估决策模型和学习效果

**产出物**：沙盘工具包（风险卡/决策卡/复盘模板）+ 配套引导手册

---

### 产品2：「跨利益相关者沟通实战工作坊」🤝

**定位**：面向海外项目经理/驻外人员，解决"跟谁说什么怎么说"的问题

**适用客户**：所有有海外运营团队的企业

**设计思路**：
- 基于危机案例的"利益相关者地图"方法
- 五类关键对象：政府（密松/菲律宾北铁）、工会（上汽双龙）、社区（五矿秘鲁）、媒体（振华质量门/华为遭封杀）、合作伙伴（中海外波兰）
- 每类提供：沟通策略模板 + 底线检查清单 + 失败案例 + 成功案例
- 结合华为5R法则设计沟通SOP

**差异化卖点**：目前市场上几乎没有"利益相关者沟通"这个方向的跨文化培训产品

**课程模块**：
1. Module 1: 利益相关者识别与分层（华为GME模型）
2. Module 2: 政府沟通——政治风险评估与关系建立
3. Module 3: 工会沟通——劳资文化与谈判策略
4. Module 4: 社区沟通——ESG与"社会许可证"获取
5. Module 5: 媒体沟通——危机声明与舆论监控
6. Module 6: 合作伙伴管理——合同、信任与冲突解决

---

### 产品3：「海外项目负责人能力进阶」📈

**定位**：面向企业从"出海新人"到"海外项目总监"的完整能力发展路径

**适用客户**：正在建设海外人才梯队的企业

**设计思路**：
对标华为P1-P5五级人才梯队 + 转身四步（角色认知→履职能力→行为改变→绩效结果）

**L1 海外新人（对应P1）**：
- 跨文化意识启蒙（密松/上汽案例引入）
- 基础商务英语+项目英语
- 海外安全与合规培训

**L2 海外项目骨干（对应P2-P3）**：
- 跨文化团队协作（G项目群案例）
- 项目全周期沟通（GME/5R/一会一报一表）
- 属地化团队管理

**L3 海外项目负责人（对应P4-P5）**：
- 海外项目全周期风险管理（五重风险模型）
- 涉外商务谈判与高层客户关系
- 危机应对与政治生态分析
- 项目经营责任制与授权管理

**对齐CEFR**：
- L1对应B1-B2（能独立完成基础工作沟通）
- L2对应B2-C1（能管理团队和项目沟通）
- L3对应C1-C2（能进行商务谈判和高层汇报）

## 二、现有课程的素材增强

### 跨文化培训（R.I.S.E.已有课程）
**可新增案例模块**：
- 案例1：上汽双龙——中韩文化差异导致并购失败（2小时工作坊）
- 案例2：五矿秘鲁——合法≠合理，社区信任建设（2小时工作坊）
- 案例3：华为G项目群——7000岛屿的跨文化管理（1小时案例研讨）

**新增内容**：
- "利益相关者文化地图"工具——识别目标国各类利益相关者的文化特征
- "政治文化评估清单"——进入新市场前的政治风险自检

### 涉外演讲谈判课程
**可新增案例模块**：
- 案例1：华为GME模型——演讲的"分层靶向"技术（1小时）
- 案例2：中铁建麦加——巨亏项目的多方谈判（2小时沙盘）
- 案例3：中海外波兰——索赔谈判与纠纷处理（1.5小时）

**新增内容**：
- "一会一报一表"模板——英文项目例会/汇报/跟踪的标准化结构
- 5R法则的英语培训应用——设计"对谁说+说什么"的场景练习

### 企业外语培训课程
**新增模块**：
- "危机沟通英语"模块——媒体声明、法律沟通、工会对话（基于华为遭封杀/三一诉讼/上汽双龙案例）
- "海外项目汇报英语"模块——GME三层沟通的英语表达训练
- "海外业主谈判英语"模块——索赔谈判、合同变更、价值呈现
