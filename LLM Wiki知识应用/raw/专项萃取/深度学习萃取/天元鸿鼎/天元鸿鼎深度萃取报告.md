## 二、核心方法论体系

### 2.1 项目经理系统化培养「五步法」

任职资格建立 → 队伍现状摸底 → 课程讲师开发 → 分层培训实施 → 培训项目评估

| 步骤 | 内容 | 工具/方法 |
|------|------|-----------|
| 定标准 | 建立任职资格体系 | 岗位能力素质模型 |
| 摸现状 | 能力素质测评 | 钻石模型（天元独家）、冰山模型、能力/个性/动力三维 |
| 建资源 | 课程+讲师开发 | 学习地图开发工作坊、TTT赋能 |
| 分层训 | 储备→新任→在岗→绩优四阶 | 分层课程体系 |
| 评效果 | 多维考核评估 | 考试/课堂表现/发展任务/导师评价 |

### 2.2 商务人才培养「三步走」

体系规划 → 资源开发 → 落地应用

- **体系规划**：学习地图开发 or 课程体系梳理
- **资源开发**：课程开发+讲师培养+案例开发+导师培养+训练手册+工具开发
- **落地应用**：线上/线下/混合项目设计+导师带徒+认证考核

### 2.3 内训师培养「三策略」

1. **系统规划**：业务战略→人才战略→关键岗位→课程体系→内训师人选→TTT赋能
2. **培训计划牵引**：围绕年度培训班需求→课程开发工作坊→授课技巧训练营→当年即用
3. **定期分享**：每月4人每周1人→线上直播→形成习惯

**标杆案例**：
- 中建铁投：开发128门课程（含19门微课），培养109名讲师
- 中建三局总承包：开发187门课程，培养187名讲师，录制935节视频课（16800分钟）
- 中铁建工一公司：「优师周课」每周线上直播
- 上海隧道局：每月2次线上直播沙龙

### 2.4 「促动式」行动学习

认知心理学为基础，将行动学习流程技术+促动技术+游戏化技术融合。  
聚焦关键问题 → 「破」中聚众力 → 「解」中寻突破 → 「行」上见成效。

### 2.5 敏捷课程体系搭建工作坊

2天2晚，组织各条线业务专家集中共创，产出：
- 0-1年员工学习阶段划分
- 各岗位学习资料包
- 岗位课程清单
- 现有学习资源盘点

### 2.6 项目经理能力素质「钻石模型」

天元鸿鼎独家的项目经理能力素质模型，基于冰山模型深度定制建工行业场景，将能力素质显性化、数据化。测评从三维展开：
- **能力**（能不能）— 知识技能
- **个性**（合不合）— 特质匹配
- **动力**（愿不愿）— 动机需求

## 七、对Carl的启示

### 7.1 **逆向复用**——同一批客户，换一个切入点切入

天元鸿鼎服务**同样的央企**（中建、中交、中核、中铁、中能建），但切入点完全不同：他们做建工行业人才培养，我们做**国际化人才培养**。

**可复用策略**：
- 天元鸿鼎有「项目经理一体化培养方案」，我们可以有「海外项目经理跨文化胜任力培养方案」
- 天元鸿鼎有「商务人才加速胜任一体化方案」，我们可以有「海外商务人才国际化能力方案」
- 天元鸿鼎有「内训师培养」，我们可以有「跨文化内训师培养」——帮客户培养自己的跨文化讲师

### 7.2 **市场打法值得学习**

天元鸿鼎的核心获客方式：
1. **行业论坛** → 展示案例 → 建立专业形象 → 获客
2. **行业协会背书** → 中施企协专家身份 → 官方认可
3. **文章+署名制** → 每个专家有个人品牌 → 建立信任

**对照Carl**：我们做的是企业国际化人才培养，对应的行业协会可能是对外承包工程商会、中国国际工程咨询协会等。可以参考天元鸿鼎与中施企协的合作模式。

### 7.3 「四阶分层」框架可以平移

天元鸿鼎的项目经理分层培养（储备→新任→在岗→绩优）直接对应国际化人才分层：
- 储备期：海外基础语言+跨文化认知
- 新任期：首次外派适应+商务沟通
- 在岗期：深度跨文化协作+谈判
- 绩优期：全球领导力+区域战略

### 7.4 「钻石模型」式能力建模思路

天元鸿鼎项目经理钻石模型（能力+个性+动力三维）可以复用到国际化人才建模：
- 能力：外语能力、跨文化认知、国际商务知识
- 个性：文化适应性、开放度、抗压力
- 动力：外派意愿、发展动机、全球化职业规划

### 7.5 **差异化竞争建议**

天元鸿鼎强在**体系规划**（学习地图、任职资格体系），Carl天然强在**内容层**（R.I.S.E.课程、跨文化真实场景）。两方其实可以互补而非竞争：
- 天元鸿鼎做「体系」+ 我们做「国际化内容」
- 或者我们直接建立**企业国际化人才培养体系**的完整能力，对标天元鸿鼎在横轴（建工）的生态位

### 8.1 轴一：设计理念（6条核心信条）

天元鸿鼎所有产品的设计都遵循一套内在一贯的哲学：

**① 「体系先行」——不以课为本，以体系建设为本**
他们从不卖单门课程，卖的是「一体化培养方案」「学习地图」「任职资格体系」。设计逻辑始终是：**业务战略 → 人才战略 → 关键岗位 → 课程体系 → 内训师 → 落地运营**。范金才反复强调——先有体系，后有课程。

**② 「以终为始」——内训师用不起来的根因是「先培养再找需求」**
范金才最犀利的洞察：很多企业把内训师培养当成动作（为培养而培养），结果培养完了不知道用在哪里。正确做法：先锁定年度培训计划的课程需求，再定向培养讲师。这也是他做「学习地图工作坊」的根本目的——确保内训师有课可讲的基建工程。

**③ 「场景化压倒理论化」——成人学习就是两个字：实用**
刘通在新人培养那篇写得很透彻：很多课程是"读规范"，好一些的是"讲案例"，学员听了收获不大。真正有效的是围绕**工作场景**设计课程——员工面对的是一个一个"这件事怎么解决？那件事怎么解决？"，而不是知识体系。

**④ 「训战结合是产品基因，不是附加项」**
从葛洲坝后备项目经理班16天120课时（原文原则"刚性务实管用"）到项目铁三角3天2晚沙盘（"让学员在实战模拟中暴露问题、自我觉察"），每个项目都有沙盘推演/模拟策划/行动学习。

**⑤ 「分层培养是普适框架」**
储备→新任→在岗→绩优，四阶培养的思维方式平移力极强，任何岗位都可以套用。

**⑥ 「组织智慧传承 > 外部输入」**
内部师资占比七成（葛洲坝项目原文）。培养内训师不是为了省成本，而是让企业自己沉淀优秀经验、形成可复制组织能力。

### 8.4 三轴内在逻辑：商业闭环

```
客户要"体系"（不是单课）
    ↓
天元鸿鼎设计"学习地图+任职资格体系"（体系先行）
    ↓
方法论产品化为可售卖工作坊（敏捷课程搭建+钻石模型+行动学习）
    ↓
交付时给客户有形产出（手册/策划/复盘/讲师队伍）
    ↓
行业协会背书降低信任成本（中施企协）
    ↓
进入下一个客户的采购清单
```

**Carl差异化补位机会**：天元鸿鼎把建工体系的闭环跑通了，但**国际化人才培养**这条链上没有一个同样级别的玩家。如果我们能建立对标的命名体系+产品化工作坊+行业协会背书，在同一批央企客户中竞争壁垒非常清晰。
