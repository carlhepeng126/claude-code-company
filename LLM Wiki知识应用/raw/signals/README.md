# Signals · 信号日志

> **来源**: 每日信息扫描中的趋势信号
> **处理规则**: 定期汇总 → `signal-trends-summary-*`；可行动信号 → Action

## 处理后流向

| 模式 | 操作 |
|---|---|
| 单一事件信号 | 保留在 raw/signals/，标记日期 |
| 形成趋势（≥3个同类信号） | → concepts/`signal-trends-summary-2026-XX` |
| 可转化为业务行动 | → ACT `3-Action/` |

## 处理周期
每周回顾一次：合并同类信号，判断是否形成趋势或行动项
