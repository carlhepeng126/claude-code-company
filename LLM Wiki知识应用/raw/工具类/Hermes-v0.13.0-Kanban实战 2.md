# Hermes Agent v0.13.0 - Kanban实战

公众号：量子智元
发布时间：2026-05-08

Hermes Kanban功能：一句话指令 → AI自动拆分子任务 → 拉出多个研究员角色同时干活 → 等全部完成后第三角色合并输出。全程无需人工干预。

核心理念：以前AI做多步任务让人不踏实，Kanban模式让"说一句话，AI帮你组了个小团队"成为现实。
