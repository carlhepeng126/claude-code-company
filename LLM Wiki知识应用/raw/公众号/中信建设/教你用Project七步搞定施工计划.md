#  教你用 Project 七步搞定施工计划

编制施工进度计划  ** 。工程部员工在不熟知工程进度计划编制的情况下，通过学习本微课，可以七步实现自己编写工程进度计划。  

  

大家好，今天要给大家介绍的是，利用 Project 编制施工进度计划。我们知道，  **施工进度计划**
是施工过程中的一个重要指标，其编制的先进性、合理性将直接影响到整个施工的全过程。同时呢，  **Project 作为一款非常好用的项目管理工具**
，凝集了许多成熟的项目管理现代理论和方法，能够让我们对项目进度了如指掌，在如今的许多工程项目中，应用颇为广泛。

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9OfKLiaiagFy9Aic6a15iaqoDzORkI1u64opee9iatjunne8D6AQ1I2lOdOJA/640?wx_fmt=png)

  

为了让大家对 Project 有个整体的认识，那么先给大家介绍一下软件的操作界面，我们将 Project 的  **操作界面**
大致的分为了以下的5个部分，分别是标题栏、菜单栏、每个菜单栏所对应的工具栏、日程表、任务编辑区以及视图显示区。那么，整个的编制过程就是在左侧的任务编辑区输入任务之后，利用菜单栏下的工具进行相关编辑，然后项目任务的甘特图就会在右侧的视图显示区显示出来了。

  

事情是一步一步来，进度计划的编制也是按部就班来进行的，以下的  **7个步骤**
将让你更清晰地了解进度计划的编制流程，分别是设置项目开始时间、设定工作时间、添加任务名称、设定任务级别、设置任务工期、设定任务间的关系、显示项目关键任务。接下来我们将以一个桥梁施工项目为例进行讲解。

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9O8SLIl7ZLKnZj6paAmkvcSpDpJjqjY0A7oViayLbkBusfpXgFwegaV8w/640?wx_fmt=png)

  

第一步是设置项目开始时间，顾名思义，项目开始时间即为整个项目的起始时间，在项目菜单栏下的任务信息中进行设置，比如说我们将项目的开始时间设置为2019年12月5日，点击确定，设置完毕。

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9ORPu8JKLM52GgIiaKYv0MUbhC9xsaibj0QNC1Ijg5nicJOG4I2Rv7CQw4g/640?wx_fmt=png)

  

第二步是设定工作时间，这步操作其实就是更改工作时间，对项目日历进行设置，在这里我们将标准日历为例进行说明。我们知道，在施工项目中，周六日也为工作时间，在工作周中进行设置，将这两天时间设置为工作时间。同样也可以在例外日期中设置节假日如春节中秋啊等其他日子。

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9O0Vp5tvBtr1zYLldbXJUAb58NpHlicquoUYicXOyAQ47vibhMk24lFDNXQ/640?wx_fmt=png)

  

项目开始时间和工作时间是对一个项目最基本的设置，那么第三步就需要添加任务了，为了便于工期的自动计算，我们将任务模式调整为自动计划，然后输入任务名称。可以看到，在自动计划模式下，各个任务的工期都默认为一个工作日，开始时间都默认为项目的开始时间。这就是自动计划模式的优势啦。

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9OqJH6uGcc5lAAOa9fsatR4BZ6x4bmeNlMJo6rNauuo11icicic68Q9nrrw/640?wx_fmt=png)

  

任务添加进去之后，第四步就需要设定任务级别了，其实就是将项目进行WBS工作分解，设置摘要任务与子任务。该操作是通过任务菜单栏下的升降级工具来实现的，选中要升降级的任务，点击升降级工具就可以实现了。在project中，可以设置多级摘要任务，例如，将公路桥梁施工设置为一级摘要任务，施工准备设置为二级摘要任务，同样，1#桥墩也设置为二级摘要任务。

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9OpdolwNCx24AOkC0jNnmgG6fm3RKicWAcnCuJCdvNpwRuLLicCMajHlDw/640?wx_fmt=png)

  

第五步是设定任务工期，就是将各个任务的工期输入到相应的工期栏中，其中，摘要任务的工期不需要编辑，它的工期是子任务工期的时间跨度，自动计算。需要说明的是，工期为零的任务为里程碑任务。

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9OZQnTmMLFQQ7VEiaRvr052boLhGDCgRLRRazNIYb4AlhkicV2nHY6cquw/640?wx_fmt=png)

  

项目的各个任务之间会存在着时间和逻辑的联系，这就需要对任务进行链接，第六步便是设定任务间的关系。该操作是通过前置任务进行实现的，就是将一项任务的编号输入到另一任务的前置任务栏中，就能实现两项任务的连接了。任务链接后的甘特图如图所示。

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9OxNnKgrBGnvzEU1aUMQM9kkg6tv57jZn0jE1qP0m8OvLhrBPIs1ILyQ/640?wx_fmt=png)

  

到这里，项目进度计划的编制就差不多了，在工程项目中，通常需要找出项目的关键路线，那么第七步就要显示项目关键任务。在格式菜单栏中勾选关键任务即可显现，如图，在甘特图中显示为红色的任务即为项目关键路线上的任务。同时，我们也可以突出显示关键任务，在视图菜单栏下的突出显示中选择关键，即可突出标黄关键任务，如图所示。至此，编制工程进度计划的7个步骤就结束了，大家学会了吗？

  

![](https://mmbiz.qpic.cn/mmbiz_png/x3uBG25kUErO9LOhZBuj5vsWbaWnja9OCHtjYrtJpoTQanr7iaG3C21RUM7LiaFicKfGib3LiaWjLicnPydWic9om5mIA/640?wx_fmt=png)

  

最后，如果需要提交纸质文件，或者存档查阅，那么就需要将文件进行打印处理，在这里要提醒大家，一般工程进度计划时间跨度较大，可以在打印设置中选择打印某一时间段的计划哦，同时也可以在页面设置中对页眉页脚图例进行设置，设置完毕收，打印预览如图所示。

  

到这里就全部结束喽！谢谢大家！
