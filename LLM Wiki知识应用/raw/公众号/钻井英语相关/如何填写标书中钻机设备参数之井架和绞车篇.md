#  如何填写标书中钻机设备参数之井架和绞车篇

在投标过程中，根据甲方招标文件中钻机设备的规范，填写专业钻井公司所提供的设备参数，是提交给甲方招标文件中技术标必有的一项工作。甲方，以此来评定钻井公司提供设备是否满足甲方的需要。钻机涉及很多关键设备，每个设备又有许多不同参数，该如何决定用哪个参数填写该设备呢？笔者，结合近来的工作以及IADC对钻机设备的规范要求，整理出通用的设备参数，供各位海外钻井同仁们参考和学习。

I.Mast&Substructure(井架和底座)

1.井架（Mast/Derrick）

该部分主要用到的参数如下:Make/type-井架生产厂家和类型;height-井架高度;static hook load-大钩静载荷;max.hook
load with 6 lines strung-6*6绳系最大钩载

2.Racking Platform/Monkey Board(二层台)

make/type-生产厂家和类型;Racking platform capacity of drill pipe/drill
collar:size/no.二层台钻杆/钻铤立根容量，钻杆、钻铤尺寸/数量

3.Substructute/Work Floor(井架底座)

make/type-生产厂家和类型;height-底座高度;width-底座宽度;setback capacity-立根载荷;rotary
capacity-转盘载荷;clear height below rotary table beam/work floor-转盘梁底面高度;

4.Rig Floor Winches(钻台绞车)

make-厂家;quantity-数量;type-类型;capacity&S.W.L.-载荷&安全载荷;wire diameter-
钢丝绳直径;automatic brake-是否自动刹车  ;automatic spooling-是否自动排绳;

5.Monkey Board Work Winch（二层台绞车）

make-厂家;quantity-数量;type-类型;capacity&S.W.L.-载荷&安全载荷;wire diameter-
钢丝绳直径;automatic brake-是否自动刹车  ;automatic spooling-是否自动排绳;

6.Utility Winch(for Catwalk or man riding)-多功能绞车（猫道使用或载人）

make-厂家;quantity-数量;type-类型;capacity&S.W.L.-载荷&安全载荷;wire diameter-
钢丝绳直径;automatic brake-是否自动刹车  ;automatic spooling-是否自动排绳;

7.Catwalk/Pipehandler(猫道/排管装置)

make-厂家;type-类型;capacity&S.W.L.-载荷&安全载荷;integral pipe
racks:是否是整体式钻管支架;standalone pipe rack:是否为单独钻杆支架

II.Drawwork& Associated Equipment(绞车及相关设备)

1.Drawwork(绞车)

make-厂家;type-类型;drum type-滚筒类型;spinning cathead type-上扣猫头类型;breakout cathead
type-卸扣猫头类型;traveling blocks-crown block anti-collision safety device
type/model-游车天车防碰安全装置类型/型号;traveling block-drill floor anti-collision safety
device type/model-游车钻台防碰安全装置类型/型号;rated capacity-最大快绳拉力;input power
continuous-持续输入功率;brake type-刹车类型;maximum safe hook load 6
lines-6绳系大钩最大安全载荷;maximum safe hook load 4 lines-4绳系大钩最大安全载荷;

2.Drawwork Power(绞车动力)

number of electric motors-电机数量;engine make-电机厂家;engine model-电机型号;maximum
input power continuous-持续输入最大功率;

3.Auxiliary Brake(辅助刹车)

make-厂家;type-类型;model-型号
