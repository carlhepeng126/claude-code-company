#  钻杆和接头分类的色彩编码系统

国际钻井承包商对钻井现场使用的钻杆和钻杆接头应用颜色编码识别系统（color coding identification
system）来对钻杆的等级(drill pipe class)或是接头的等级(tool joint
class)进行分类，并用不同数量的颜色带（number of color band)来表示,具体如下：

1.Premium Class(高级别)钻杆或接头对应着2 条白色带(2 White Bands)

2.Class 2钻杆或接头对应着一条黄色带(1 Orange Band)

3.Class 3钻杆或接头对应着一条橙色带(1 Orange Band)

4.Scrap(报废)钻杆或接头对应着一条红色带（1 Red Band）

使用color coding system还可以对钻杆接头情况进行分类，具体如下：

1.如果接头是报废或是返厂修理（Scrap or Shop Repairable）那么用红色带表示。

2.如果接头在现场可修理（Field Repairable）,则用绿色带表示。
