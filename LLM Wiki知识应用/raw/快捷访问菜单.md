## 🔥 快捷指令（精确路径）

| 你说 | Hermes 会做什么 |
| --- | --- |
| `今天任务` / `今天` / `日计划` | → 打开 `ACT笔记-深度思考/5-Time/5-4-Daily-日计划/` 下**对应今天日期的文件**（如 `2026-05-05.md`） |
| `本周任务` / `本周` / `周计划` | → 打开 `ACT笔记-深度思考/5-Time/5-3-Weekly-周计划/` 下**本周的文件**（如 `2026-05-W01.md`） |
| `核心聚焦` / `聚焦` | → 列出 `ACT笔记-深度思考/3-Action/3-1-Focus-聚焦承诺/` 中**所有项目的概要情况列表**（背景/截止/下一步） |
| `核心卡` / `maincard` | → 列出 `4-Card/4-3-MainCard-核心卡/` 中**与当下项目/任务相关的卡片**，按项目分类展现 |
| `概念` / `concepts` | → 只列**最近两周形成、与当下工作有关联**的概念文件 |
| `统计` / `数量` / `看看` | → 各文件夹文件数速览 |
|| `收件箱` / `inbox` | → 列出 `收件箱：同步助手/企业出海日报/` 下当天文件，按来源分类统计数量 |
| `打开 <文件名>` | → 读取文件内容（需包含路径或从当前上下文推断） |
| `搜 <关键词>` | → 全库搜索 |
| `巡检` / `wiki巡检` / `知识库巡检` | → 扫描 `concepts/` + `entities/`，删除无效笔记，孤立内容建立关联，断链修复 |
| `月报` / `月度分析` / `月度工作复盘` | → 综合分析当月ACT笔记（聚焦/核心卡/周报/日报），输出工作状况+存在挑战+挑战解决洞察+持续优化方向 |
| `阅读卡` / `bibcard` / `读书卡` | → 列出 `4-Card/4-2-BibCard-阅读卡/` 下**近1个月**的卡片，带序号返回，你发序号给我我就与你深度交流 |
