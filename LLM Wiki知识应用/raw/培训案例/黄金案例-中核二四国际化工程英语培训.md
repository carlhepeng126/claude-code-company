# 黄金案例：中核二四国际化工程项目英语人才脱产培训

## 一、客户说了什么

| 维度 | 信息 |
|------|------|
| 客户 | 中核二四建设有限公司（中核集团子公司） |
| 背景 | 公司"十四五"规划海外市场拓展战略目标，首批"走出去"人才培养 |
| 需求表述 | "储备一批具备国际化工程知识和英语沟通能力的涉外工程人才" |
| 学员 | 23名具有一定英语水平的专业骨干 |
| 形式 | 全封闭脱产培训，70天 |
| 合作方 | 华北科技学院（校企联动） |
| 内容 | 涉外法律英语、国际工程承包和管理实务英语、国际工程案例教学、国际巡礼课程 |
| 高层参与 | 总经理、党委副书记出席，三点动员讲话（责任感、视野格局、纪律） |
| 时间 | 2022年11月 |

## 二、客户画像速写

- **行业**：核电工程建设
- **出海阶段**：准备出海（战略规划已定，人才储备先行）
- **驱动类型**：战略驱动型（自上而下，总经理站台）
- **紧迫度**：中高（"十四五"目标倒逼）
- **人员规模**：23人，精挑细选的骨干
- **合作模式**：校企联动（与华北科技学院深度战略合作）

## 三、【待补充】诊断过程

> 你当时问了什么关键问题？发现了客户没说出来的什么？

```
1. 第一轮沟通，客户原话是什么？

2. 你追问了什么？（比如：23人选拔标准是什么？70天怎么定出来的？为什么是校企联动而不是请外部讲师？）

3. 发现了什么客户没意识到的问题？（比如：英语水平差距比他们认为的大/小？培训目标太泛？脱产70天对业务影响？）

4. 你的核心判断是什么？（为什么必须这么做，而不是更短/更长/另一种形式？）
```

## 四、【待补充】方案决策理由

> 为什么是这个方案，而不是别的？

| 决策点 | 你选了什么 | 为什么没选替代方案 |
|--------|-----------|-------------------|
| 方案类型 | （单次培训 / 系列课程 / 体系项目） | 替代方案为什么不行？ |
| 培训时长 | 70天 | 为什么不选30天或90天？ |
| 合作模式 | 校企联动（华北科技学院） | 为什么不选外部讲师/内部培养？ |
| 学员选拔 | 23人专业骨干 | 为什么不扩大/缩小范围？ |
| 内容重点 | 法律+承包管理+案例+巡礼 | 为什么不加/减某些模块？ |

## 五、最终交付方案

- **类型**：单次脱产培训
- **时长**：70天全封闭
- **人数**：23人
- **师资**：华北科技学院 + 校企联动
- **内容模块**：
  1. 涉外法律英语
  2. 国际工程承包和管理实务英语
  3. 国际工程案例教学
  4. 国际巡礼课程
- **特色设计**：英语+工程双线并行，不是纯语言培训
