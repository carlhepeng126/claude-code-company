# 中国国新控股海外业务与培训调研报告

## 一、企业概况

中国国新控股有限责任公司（中国国新）是国务院国资委监管的中央企业，成立于2010年12月。2016年初被确定为国有资本运营公司试点，2022年12月转入持续深化改革阶段。公司以"人才+科技+资本+文化"四位一体运营模式，构建基金投资、直接投资、股权运作、资产管理、金融服务、境外投资等业务格局。截至2025年底，资产总额超万亿元，利润总额连续五年保持在200亿元以上。[L1]

## 二、海外项目情况

### 2.1 境外投资平台概况

中国国新依托**国新国际投资有限公司**打造境外投资业务平台，以市场化方式联合中国企业实施"走出去"战略，推动"一带一路"倡议。[L1]

| 投资指标 | 数据 |
|----------|------|
| 累计投资项目 | 超110个 |
| 累计投资金额 | 超2,500亿元 |
| "一带一路"占比 | 约80% |
| 覆盖国家和地区 | 近30个"一带一路"沿线国家 |
| 提供就业培训机会 | 十数万个 |

### 2.2 境外投资重点领域

**投资方向**：
- 能源资源安全
- 高端技术并购
- 国际产能合作

**投资类型**：
- 直接投资
- 股权投资
- 基金投资

**支持服务**：
- 央企国际化经营资金支持
- 跨境资本运营
- 海外项目投融资

### 2.3 国际合作动态

**阿塞拜疆合作（2026年3月）**：
- 与阿塞拜疆交通与通信控股公司座谈
- 聚焦交通、通信、基础设施建设、能源资源等领域
- 探讨"金融+产业"合作模式
- 阿塞拜疆交通与通信控股公司于2024年11月成立，统筹管理阿13家交通、通信国有企业[L1]

**海南自贸港战略（2025年12月）**：
- 把握海南自贸港封关运作机遇
- 深化与"一带一路"沿线国家合作
- 提升跨境资本融通能力
- 探索境外资本运作新机制

### 2.4 服务央企国际化

**累计服务成效**：
- 累计为近40家中央企业国际化经营提供约361亿美元资金支持[L2]
- 助力央企"走出去"战略实施
- 支持央企境外项目投融资

## 四、培训体系与涉外培训

### 4.1 人才战略

**核心理念**：人才是中国国新"人才+科技+资本+文化"轻资产运营模式的第一资源。

**人才方针**：
- "运营报国，人才第一"
- 坚定不移实施人才强企战略
- 统筹推进经营管理、投资运营、产业科技三支人才队伍建设

### 4.2 培训体系

**四类培训内容**：
- 党的理论教育
- 党性教育
- 履职能力培训
- 知识培训

**八个方面体系**：
- 管理体系
- 组织体系
- 评估体系
- 转化体系

**"四梁八柱"格局**：分层分类开展培训，构建具有运营公司特色的培训体系

### 4.3 "青马工程"暨青年干部培养"新锐行动"

**项目定位**：公司战略性青年人才培养工程

**培养目标**：
- 信仰坚定
- 能力突出
- 素质优良
- 作风过硬

**培养成果（第二期）**：
- 系统研学党的创新理论
- 参与公司"十五五"规划建言献策
- 形成多篇高质量调研报告
- 多名学员获得晋升或表彰

**第三期安排**：
- 北京、延安两地培训
- 16天集中培训
- 理论素养、创新思维、实践本领三重品格塑造

### 4.4 人工智能专题培训班

**"进产业央企·育产融人才"行动**：
- 第2期人工智能专题培训班（2025年10月）
- 聚焦产业前沿
- 邀请中国社科院、清华大学、中国电信、鹏城实验室等专家授课

**培训主题**：
- 人工智能和大模型技术
- 智能体和具身智能产业
- 算力基础设施
- 人工智能行业应用

**走进产业央企**：组织学员走进中国电信，强化产融人才联合培养

### 4.5 "国脉新星计划"管培生

**培养特点**：
- 面向全球知名顶尖高校硕博士研究生
- 高潜人才定制化培养
- 配备专属导师
- 多岗位轮岗
- 优先参与重大专项任务

## 五、培训缺口分析

### 5.1 境外投资专业培训缺口

| 缺口领域 | 现状 | 问题 |
|----------|------|------|
| 跨境并购实务 | 有基础培训 | 系统性课程不足 |
| 国际金融产品 | 分散培训 | 缺乏体系化 |
| 境外合规风控 | 有合规培训 | 专业深度不够 |
| 海外尽职调查 | 在实践中培养 | 培训资源有限 |

### 5.2 "一带一路"国家培训缺口

**现状**：
- 投资覆盖近30个"一带一路"国家
- 涉及多元文化和法律体系

**缺口**：
- "一带一路"国家国别研究培训
- 多元文化管理培训
- 小语种能力培训
- 区域政治经济风险培训

### 5.3 国际化人才培养缺口

**现状**：
- 有"青马工程"等青年干部培养项目
- 有"国脉新星计划"管培生项目

**缺口**：
- 境外项目一线实践机会有限
- 国际机构交流合作不足
- 海外考察学习项目偏少

### 5.4 科技金融融合培训缺口

**现状**：
- 已开展人工智能专题培训
- 与产业央企开展"进产业央企"合作

**缺口**：
- 科技投资专业能力培训
- 新兴产业趋势判断培训
- 科技创新与资本运营融合培训

## 六、商业机会

### 6.1 境外投资专业培训

**机会描述**：中国国新累计为近40家央企提供361亿美元国际化资金支持，境外投资专业培训需求持续。

**合作方向**：
- 跨境并购实务培训
- 国际投融资培训
- 境外项目尽职调查培训
- 跨境合规风控培训

### 6.2 "一带一路"国别培训

**机会描述**：投资覆盖近30个"一带一路"国家，需要深入了解各国政治、经济、法律、文化环境。

**合作方向**：
- "一带一路"重点国家国别培训
- 多元文化管理培训
- 国际工程风险管理培训
- 区域经济形势分析培训

### 6.3 科技投资培训

**机会描述**：战略性新兴产业投资是核心方向，科技投资专业能力培训需求旺盛。

**合作方向**：
- 硬科技投资方法论培训
- 新兴产业趋势判断培训
- 科技创新与资本运营融合培训
- 科技成果转化培训

### 6.4 产融结合培训

**机会描述**：国有资本运营公司定位需要金融与产业深度融合。

**合作方向**：
- 产融结合案例培训
- 科技金融产品创新培训
- 上市公司资本运营培训
- ESG与可持续发展培训

### 6.5 国际化人才培养合作

**机会描述**：服务央企国际化的同时，需要提升自身国际化人才培养能力。

**合作方向**：
- 国际机构合作培训
- 海外考察学习项目
- 境外项目实践交流
- 跨文化领导力培训

## 七、总结与建议

### 7.1 主要发现

1. **境外投资以资本运营为主**：累计投资超2500亿元，80%投向"一带一路"国家
2. **服务央企国际化定位明确**：累计为近40家央企提供国际化资金支持
3. **人才培养体系特色鲜明**："青马工程"、"国脉新星计划"等品牌项目
4. **科技投资培训起步**：人工智能专题培训与产业央企合作
5. **属地化培训有限**：境外投资项目培训主要依托属地资源

### 7.2 商业机会优先级

| 优先级 | 机会领域 | 市场规模 | 切入难度 |
|--------|----------|----------|----------|
| 高 | 境外投资专业培训 | 中 | 低 |
| 高 | 科技投资培训 | 中 | 中 |
| 中 | "一带一路"国别培训 | 中 | 中 |
| 中 | 产融结合培训 | 中 | 低 |
| 低 | 国际化人才培养合作 | 小 | 中 |

### 7.3 建议

1. **深化与产业央企培训合作**：延续"进产业央企"行动，拓展合作范围
2. **加强"一带一路"国别研究**：建立重点国家培训资源库
3. **强化科技投资能力培训**：聚焦战略性新兴产业投资方法论
4. **拓展国际机构合作**：与国际组织、跨国企业建立培训合作
