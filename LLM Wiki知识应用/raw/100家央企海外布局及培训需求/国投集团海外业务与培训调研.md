# 国家开发投资集团海外业务与培训调研报告

## 一、企业概况

国家开发投资集团有限公司（简称"国投集团"或"国投"）是中央直接管理的国有资本投资公司，成立于1995年，注册资本338亿元。截至2024年底，集团资产总额8,721亿元，控股装机容量4,463万千瓦，清洁能源占比68.95%。集团连续21年荣获国务院国资委经营业绩考核A级。[L1]

## 二、海外项目情况

### 2.1 英国英奇角海上风电项目

| 项目参数 | 详情 |
|----------|------|
| 投资主体 | 国投电力（上市公司）通过红石能源 |
| 合作方 | 爱尔兰电力公司（ESB），各持股50% |
| 项目公司 | InchCape Offshore Limited（ICOL） |
| 所在国 | 英国苏格兰 |
| 项目名称 | 英奇角（Inch Cape）海上风电 |
| 装机容量 | 1,080MW（72台风机） |
| 总投资 | 41.66亿英镑（约384.52亿元人民币） |
| 水深 | 40-60米 |
| 距离海岸 | 约15公里 |
| 平均风速 | 9.9m/s |
| 发展阶段 | 2024年12月全面开工，预计2027年8月全容量投产 |
| 建设周期 | 约33个月 |
| 购电协议 | 英国政府15年差价合约合同 |

**特点**：
- 英国市场稀缺的优质海上风电资源
- 全投资回报率6.58%，股权出资回报率8.00%
- 已获英国政府长期购电协议保障
- 中国电建集团完成可行性研究评审[L1]

## 五、培训缺口分析

### 5.1 国际化人才培训缺口

| 缺口领域 | 现状 | 问题 |
|----------|------|------|
| 海外风电运维 | 英国项目在建 | 海上风电专业培训体系缺失 |

**英奇角海上风电项目**：
- 项目刚开工，培训需求将快速增长

**泰国钾盐项目**：

- 中泰文化融合培训需求

- 国际前沿技术培训

## 七、总结与建议

### 7.1 主要发现

1. **海外项目规模相对有限**：英国风电项目（384亿元投资）和泰国钾盐项目是主要海外项目
2. **海外业务处于拓展期**：泰国钾盐项目2024年新签，英国项目在建
3. **国际化人才培训基础薄弱**：缺乏系统性海外项目培训体系
4. **科技创新培训有特色**：科改示范企业培训成效显著
5. **国内业务培训体系完善**："十四五"人力资源规划覆盖全面

### 7.2 商业机会优先级

| 优先级 | 机会领域 | 市场规模 | 切入难度 |
|--------|----------|----------|----------|
| 高 | 海上风电运维培训 | 中 | 中 |
| 高 | 海外矿业开发培训 | 中 | 高 |
| 中 | 跨境投资并购培训 | 中 | 低 |
| 中 | 国际工程管理培训 | 中 | 中 |
| 低 | 科改企业创新培训 | 小 | 低 |

### 7.3 建议

1. **优先布局海上风电培训**：英奇角项目投产前建立培训体系
2. **跟进泰国钾盐项目培训需求**：项目开发期是培训需求高峰期
3. **整合集团内部培训资源**：借助国投罗钾等科改企业培训经验
4. **建立国际工程培训合作网络**：与英国、泰国等合作方共建培训体系
