# 第一章：课程体系总览

## 1.1 模块进阶金字塔

```
                    ┌──────────────┐
                    │  M4 区域策略  │  ← 塔尖：差异化落地
                    │  策到位      │        欧美 | 拉美 | 东南亚中东 | 非洲
                    ├──────────────┤
                   ┌┤  M3 合规风控  │  ← 护栏：风险防线
                   ││  防住险      │        制度层 · 流程层 · 资源层
                  ┌┤├──────────────┤
                  │││  M2 全周期培养│  ← 支柱：赋能引擎
                  │││  育成材      │        派前集训 → 派中支持 → 派后回流
                 ┌┤├┤──────────────┤
                 ││││  M1 科学选拔  │  ← 基座：人才入口
                 ││││  选对人      │        四维胜任力 · 八步流程 · 评鉴中心
                 └┴┴┴──────────────┘

   横向贯穿：M5 工具箱（26套模板 + 24份文档，覆盖全周期）
```

**进阶逻辑**：
1. **选是基座** — 选错人，后面所有投入打折扣。从"谁有空/谁想去"变成"谁最适合这个市场"
2. **育是引擎** — 选对之后，系统培养决定走多远。覆盖派前-派中-派后全周期
3. **防是护栏** — 业务跑得快，合规跟不上就是定时炸弹。三层防线嵌入每个流程节点
4. **策是落地** — 同一套体系，欧美/拉美/东南亚/非洲打法完全不同。固定框架60%+可变内容40%
5. **器贯穿始终** — 每个阶段有工具可用，不空谈

#### 5B：跨文化场景训练卡（精选10张）

```
每卡15-20分钟，1对1角色扮演或小组讨论。

场景01·高语境沟通（东南亚/日本）
你在会议上提出一个改进建议，本地同事一片沉默。
你追问"有什么意见吗？"，依然是沉默。
→ 这不代表同意，而是"我需要时间消化"或"不便公开反对"。
   正确做法：会后1对1私下征求意见。

场景02·低语境反馈（德国/荷兰）
你需要给一位德国下属提改进意见。你选择委婉表达。
→ 对方可能完全没意识到你在提意见。
   正确做法：直接、具体、基于事实，不要铺垫太多。

场景03·高权力距离（沙特/印尼）
你在没有提前和本地高管一对一沟通的情况下，
在会议上公开提出了不同意见。
→ 这在当地文化中是严重的冒犯。
   正确做法：提前私下沟通，会议上保持一致。

场景04·弹性时间（巴西/印度）
你说"下周初给我"，巴西同事周四才提交。
→ 他不是在拖延，而是"下周初"在这个文化中≈"下周内"。
   正确做法：给出具体日期+时间，而不是模糊时间词。

场景05·关系优先（墨西哥/中东）
商业伙伴邀请你参加家庭聚会，你以"社交负担"婉拒。
→ 在这个文化中，这意味着"你不把我当朋友"，生意也可能就此终止。
   正确做法：接受邀请，这是建立商业信任的必要投资。

场景06·辩论文化（法国/以色列）
法国下属在会议上公开对你say no，并激烈辩论。
→ 这通常意味着"我尊重你的专业性所以认真对待你的观点"。
   不是不服从，是智力上的尊重。

场景07·宗教习俗（中东/印尼·斋月）
你在斋月期间安排了一次全天客户拜访。
→ 你的穆斯林同事和客户可能在禁食中，精力和耐心都有限。
   正确做法：缩短会议时间，避免午餐会，不要在公共场合饮食。

场景08·送礼禁忌（欧美）
中国总部寄来高档茶叶让你送给当地客户。
→ 在美国/英国/德国，这可能触发反腐败调查。
   正确做法：了解当地的礼品金额上限和申报要求。

场景09·面子文化（越南/韩国）
你在团队会议上点名批评了一位越南员工的错误。
→ 该员工第二天提出了辞职。
   正确做法：公开表扬，私下批评。给台阶下是基本尊重。

场景10·间接拒绝（印度）
印度同事对你说"I will try"，然后什么都不做。
→ 在印度英语中，这常常等同于"我做不到但不好意思直说"。
   正确做法：追问具体的时间节点和需要什么支持。
```
