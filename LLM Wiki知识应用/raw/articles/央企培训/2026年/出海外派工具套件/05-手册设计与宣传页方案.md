## 二、设计风格定义

### 品牌色板

```
主色：深蓝 #1B2A4A  — 专业、信任、国际化
辅色：金色 #C8963E  — 品质、高阶、央企调性
底色：米白 #FAF8F5  — 温暖、阅读舒适
强调色：藏青 #2C5282  — 图表、标题
警示色：暗红 #9B2C2C  — 仅用于风险/合规模块点缀
```

### 字体建议

```
标题字体：思源宋体（Source Han Serif）— 庄重感
正文字体：思源黑体（Source Han Sans）— 易读
工具模板内字体：等宽字体（Consolas / Source Code Pro）— 工具感
```

### 视觉元素

```
图标风格：线性图标（Line Icons），金色描边
数据图表：扁平化柱状图/环形图，藏青色填充
分隔线：金色细线（1px）+ 深蓝粗线（3px）组合
页码：深蓝色圆形底+白色数字
```

## 四、排版制作实操指南

### 方案A：用Canva制作（推荐，最快上手）

1. 注册Canva，搜索"企业报告"或"咨询报告"模板
2. 推荐模板关键词：`corporate report` `consulting proposal` `training manual`
3. 自定义色板（输入上文色号）
4. 按手册结构逐页填充内容
5. 导出为PDF（印刷级）/ PNG（投屏用）

**Canva具体操作步骤**：
```
Step 1: 创建品牌色板
  → 品牌工具 → 调色板 → 添加 #1B2A4A #C8963E #FAF8F5 #2C5282

Step 2: 选择母版
  → 模板中心搜索 "company profile" 或 "training booklet"
  → 选择干净、多留白、几何风格的模板

Step 3: 设置页面尺寸
  → A4 (210×297mm) / 或自定义 200×260mm（更精致的手册尺寸）

Step 4: 逐章制作
  → 封面用提示词1生成图后导入
  → 章节页用提示词2生成图
  → 工具内页在Canva中手动排版（复制母版→改文字）

Step 5: 导出
  → PDF印刷（CMYK, 300dpi）
  → PNG单页（投屏用，RGB, 72dpi足够）
```

### 方案B：用PPT/Keynote（最通用）

适合内部修改和更新：

1. 设置幻灯片大小为A4（设计→幻灯片大小→自定义）
2. 在母版视图中设定统一的页眉/页脚/页码
3. 创建章节母版和工具内页母版
4. 工具模板用表格+形状构建
5. 导出为PDF

### 方案C：用Indesign（专业印刷级）

适合需要印刷成册的情况：

1. 新建文档，A4或自定义开本
2. 设置主页（Master Page）统一页眉页脚
3. 段落样式：标题/副标题/正文/工具标签 各设一种
4. 表格样式：统一工具模板的表格样式
5. 导出：印刷PDF（含出血和裁切线）
