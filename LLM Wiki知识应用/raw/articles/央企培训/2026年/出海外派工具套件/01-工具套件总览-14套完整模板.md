│                                                       │

│   □ 风险判断速度（是否立即意识到严重性）                │
│   □ 合规底线（是否在任何压力下守住法律底线）            │
│   □ 多方沟通（能否同步安抚总部/员工/本地监管）          │
│   □ 制度建设（是否提出防止同类问题再发生的机制）        │
│                                                       │
│ 【剧本3】总部资源争取                                   │

│       你将面对由高管扮演的"总部投资委员会"。            │

│                                                       │
│ 【剧本4】本地员工离职潮应对                             │

│   □ 短期方案（是否有立即可执行的留人措施）              │

## 附录：工具使用优先级建议

| 企业阶段 | 优先使用的工具编号 | 理由 |
