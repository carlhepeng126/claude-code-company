# Presentation Skills Workbook（Johns Hopkins）

## 简介
Johns Hopkins Carey Business School 演讲技能工作坊手册（22页/619行），覆盖演讲全流程：自评→受众分析→目标设定→结构化→PREP模型→视觉设计→演练→呈现→问答→紧张管理→评分。

## 核心方法论：PREP模型
**Point → Reason → Example → Point**
- Point：10字以内陈述观点
- Reason：为什么听众需要记住
- Example：真实案例/故事/数据/类比
- Point：重述核心观点

适用场景：演讲观点展开、谈判论点陈述、会议发言、电梯演讲

## 关键概念清单
| 概念 | 说明 |
|------|------|
| Walk-Away Message | 先想好听众离开时唯一记住的一句话 → 再设计演讲 |
| 10:20:30 Rule | ≤10页PPT、≤20分钟、≥30pt字体 |
| 1hr/1min Rule | 1分钟演讲需要1小时准备 |
| Primacy & Recency Effect | 开头和结尾是最有影响力的位置 |
| Hook | 7种开场技巧：Fact/Quotation/Question/Story/Example/Joke/Realia |
| Q-BEST-Q | Question→Bottom Line→Evidence→Summarize→Time→Ask |
| 7-38-55法则 | 内容7% + 语调38% + 肢体语言55% |
| Power Pose | Amy Cuddy的高力量姿势法 |
| Nervous vs Confident | Option A（紧张型）vs Option B（自信型）心态转换 |

## 15维评分量规
1. 受众识别 2. 导语/注意力 3. 核心论点 4. 论证 5. 组织 6. 结论 7. 时间控管
8. 视听辅助 9. 参与度 10. 发音 11. 音量 12. 节奏 13. 非语言 14. 应对 Q&A 15. 着装

# PREP模型

## 定义
Point（观点）→ Reason（理由）→ Example（例子）→ Point（重申观点）

## 来源
Johns Hopkins Carey Business School Presentation Skills Workbook（2014, Storey）

## 适用场景
- 演讲中的论点展开
- 商务会议中的观点陈述
- 谈判中的主张表达
- 电梯演讲

## 教学优先级：P0
比SCQA更适合中国企业学员初学结构化表达：
1. 更短（4步 vs 4步/但PREP更线性直觉）
2. 更贴近中国人「先结论后说明」的思维习惯
3. 更容易记住（P-R-E-P首字母组合）

## 与R.I.S.E.框架的关系
S层（Speak）的「结构化演讲模块」使用PREP入门，SCQA进阶
I层（Interact）谈判模块复用PREP作为论点展开工具

### P0级案例识别

**P0案例 #1：PREP模型**
- **为什么是P0：** 极简、好记、好用。4步就让一个不会演讲的人有框架可依。比任何复杂演讲模型都更适合中国企业学员。
- **使用方法：** S层第一节课——用PREP教每个人做「1分钟自我介绍」。课前练1遍、课中教模型后再练1遍，对比效果。

**P0案例 #2：15维评分量规**
- **为什么是P0：** 培训验收的杀手锏——课前测→课后测→雷达图对比。HR最缺的就是「培训效果可视化」的工具。
- **使用方法：** 每个演讲培训项目都附赠课前/课后评估各一次。课前发（学员自评+教练评估）、课后发（重评）、输出雷达图对比PPT。

**P1案例 #3：7-38-55法则**
- **为什么是P1：** 认知开关级别——知道这个数据后学员对演讲的认知彻底改变。且极好分享（看到就忍不住发朋友圈）。
- **使用方法：** R层第一节课的开场——用数据打脸「你以为的内容为王」。
