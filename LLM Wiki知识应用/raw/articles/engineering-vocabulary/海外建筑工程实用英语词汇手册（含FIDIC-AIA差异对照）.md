# 海外建筑工程实用英语词汇手册（含FIDIC/AIA差异对照）

### 前言

在海外工程项目中，对建筑工程英语词汇的不理解和错误使用直接关系到合同履约、现场沟通、索赔维权等环节。本文涵盖了常用的建筑工程英语词汇，并对FIDIC（国际咨询工程师联合会）与 AIA（美国建筑师学会）两大合同体系下的英语词汇差异做了比较。希望能给大家提供帮助！

-

## 一、核心词汇分类汇总（中英对照 + 使用贴士）

### （一）合同商务类

| 英文词汇 | 中文释义 | 使用贴士 |
| --- | --- | --- |
| Tender / Bid | 投标（英式 / 美式） | 同一份标书不可混用，英联邦用 Tender，欧美用 Bid |
| Tender Document / Bidding Document | 招标文件 | 规范写法，避免写成 Bid Document |
| Bidder / Tenderer | 投标人 | 投标阶段专用，不可混淆 Supplier（供应商）、Contractor（承包商） |
| Employer（FIDIC）/ Owner（AIA） | 雇主 / 业主 | 两大体系核心称谓，详见差异对照表 |
| Engineer（FIDIC） | 业主方工程师 / 监理 | 非普通工程师，代表业主管理合同、签发指令 |
| EPC | 设计 - 采购 - 施工总承包 | 模式称谓，常与 LSTK（固定总价交钥匙）搭配使用 |
| BOQ (Bill of Quantities) | 工程量清单 | 海外标书核心文件，缩写必须大写 |
| Lump Sum | 固定总价 | 中标后除变更外不调价，风险较高需谨慎 |
| Unit Price | 单价 | 写错可能导致废标或巨额亏损，需反复核对 |
| Provisional Sum | 暂列金额 | 业主指定用途，承包商不可随意使用 |
| Contingency Sum | 不可预见费 | 承包商风险备用金，与暂列金额不可混用 |
| Validity of Bid | 投标有效期 | 过期自动失效，延长需书面确认 |
| Variation Order (VO) | 变更令 | 无 VO 的施工，业主有权拒绝付款 |
| Defect Liability Period (DLP) | 缺陷责任期 | 区别于 Warranty（保修期），侧重工程照管责任 |
| Bid Bond / Tender Guarantee | 投标保函 | 过期、金额不足直接废标 |
| Performance Bond | 履约保函 | 海外多为无条件见索即付，风险需提前预判 |
| Retention Money | 保留金 / 质保金 | 通常为进度款的 5%-10%，竣工后分期返还 |
| Claim | 索赔 | 合同中专用含义，不可译为 “宣称、要求” |
| Force Majeure | 不可抗力 | 仅含战争、暴乱、自然灾害，疫情 / 断货 / 汇率波动通常不算 |
| Liquidated Damages (LD) | 误期损害赔偿金 | 补偿性质，不可写为 Penalty（罚款） |

## 二、FIDIC 与 AIA 核心词汇差异对照表

| 应用场景 | FIDIC（英美 / 英联邦常用） | AIA（美式常用） | 核心差异说明 |
| --- | --- | --- | --- |
| 业主称谓 | Employer（雇主） | Owner（业主） | FIDIC 指投标函附录中指定的当事人及合法继承人；AIA 指协议中确定的当事人及授权代表 |
| 投标相关 | Tender, Tender Document, Tenderer | Bid, Bidding Document, Bidder | 纯称谓差异，含义一致，需按项目所在区域统一使用 |
| 监理 / 设计职能 | Engineer（业主方工程师） | Architect（建筑师） | FIDIC 中 Engineer 不负责设计，仅代表业主管理合同；AIA 中 Architect 兼具设计与部分监理职能 |
| 业主代表 | Employer’s Representative（仅银皮书有） | Owner’s Representative（书面指定） | AIA 要求业主必须书面指定代表；FIDIC 红 / 黄皮书无此概念 |
| 缺陷责任期 | Defect Liability Period (DLP) | Warranty Period | 表述不同，核心均为工程质量责任期限，DLP 侧重照管责任 |
| 工程量清单 | Bill of Quantities (BOQ) | Quantity Takeoff (QTO) | FIDIC 体系核心文件，AIA 常用 QTO 表述，内容实质一致 |
| 变更令 | Variation Order (VO) | Change Order (CO) | 称谓差异，均指工程变更的正式文件 |
| 暂列金额 | Provisional Sum | Contingent Allowance | 用途一致，均为业主预留的不确定费用 |
| 投标保函 | Tender Guarantee | Bid Bond | 含义完全一致，地域习惯称谓不同 |

## 三、高频易错词汇

1.  **Day 的区分**：Calendar Day（自然日）≠ Working Day（工作日），工期、保函期限必须明确标注，避免亏损；
    
2.  **Claim 的使用**：合同语境下一律译为 “索赔”，不可按字面意思译为 “要求” 或 “宣称”；
    
3.  **EPC 与 LSTK**：EPC 是承包模式（设计 - 采购 - 施工），LSTK 是计价方式（固定总价交钥匙），不可混淆概念；
    
4.  **Provisional Sum vs Contingency Sum**：前者是业主指定用途的暂列金额，后者是承包商自用的不可预见费，混用可能导致费用纠纷；
    
5.  **Submission Deadline**：海外项目对截止时间要求极严格，晚 1 秒即视为逾期拒收，需提前规划递交时间。
