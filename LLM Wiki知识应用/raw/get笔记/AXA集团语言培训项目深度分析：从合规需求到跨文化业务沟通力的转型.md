### **🏢 AXA集团背景与挑战（背景）**

**集团概况**
- **业务范围**：全球领先金融与保险集团，覆盖**保险、健康管理及资产管理**等领域，服务遍及数十个国家和地区。
- **核心优势**：高度国际化，行业创新与规范发展前沿，对专业能力、服务质量及合规管理有极高标准。

**关键挑战**
- **国际化运营需求**：跨国团队协作与国际客户沟通频繁，员工需兼具保险专业知识、**外语能力**及**跨文化沟通意识**。
- **数字化转型压力**：业务数字化与国际协作加深，语言能力成为支撑全球运营与客户服务质量的**关键基础能力**。

### **📋 客户核心需求（需求）**

**行业特性驱动的高标准要求**

| 需求维度 | 具体内容 |
| :--- | :--- |
| **内容相关性** | 需围绕**保险与金融业务**，覆盖客户服务、健康保险、理赔沟通及行业趋势等实际场景，非通用英语 |
| **合规要求** | 符合保险行业监管要求，学习时长需计入**年度强制合规培训体系** |
| **系统集成** | 无缝集成至现有LMS（**Cornerstone OnDemand**），实现统一入口、数据管理与进度追踪 |
| **项目定位** | 不仅是语言培训，更是**人才发展与合规体系深度融合**的能力建设升级 |

### **🔧 解决方案（方案）**

**构建语言×业务×AI的一体化学习体系**
- **系统集成**：与AXA现有LMS深度集成，员工通过统一平台访问Speexx，完成测评、课程学习、一对一辅导及线上小班课程。
- **内容设计**：提供符合**IDD标准**（保险分销指令）的保险行业专属语言课程，涵盖健康保险、保险服务沟通、金融科技趋势等主题。
- **学习方式**：结合**AI智能测评**与个性化学习路径推荐，匹配不同岗位、语言水平员工的学习节奏；通过线上小班课程与一对一辅导，强化真实业务语境中的专业表达。

### **📊 项目成果（成果）**

**量化成效与战略价值**
- **员工参与度**：约**40%** 的员工持续活跃使用平台，人均每月学习时长**3.7小时**，语言学习融入日常工作节奏。
- **合规与人才发展**：IDD合规内容使语言学习正式纳入**年度强制培训体系**，实现能力提升与合规要求双重满足；数据追踪与学习报告为管理层提供决策支持。
- **战略升级**：从传统语言培训升级为**跨文化业务沟通力**建设，强化国际化团队协作能力，支撑全球业务稳健运营。

### **💡 补充细节**
- **IDD标准**：即Insurance Distribution Directive（保险分销指令），欧盟针对保险产品销售的监管框架，要求从业人员具备相应专业能力。
- **Speexx核心优势**：提供英语、法语、德语、意大利语和西班牙语5大语种培训，支持14种界面语言，全球服务1500多家顶尖企业，总部位于德国慕尼黑，在上海等7个城市设有分公司。
- **项目定位**：并非单点项目，而是AXA与Speexx长期共建的**全球化人才体系**一部分，聚焦“用得上、用得久、用得好”的语言解决方案。
