### **🏆 核心价值主张（背景）**

**核心定位**  
- **目标**：将销售谈判从"经验驱动"转化为"系统驱动"，实现业绩可复制。  
- **核心理念**："一流的销售拼天赋，顶尖的销售拼系统"，通过标准化流程将偶然成交转化为必然结果。  

**关键痛点解决**  
| 常见销售难题 | 对应解决方案 |
| :--- | :--- |
| 客户开发触达难 | 系统化信息收集与精准定位 |
| 信任建立难 | 标准化初次接触与案例呈现 |
| 价格争议多 | 价值导向报价与阶梯策略 |
| 决策链复杂 | 需求挖掘与关键人识别 |
| 谈判能力弱 | 异议处理与成交促成技巧 |
| 漏单频出 | 售后跟进与复盘优化机制 |

### **📊 九大核心维度（流程框架）**

**完整谈判流程：从准备到成交**  
1. **客户信息收集**（准备阶段）  
   - **核心动作**：通过官网、LinkedIn、B2B平台搜集公司信息，确定决策人及采购周期。  
   - **输出成果**：客户信息表、决策人清单。  

2. **初次联系与建立信任**（接触阶段）  
   - **实操技巧**：邮件模板（公司+产品价值）、电话开场（问候+痛点发现）、提供成功案例。  
   - **应用流程**：初次联系→提供案例→回答问题→建立信任。  

3. **需求挖掘与价值呈现**（沟通阶段）  
   - **核心方法**：  
     - **SPIN提问法**：情境→难点→暗示→需求。  
     - **FAB法则**：特征（Feature）→优点（Advantage）→利益（Benefit）。  
   - **输出成果**：需求分析表、FAB价值呈现文档。  

4. **报价策略与价格谈判**（谈判阶段）  
   - **关键策略**：强调价值而非价格、拆分成本效益、阶梯报价、"让步交换"原则（如降价需缩短账期）。  
   - **应用流程**：价值呈现→报价策略→价格谈判→达成共识。  

5. **异议处理技巧**（攻坚阶段）  
   - **处理步骤**：识别异议→倾听理解→提供方案→解决顾虑。  
   - **实操工具**：常见异议库（价格/规格/付款方式）、标准回答模板、权威客户佐证。  

6. **成交促成方法**（关键阶段）  
   - **技巧**：分步骤报价（样单→全额订单）、设定明确下一步、制造轻紧迫感。  
   - **流程规范**：报价确认→合同确认→发票开具→收款确认。  

7. **售后跟进与维护**（维护阶段）  
   - **跟进节点**：2天/5天/10天固定跟进，内容递进（产品介绍→疑问解答→报价）。  
   - **长期价值**：收集反馈形成案例库，推荐相关产品或升级方案。  

8. **复盘优化与经验沉淀**（提升阶段）  
   - **核心动作**：每周复盘跟进量/成交率、总结成功/失败案例、团队复盘TOP3障碍。  
   - **输出成果**：团队复盘表、SOP优化方案。  

9. **谈判核心原则**（原则阶段）  
   - **五大原则**：充分准备、发掘客户利益需求、建立客观标准、关键时刻妥协、预留回旋空间。  
   - **成功守则**：写下计划再决策、专注倾听不评论、不被权位吓倒、追求双赢方案。  

### **🔑 关键洞察**
- **流程闭环设计**：从信息收集到复盘优化形成完整循环，确保经验可沉淀、能力可复制。  
- **工具化输出**：每个阶段均明确"输出成果"（如客户信息表、FAB价值文档），降低执行门槛。  
- **价值导向谈判**：通过SPIN提问和FAB法则将价格争议转化为价值共识，避免陷入价格战。
