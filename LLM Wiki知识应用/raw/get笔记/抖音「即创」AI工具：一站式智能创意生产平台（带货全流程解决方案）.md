🔧 **核心功能概览**  
- **AI视频**：一键生成视频脚本（输入抖音商品ID+自定义热点/卖点/风格）、数字人成片（选形象/背景/配音）、智能剪辑（通用/短剧场景）  
- **AI图文**：商品卡工具（生成商品图）、图文工具（生成带货短视频）  
- **AI直播**：直播间装修（生成直播背景/标题）、AI直播脚本（生成带货话术）  
- **其他**：控评检测、投后诊断、投教数据等  

📊 **关键效率提升**  
- 传统手工剪辑需数小时/视频 → 「即创」1小时内完成  
- 脚本生成仅需10秒（输入参数后）  
- 短剧全自动剪辑：上传5集8分钟内片段 → 自动生成剪辑视频  

🎯 **案例场景（以一字拖鞋为例）**  
1. **AI视频脚本**：输入商品ID+“双十一/真皮材质/年轻女性” → 生成口播文案（含价格优惠+痛点解决）  
2. **数字人成片**：选“林小雅”形象+户外背景 → 生成数字人口播视频  
3. **商品卡工具**：输入商品ID+“简约风格” → 生成5张商品图（含价格/卖点标签）  
4. **直播脚本**：输入商品ID+“拍一发三/限时优惠” → 生成直播话术
