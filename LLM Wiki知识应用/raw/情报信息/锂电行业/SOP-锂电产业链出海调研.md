# 锂电产业链出海布局与人才培养机会调研 SOP

## 适用场景
用户提出类似需求：
- "调研XX公司的海外项目及上下游合作伙伴"
- "分析XX产业链出海布局与人才培养机会"
- "调查XX行业海外项目的人才匹配情况"

## 一、输入信息

用户需提供（缺少的主动追问）：
1. **目标企业/项目**：公司全称 + 具体海外项目名称/地点
2. **调研范围**：上下游多少家？是否有指定企业？
3. **重点关注**：人才培训？商业机会？合作伙伴？还是全都要？
4. **输出格式**：Markdown / HTML网页 / PDF（默认Markdown+HTML）

## 二、调研流程（6步串行，可内部并行搜索）

### Step 1：核心项目全景扫描
**目标**：搞清楚项目本身的基本面

搜索内容：
- 项目投资规模、选址、产能规划、建设进度、投产时间
- 合资/独资模式、股权结构
- 上游供应商（原材料、设备、能源）
- 下游客户（已签约/意向）
- 政策支持（政府补贴、产业政策）
- 风险点（地缘政治、合规、坏账等）

搜索关键词：
- "公司名 + 海外项目名/国家 + 投资/产能/进度"
- "公司名 + 供应商/客户/战略合作 + 海外"
- "公司名 + 年报/公告 + 海外项目"
- "国家名 + 中资企业 + 政策/补贴"

信息标注：[L1-权威源] / [L2-已验证] / [L3-逻辑推断]

### Step 2：上下游合作伙伴海外布局
**目标**：识别每个合作伙伴的海外项目

对Step 1识别出的每个上下游企业，搜索：
- 海外生产基地/项目（国家、投资、产能、进度）
- 海外资源布局（矿权、资源锁定）
- 海外客户合作（是否进入主流供应链）
- 海外认证与合规

搜索关键词：
- "公司名 + 海外项目/海外基地/海外工厂"
- "公司名 + 出海/国际化/海外布局"
- "公司名 + 印尼/欧洲/北美/非洲 + 项目"

### Step 3：企业培养体系与涉外培训
**目标**：摸清每家企业的培训家底

对每个合作伙伴，搜索：
- 培养体系名称与架构（层级、项目名）
- 各培养项目详情（对象、内容、周期、规模）
- 海外/涉外培养专项
- 英语/小语种培训项目
- 培训数据（年报/ESG披露的人次、时长、投入）
- HR联系方式

搜索关键词：
- "公司名 + 培养体系/人才培养/培训体系"
- "公司名 + 培训学院/管培生/人才计划"
- "公司名 + 英语培训/涉外培训/国际化人才"
- "公司名 + ESG/年报 + 员工培训"
- "公司名 + HR电话/招聘热线/联系方式"

### Step 4：培训趋势与海外项目关联分析
**目标**：找到培训与出海之间的因果链条

分析维度：
- **投入趋势**：近年培训数据变化（2022-2025）
- **内容趋势**：通用→专业→国际化的演变
- **模式趋势**：自建→政企校合作→跨国联合培养
- **对象趋势**：本土→外派→海外本地员工
- **时间关联**：培训是否先行于海外项目1-3年
- **内容关联**：海外项目催生的合规/跨文化/语言培训需求
- **效果关联**：本地化率、适应期、合规改善等

搜索关键词：
- "公司名 + 海外项目 + 人才准备/人才储备"
- "中国锂电企业 + 出海 + 人才培养趋势"
- "中国企业出海 + 培训/合规/跨文化"

### Step 5：海外项目与人才匹配分析
**目标**：找缺口——需求侧 vs 供给侧

**需求侧**：每个海外项目的人才需求画像
- 关键岗位需求（技术/管理/合规/本地化运营）
- 项目阶段与人才需求节奏（筹备→建设→试产→量产）
- 本地化率法规要求 vs 实际水平

**供给侧**：每家企业的人才供给能力
- 涉外人才储备（海外经验、小语种）
- 培训供给能力（体系覆盖、培训周期匹配度）
- 人才获取渠道

**匹配度评估**：
- 🟢 高匹配：供给跟得上
- 🟡 中匹配：部分缺口可补
- 🔴 低匹配：关键岗位严重缺人

**四类缺口**：
- 时间缺口：项目快于人才培养
- 能力缺口：有技术缺合规/跨文化
- 规模缺口：数量不足
- 结构缺口：人员结构不合理

### Step 6：前置化机会点与行动建议
**目标**：从缺口中识别商业机会

六大机会点评估：
1. **猎头/人才中介**（高端岗位奇缺）
2. **合规培训**（欧盟电池法等刚性需求）
3. **跨文化管理培训**（普遍痛点）
4. **产教融合服务**（高校联合培养）
5. **小语种培训**（芬兰语/匈牙利语/葡萄牙语）
6. **数字化培训产品**（VR/AI）

每个机会评估：需求紧迫度 / 市场规模 / 竞争格局 / 进入壁垒 / 付费意愿

输出机会优先级矩阵 + 行动路线图 + 时间窗口判断

## 三、输出规范

### 报告结构
1. 第一章：核心项目全景
2. 第二章：上下游合作伙伴海外布局
3. 第三章：企业培养体系与涉外培训
4. 第四章：培训趋势与海外项目关联
5. 第五章：海外项目与人才匹配分析
6. 第六章：前置化机会点与行动建议
7. 附录：信息源清单、HR联系方式、风险声明

### 每章开头
- 本章要点（3-5条核心结论）

### 信息标注
- [L1-权威源]：官方公告、年报、ESG报告、工商数据
- [L2-已验证]：多源交叉验证的媒体/行业报道
- [L3-逻辑推断]：基于数据的合理推理
- [数据未获取]：搜索未找到，标注原因

### 质量红线
- 严禁杜撰，找不到写"未检索到公开信息"
- 每条实质性信息必须有来源
- 区分事实与推测
- 优先一手源（官网、年报、ESG）

## 四、执行方式

一次性启动，用 sessions_spawn 分步执行：
1. Step 1-3 可并行（各自独立搜索）
2. Step 4 依赖 1-3 的结果（需串行）
3. Step 5-6 依赖前序所有结果（需串行）
4. 最后整合为完整报告 + 生成HTML网页版

## 五、文件命名规范

- Markdown报告：`{主题}-调研报告.md`
- HTML网页版：`{主题}-调研报告.html`
- 整合版：`{主题}-研究报告.md` + `.html`

## 六、耗时参考

- 10家企业的完整调研：约30-40分钟（含搜索+分析+撰写）
- HTML网页生成：约5-10分钟
- 总计：约40-50分钟

## 七、变体场景

| 场景 | 调整点 |
|------|--------|
| 只看培训不看机会 | 跳过Step 6，Step 5简化 |
| 只看某几家企业 | 缩小Step 2-5的范围 |
| 非锂电行业 | 替换行业关键词，框架不变 |
| 快速版 | 每步搜索减半，输出精简版 |
