✨ **核心功能与效果**
- **生成能力**：Gemini3基于参考图举一反三，生成可定制的小红书卡片生成器
- **风格多样性**：支持冲击波、弥散光、贴纸风、手账感、电影感、科技蓝、极简白、备忘录、极客黑等多种风格
- **自定义选项**：可调整标题/正文字号、文字/背景颜色、上传背景图、标题换行、账号名称、正文自动分割等
- **一键下载**：支持打包下载多张生成的图片

🔧 **关键技术解析**
- **html2canvas**：通过遍历DOM、解析样式、Canvas画布重绘，将网页内容转换为PNG图片
- **预计算分页逻辑**：采用DOM离屏渲染技术，创建隐形容器预演内容高度，实现自动分页（Page1/Page2/...）
- **技术栈**：Tailwind CSS（样式）、html2canvas（截图）、JSZip + FileSaver.js（打包下载）、Lucide Icons（图标）

📝 **极简提示词模板**
```
# 角色：视觉克隆工程师  
## 核心流程  
分析参考图样式 + 处理文本内容 → 生成单文件HTML应用  
## 核心技术要求  
视觉复刻：CSS必须复刻参考图的色彩、布局、字体、装饰  
数据驱动：所有内容必须从 defaultData JSON对象读取，禁止硬编码  
智能分页：内容过长时自动分页，避免文字溢出  
打包下载：顶部固定工具栏含“下载ZIP”按钮  
强制技术栈：必须引入  
Tailwind CSS (样式)  
html2canvas (页面截图)  
JSZip + FileSaver.js (打包下载)  
Lucide Icons (图标)  
## 默认行为  
无参考图时，使用预设的现代流式排版风格
```
💡 **内容洞察**
- **AI效率革命**：替代传统HTML手动调试，实现“输入文字+选风格=生成图片”的高效流程
- **技术范式创新**：预计算分页逻辑采用“空盒子演练装箱”思路，确保内容不溢出
- **风格适配性**：覆盖小红书主流爆款封面格式，适配不同账号定位需求
