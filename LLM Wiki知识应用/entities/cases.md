## 1. 当升科技 — 芬兰外派人员持续赋能建议

| 字段 | 内容 |
|------|------|
| **客户名称** | 当升科技 |
| **需求背景** | 中方管理人员及蓝领工人外派芬兰，面临芬兰高管英语口音听力障碍（听懂率仅60-80%）、Small Talk困难、社交回避、SOP培训方式差异（中国现场讲解 vs 芬兰文件培训）、文化冲突（中方委婉 vs 芬兰直接）等挑战；劳动法合规风险与量产安全风险并存。 |
| **方案亮点** | 四大核心支柱设计：① **文化融合** — 产线沟通红绿灯规则、工作场景商务话术库、三级关系建立（L1安全执行→L2主动反馈→L3共建改善）；② **SOP场景沟通** — 术语打卡系统 + 指令听力训练 + 线下跟岗陪跑；③ **工具赋能** — Teams会议翻译插件、芬兰口音模拟器、微信群陪跑机器人；④ **TTT内训师**（备选）。分层设计管理/蓝领两套方案，晚饭后轻量利用（30-40分钟/次，3次/周）。 |
| **匹配课程方向** | 跨文化融合 / 产线场景英语 / 工具赋能陪跑 / 外语+技术场景实训 |

**关键数据：** 8位学员调研；听力正确率目标从65%提升至≥85%；翻译插件使用率>80%；每周参与率≥70%；管理人员L2达标率≥60%、蓝领L1达标率≥80%；沟通误解导致停线占比<5%；员工参与Fika目标≥50%。

## 2. 中国中原对外工程有限公司 — 国际市场开发专业能力提升培训

| 字段 | 内容 |
|------|------|
| **客户名称** | 中国中原对外工程有限公司 |
| **需求背景** | 市场序列人员（专员/主管/主任级混合）国际市场开发专业能力薄弱，测评暴露8条短板：信息搜集弱、市场分析弱、战略规划弱、项目策划弱、沟通欠缺、竞争意识弱、应变不足、跨部门协作弱、合同与投标风险意识弱。需系统提升海外市场开拓能力。 |
| **方案亮点** | 3天高密度实战培训：**Day1** 宏观战略与出海路径规划（地缘政治→STP→四种出海模式→竞争战略），**Day2** 市场情报与项目策划（六大信息渠道、价值构想书、FIDIC合同+EPC全过程管理、联合体与分包商），**Day3** 区域市场落地（中东/中亚/东南亚可选，含伊斯兰商业文化、Wasta信任建立）+内部资源协调。每日≥3个实战环节，总实战时长超12小时；晚间结构化分享（路演+互评）。配备5位实战派师资（刘德冰、马永强、刘松、宋广森、石悦华），产出可直接使用的工具模板。 |
| **匹配课程方向** | 出海战略规划 / 国际市场情报分析 / 国际工程投标与合同管理 / 跨文化商务谈判 / 区域国别市场准入 |

**关键数据：** 培训对象为市场序列人员；3天（6h/天教学 + 1.5h/晚结构化分享）；培训时间2026年6月；师资5人；产出含出海战略框架模板、价值构想书模板、FIDIC速查手册、中东商务礼仪清单等工具包；培训后1个月安排线上复盘会。

## 3. 某央企 — 机关管理人员英语能力提升培训（通知版）

| 字段 | 内容 |
|------|------|
| **客户名称** | 某央企（集团公司机关） |
| **需求背景** | 贯彻落实人才强企工程部署，机关管理人员涉外工作英语应用能力不足，需覆盖商务沟通、商务接待、外事协调等场景，培养懂业务、善沟通、具有国际视野的管理人才。2026年培训计划安排。 |
| **方案亮点** | **线上+线下双线闭环模式**：线上提供7000+节微课覆盖八大模块（通用英语、涉外商务沟通/演讲/谈判、利益相关方沟通、技术英语、跨文化沟通、国际视野）+每人40节欧美母语外教1对1（25分钟/节）+约6小时直播讲座；线下6天集中面授（小班20人，情景模拟+录像复盘+角色扮演），涵盖自我介绍、邮件写作、演讲训练、会议技巧、外事接待、跨文化沟通等。CEFR等级提升至少一个子级为结业参考条件。 |
| **匹配课程方向** | 机关管理干部英语分层培训 / 涉外商务英语 / 外事接待英语 / 1对1外教陪练 |

**关键数据：** 计划30-40人；2026年6月至12月共7个月；CET-4及以上；线下6天(36小时)；40节外教/人；需CEFR提升至少一个子级；线上1对1不足40次或线下出勤率低于80%不予结业。

|------|------|

| **方案亮点** | **三大模块设计**：① **英语强化** — 聚焦语言流利度与商务应用，模拟国际商务谈判、项目会议、客户接待等真实场景，外籍教师授课；② **思想武装** — 党性修养与文化素养；③ **辅助教学** — 团队协作与成果输出。考核严格，需达到集团公司出国人员英语水平考试通过标准才能进入后续培训。30天全脱产集训。 |
| **匹配课程方向** | 青年干部国际化培养 / 英语强化集训 / 商务情景模拟 / 外教授课+思想武装融合 |

**关键数据：** 培训30人；2025年8月7日开班，共30天；CET-4/托福500+；学员年龄≤38岁（管理）/≤35岁（技术）；考核含英语水平考试（30分）、商务写作（25分）、口语测试（25分）、理论学习（10分）、考勤（10分）。

| **匹配课程方向** | 机关管理干部英语分层培训 / 线上线下混合式英语培训 / 外事接待与跨文化沟通 / 情景模拟实战演练 |

**关键数据：** 2026年4月至12月共8个月；线下6天(36小时)；40次外教/人(25分钟/节)；CEFR等级提升至少一个子级；课程覆盖A1-C2全级别；考核满分100分，线上1对1不足40次或线下出勤率低于80%不予结业。
