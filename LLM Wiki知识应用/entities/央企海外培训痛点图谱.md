## 三、踩坑驱动付费意愿模型

### 核心规律
**发生合规处罚/劳动纠纷/安全事故的项目，培训付费意愿提升3-5倍，采购决策周期缩短60%。**

### 踩坑类型→培训机会映射

| 踩坑类型 | 典型事件 | 培训机会 | 付费意愿 |
|---------|---------|---------|:------:|
| **合规处罚** | 巴拿马四桥外籍劳工超限被罚 | 劳工法合规培训 | ★★★★★ |
| **政治风险** | 尼日尔政变驱逐中石油高管 | 政治风险识别+防恐培训 | ★★★★★ |
| **HSE事故** | Nexen Long Lake泄漏500万升 | HSE体系培训 | ★★★★★ |
| **合同纠纷** | 老挝EdL拖欠电费/新加坡行贿案 | FIDIC合同+FCPA | ★★★★☆ |
| **劳资冲突** | 巴拿马工会罢工/越南工人冲突 | 工会关系+属地管理 | ★★★★☆ |
| **标准冲突** | 阿尔及利亚标准对接延误 | 国际标准转换 | ★★★☆☆ |

### 客户信号识别规则
诊断对话中，若客户提及以下关键词，标注为"踩坑驱动型"：
- "被查""被罚""处罚""违规""超限"
- "罢工""纠纷""冲突""抗议"
- "事故""泄漏""爆炸""伤亡"
- "损失""赔偿""仲裁""诉讼"

**踩坑驱动型客户的方案策略**：先止血（合规/安全培训）→ 再建体系（系统化培训）。

## 四、项目生命周期→培训需求模型

### 基建项目（7阶段）

```
规划可研 → 设计 → 融资签约 → 施工准备 → 施工高峰 → 调试验收 → 运营维护
  (低)      (低)     (低)       (中)       (高)        (高)      (极高)
```

| 阶段 | 培训窗口 | 典型需求 |
|------|:------:|---------|
| 施工准备 | 提前3-6月 | 属地劳工法规、HSE、供应链管理 |
| 施工高峰 | 持续 | 属地工人技能认证、语言培训、大型设备操作 |
| 调试验收 | 提前6-12月 | 设备调试、验收标准、运维SOP |
| **运营维护** | **提前12-18月** | **属地运营人才全链条培训（量最大、单价最高）** |

**关键判断**：运营期培训需求是施工期的3-5倍。2026-2027年50+重大项目从施工转入运营，是密集需求窗口。

### 油气项目（6阶段）

```
勘探 → 开发建设 → 投产运营 → 产量递减/退出
(低)     (中)        (高)       (中)
```
