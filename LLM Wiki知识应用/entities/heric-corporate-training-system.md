# Heric — 企业培训体系

## 核心观点

企业培训体系是企业人才发展的核心引擎。通过对丹纳赫DBS沉浸计划、方太集团"仁"才培养体系、赛诺菲数字化学习转型等实践的梳理，提炼以下核心观点：

1. **培训的本质是促进行为改变**，其深层动力源于思维模式的转型。培训的核心价值应聚焦于帮助学员构建系统、高效的决策思维体系。

2. **培训中"不变"的三要素**：需求为先（从真实需求出发）、内容为王（扎实的内容是学习的主体）、不着其相（形式服务于内容与目的，不被花哨形式迷惑）。

3. **AI时代培训从业者的核心职能正在从"内容制作"转向"学习体验设计与效果验证"**，未来培训从业者需要向"顾问式培训师"转型。

4. **优秀的培训体系一定有"一把手"重视**：方太董事长"教育三问三答"明确教育是文化建设最重要、最不能舍弃的事情。

5. **AIGC时代隐性经验数字化沉淀成为关键**：将"老师傅经验"转化为可复用、可持续的数字化资产，是AI时代企业培训的新方向。

## 框架模型

### 1. 丹纳赫DBS沉浸计划（70-20-10学习发展模型）

**核心机制**：新员工入职前60-90天不安排"本职工作"，专注深度融入企业文化

**70-20-10模型**：
- **70%** 挑战性的在职任务（现场改善活动、标杆学习、客户拜访）
- **20%** 他人合作与洞察（导师制、DBS实践标杆网络）
- **10%** 正式和虚拟课堂学习（DBS工具学习、问题解决流程）

**黄金法则**：DBS沉浸式学习 + 优秀导师 = DBS领导力

**四大核心要素**：
| 要素 | 内容 |
|------|------|
| 全身心投入 | 通过实践体验学习，加快DBS"顿悟时刻" |
| 个性化发展 | 定制化、差异化的发展路线图和课程 |
| 精准辅导 | 领导者作为导师和教练，确保核心内容传递 |
| 人脉建设 | 建立内部人际网络，快速适应并找到归属感 |

### 2. 方太"仁"才培养六大原则

| 原则 | 核心内涵 |
|------|---------|
| 密联战略 | 将公司战略主题分解为组织能力和岗位能力，匹配解决方案 |
| 双能提升 | 既提升能力（个人/组织），也提升能量（个人/团队） |
| 双线发展 | 人才发展线（自下而上）+ 组织发展线（自上而下） |
| 训战一体 | 仿真场景训练+作战项目锻炼，战训结合 |
| 知识经营 | 从人力资源管理到知识资源经营，知识是决策的根本 |
| 能者为师 | 用优秀的人培养更优秀的人（1000+师资队伍） |

### 3. 方太干部培养六步法

```
标准构建 → 人才盘点 → 发展赋能 → 干部任用 → 结果评价 → 激励保留
```

**标准构建**：
- 岗位画像（用于选拔招聘）：基于真实业务特性与挑战，每条能力要求支撑到业务子维度
- 关键行为模型（用于人才培养）：岗位使命→关键职责→关键任务→关键能力

**人才盘点**：铁三角（业务部门+人文体系+方太学校）共同完成
- 人员盘点：人文体系牵头
- 能力盘点：方太学校牵头

### 4. 培训的"变与不变"框架

| 不变的（本质） | 变化的（形式） |
|--------------|--------------|
| 需求为先 | 培训渠道（线下→线上→混合） |
| 内容为王 | 交付方式（讲授→工作坊→AI互动） |
| 不着其相（形式服务于目的） | 技术工具（PPT→学习平台→AI+数字人） |
| 行为改变是终极目标 | 评估手段（问卷→数据看板→A/B测试） |

### 5. 赛诺菲数字化学习转型框架

**三大内容板块**：
1. **计划学习**：考试互动、人机互动、答题互动
2. **生成式AI互动**：工作场景模拟模型
3. **智能测评**：发现差距→生成补习计划→推送员工

**转型关键**：
- 将线下内容转为线上视频课程+AI互动
- 成本控制到线下的80%
- 人工培训师转型为顾问式培训师：诊断问题、判断AI方案、组织共创活动

### 6. 培训效果验证方法（AI时代）

| 方法 | 具体操作 |
|------|---------|
| AI陪练评估 | 设定学习目标与评估标准，系统生成融合练习与测评的交互程序 |
| A/B测试 | 对比实验组与对照组的业务表现差异 |
| 相关性分析 | 分析培训成绩与业绩的相关性 |
| 智能测评对比 | AI对比实际表现与标准模型差距，生成针对性补习方案 |

## 可复用方法论

### 方法论一：新员工沉浸式入职设计

参考丹纳赫DBS沉浸计划：

**第一阶段（第1-30天）：文化浸润**
- 了解企业整体情况、文化、价值观、核心行为
- 学习核心方法论/管理工具（如DBS）
- 撰写沉浸日志并进行反思总结

**第二阶段（第31-60天）：深入一线**
- 深入现场了解业务、产品、客户和市场
- 参加与本职无关的会议（全面感知）
- 拜访客户、参观工厂和设施
- 学习标杆和最佳实践经验

**第三阶段（第61-90天）：关系与规划**
- 建立内部人际网络（导师、标杆、利益相关者）
- 了解团队成员
- 明确工作优先级，制订行动计划
- 确保能带着组织视角回归岗位

### 方法论二：培训需求诊断"剥洋葱法"

1. **业务对话**：走进业务场景，听懂业务语言
2. **还原场景**：像内部顾问一样，层层还原实际工作场景
3. **根因分析**：从表象问题（"执行力不够""成长太慢"）追溯到真正的根源
4. **需求确认**：确认"培训是不是真的能解决这个问题？"

**关键原则**：无需求，不培训。培训是一场投入，必须落在真实需求上。

### 方法论三：AI时代的培训内容生产模式

| 传统模式 | AI时代模式 |
|---------|-----------|
| 需求调研→课程开发（数周至数月） | 业务方直接运用AI寻找解决方案 |
| ADDIE系统化模型 | 输入核心知识点→AI自动生成70-80%可用度课程原型 |
| 专家集中授课 | 数字人讲师+AI陪练+碎片化微课 |
| 人工评估反馈 | 智能测评+自动分析薄弱点+针对性推送 |
| 依赖外部机构制作课程 | 内部团队自主完成高质量课程开发 |

### 方法论四：岗位画像与能力模型构建

以方太实践为例：

1. **岗位画像构建**：
   - 梳理业务特性与挑战
   - 每条能力要求支撑到业务子维度
   - 不同岗位在关键任务和关键能力上差异化设置

2. **关键行为模型构建**：
   - 岗位使命 → 关键职责 → 关键任务 → 关键能力
   - 从内到外，对人才培养起牵引作用

3. **赋能框架**：
   - 战略主题 → 组织能力 → 岗位能力 → 解决方案
   - 确保培训密联战略

### 方法论五：数字化学习成本效益模型

**投入**：供应商服务 + 运营支持
**产出**：相比传统线下模式成本降至80%

**人效提升**：
- 传统：7-8名培训老师教一门课程
- 数字化：1位运营 + 1位培训老师

**效果增强**：
- 碎片化短视频（30秒）+ 即时互动练习 > 传统45分钟课堂
- AI工具可模拟线下课堂并有更好的练习效果
