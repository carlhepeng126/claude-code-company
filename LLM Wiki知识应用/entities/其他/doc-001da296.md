## 背景
本流程源自两次实战萃取经验：第一轮跨文化管理案例（13案例/3502行），第二轮中国对外投资项目案例（7案例/5239行）+ 国家风险评级报告（52110行）。适用于任何素材（书籍/PDF/文章/报告）→ 情报虾7大方向洞察 + 爆红虾小红书帖子 + 知识库补充 + 培训应用的完整链路。

## 核心内容

### 总纲：五层萃取法
```
素材 → ①打标分桶 → ②情报虾7方向 → ③爆红虾帖子 → ④知识库 → ⑤培训应用
```
每层必须回答：**Carl能用这个做什么？**（设计课程/赢单/服务客户/做决策）

### Phase 0：前置判断
区分两种模式：
- **快速感知（先读后问）**：叙事型（传记/小说/商业故事）→ 通读给摘要
- **全链路萃取**：知识型（报告/案例集/白皮书）→ 直接走五层完整产出

### Phase 1：素材类型识别+结构扫描
判断文件类型（书籍型叙事/案例集/报告型/制度型）+ 格式（PDF文字/扫描件/MD/TXT）+ 数据结构（目录/页码/附录）

### Phase 2：素材打标分桶（最关键的中转步骤）
逐章逐节阅读，按7大方向分桶：企业外语培训、跨文化、属地化、涉外演讲与谈判、海外项目、央企动态、行业趋势。每个桶至少3条素材，每桶不足3条说明方向不敏感。

**阅读策略**：
- 大文件先读目录定位核心章节
- 案例型只读前1/3（背景+关键数据）+ 最后的启示/总结
- 数据密集型用并行定向采样
- 用todo系统记录阅读进度

### Phase 3-5：逐层产出
- **情报虾7方向**：每条洞察含[素材→Carl业务→具体行动]三层
- **爆红虾帖子**：面向不同受众（培训负责人/HRD/从业者）的小红书风格内容
- **知识库+培训应用**：沉淀到知识库并产出可执行的培训产品方案
