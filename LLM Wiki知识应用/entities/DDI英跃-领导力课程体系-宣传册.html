P02

核心方法论：行为模式法 + 四步学习闭环

行为模式法（Behavioral Modeling Method）

DDI将领导力领域中一系列能力一一拆解成可转化为实践的"关键行动"，让学员掌握展现某项领导能力所需要展现的关键行为和步骤。这是培训内容中绝对不可缺少的让学习能够"落地"的关键。过去五十年，DDI已通过诸多实际案例证明了抓住关键行动路径的有效性。

四步学习闭环

每门课程遵循同一教学设计逻辑：诊断 → 学习 → 模拟 → 考核，形成完整的行为改善闭环。

业务管理 · 拿结果的能力

Business Management — 5门核心课程

P

计划与组织

"计划从容不迫，行动才能游刃有余"
核心交付：一整套制定计划、执行计划的步骤和技巧
课程要点：厘清任务核心目的 → 轻重缓急矩阵定优先级 → 任务细分与资源清单 → 内外部资源协调 → 专注力与双指标追踪 → 5项关键行动
情景模拟：海外事业部的挑战 — 任务重重，状况不断，如何从容应对？
D

决策

"一个错误的决策，100个行动也无法挽救"
核心交付：完整实用的决策步骤和技巧，帮助高效自信地做出业务决策
课程要点：辨识四种决策盲点 → 高效决策五步骤 → 集思广益拟定可选方案 → 系统分析利弊找最佳解 → 沟通实施决策 → 7大关键行动
适合人群："纠结"的决策者 / "焦虑"的决策者 / "小白"的决策者
Pf

达至最佳绩效

"绩效管理很关键，但做绩效管理很糟心"
核心交付：绩效三模块体系梳理 — 不要完美的规划，只要可见的绩效
课程要点：绩效管理本质理解 → 公司愿景与个人绩效关系 → 定目标（关键结果领域+SMART）→ 定能力 → 个人发展计划（学习·应用·反馈）→ 绩效计划讨论达成共识
Ex

执行力 NEW

"没有执行力，再好的战略也无法实现 — 企业成功5%在战略，95%在执行"
核心交付：执行力三大实战要领 — 聚焦战略，明确分工，有效衡量
课程要点：保持战略一致性的"硬件"+"软件" → 聚焦"灵魂三问" → 时间管理技巧 → 进度指标vs成果指标 → 识别关键成功要素 → 选对人+追踪问责+复盘四部曲
In

推动创新 NEW

"创新不仅是一种思维，更是一项技能"
核心交付：成为团队创新的孵化者，让创新从单打独斗变成协同作战
课程要点：心（建立创新观念·四步培养好奇心）→ 脑（挑战思维定式·四种方式转变思维）→ 眼（识别机会·发散→收敛）→ 手（创新方案·小步推进）→ 脚（走出去·影响利益相关人·五大策略）→ 五大关键行动

团队管理 · 带队伍的能力（上）

Team Management — 8门核心课程 · 第1-4门

TS

目标选才

"求职者越来越会'面试'，做面试官，没技巧怎么行？"
核心交付：系统学习目标选才，掌握面试前、中、后的技能，拥有"火眼金睛"
课程要点：选才目标定义 → 面试分工表+准备步骤 → STAR行为事例提问与追问 → 动力适配性（Motivational Fit）→ 控制面试气场
核心工具：行为事例（STAR）提问法、动力适配性评估
Co

辅导

"卓越的管理者，是下属的教练兼老板，而非团队管理上的'消防员'"
核心交付：识别问题和机会，学会高效指导下属的解题思路
课程要点：三种辅导类型认知 → 识别真正需要辅导的问题（找根本原因）→ 轻度辅导：日常反馈三原则 → 完整辅导：沟通五步法 → 两大讨论技巧（防跑偏+理解对齐）
适合人群：初带团队新经理 / 有心无力的团队管理者 / 下属状况百出的经理人
De

授权

"授权，还是不授权？真是一个两难的选择"
核心交付：习得授权的要点和关键技巧，助管理者腾出时间专注于更有意义的职责
课程要点：五种任务分派方式 → 选人做事四维度 → 五方面授权信息交代 → 互动流程+五大人际基本原则进行授权沟通 → 授权跟进五要点 → 四项关键行动
情景模拟：未来智播间 — 麾下人才济济，谁是合适人选？
HT

打造高效团队

"将领的作用不是能否冲锋陷阵，而是在黑暗中用微光带领团队前进"
核心交付：了解高效团队五大成功要素，找准问题切入团队管理
课程要点：团队领导者三大挑战 → 成功五要素：流程（角色职责+章程）→ 结果（任务优先级）→ 沟通（信息交换+有效互动）→ 承诺（合作意识+行动）→ 信任（信任方程式四因素）→ 虚拟团队最佳实践 → 六项关键行为
适合人群：带团队的管理者 / 新团队组建人 / 刚上任的新经理
情景模拟：火星移民计划 — 身处外太空，团队挑战如何解决？

团队管理 · 带队伍的能力（下）

Team Management — 8门核心课程 · 第5-8门

Ch

驱动变革

"为什么70%的变革计划以失败告终？变革真正的成功来自人们行为模式上的改变" — John P. Kotter
核心交付：综合运用3个变革加速器，化解抵触情绪，成功实现加速变革
课程要点：变革反应四阶段（抗拒→观察→适应→拥抱）→ 个人变革章程 → 加速器一：说清楚（什么在变+为什么变+什么收益）→ 加速器二：解除情绪警报赢得认同 → 加速器三：掌控进程加速变革
适合人群：即将负责变革者 / 正处于变革中 / 积极迎战变革者
MR

激励与保留人才 NEW

"留任人才的关键管理法则 — 流失一个人才的损失是培养新员工成本的6倍"
核心交付：三大关键因子，提前识别风险，提升员工敬业度
课程要点：激励的定义与正面影响 → 日常激励的"一颦一笑、一言一行、一举一动" → 驱动因子一：有意义的工作（目的关联+信息分享+充分授权）→ 驱动因子二：积极的工作环境（信任+尊重+合作机制）→ 驱动因子三：个人价值感（认可+成长发展）→ 激励因素谈话 → 识别并留住人才
适合人群：中基层管理者 / 人力资源伙伴
VT

领导虚拟团队 NEW

"数智时代团队效能加速器 — 时间或空间的阻隔，是否成为你带队冲锋的阻碍？"
核心交付：领导远程协作团队技巧，让团队在缺少面对面互动的情况下依然高效运作
课程要点：领导虚拟团队挑战与四大招式 → 创造协作环境（团队工具+章程+远程能力培养）→ 远程沟通精要（打破参与壁垒+非正式沟通）→ 确保高产出（短周期目标+跟踪复盘四要点）→ 关注成员身心状态（一对一谈话+针对性支持）→ 四大关键行动
DT

培养人才 NEW

"为人才创造茁壮成长的良好环境，为团队建立生生不息的人才土壤"
核心交付：计划性地协助团队成员在知识、技术和能力等方面持续发展
课程要点：培养与辅导的差异 → 发展3A流程 → 评估（明确发展目标：高回报机会+一个强项一个待发展+以用为始）→ 学习（多样化方式+一鱼多吃+扫清障碍+评估成效）→ 应用（循序渐进计划+监控衡量+对计划外机会开放）

人际管理 · 横向影响力

Interpersonal Management — 3门核心课程

Co

协作

"VUCA环境下，建立长期稳定的协作并非易事"
核心交付：掌握伙伴关系建立的成功五要素，破解协作困局，取得双赢
课程要点：协作与伙伴关系定义 → 伙伴关系成功五要素 → 打破"本位主义"障碍（OBBSAM口诀上）→ 让伙伴凝成一股绳（OBBSAM口诀下）→ 用基本原则维护正面气氛 → 五大关键行动
适合人群：扁平组织管理者 / 赢在人际的业务人员 / 难迈协作第一步的职场人
情景模拟：跨部门协同挑战案例
In

影响力

"在跨流程、跨部门环境里达成目标总是困难重重 — 发挥非职权影响力是当代每一位管理者的必修课"
核心交付：影响力三要素实操妙招，获得他人承诺，以推动事情发生
课程要点：影响力定义与三大要素 → 了解立场获得他人承诺（界定承诺三步骤）→ 七大策略让影响力更有效 → 巧用包装打动他人（三大包装手段）→ 承诺评估与后续行动跟进 → 六项关键行动
适合人群：承上启下的中层管理者 / 需多方支持的项目管理者 / 总被"拒绝"的职场人
CR

解决冲突

"'差异'可以促进创新，'不合'会引发冲突 — 如何妥善处理矛盾，是管理的艺术"
核心交付：由道及术，运用5种策略模型，掌握6项关键行动，化冲突为共识
课程要点：冲突策略模型（五种策略）→ 强调冲突的影响 → 澄清冲突的原因 → 对各方表现出同理心 → 专注于解决冲突 → 发展他人解决冲突的想法 → 针对解决方案达成共识
情景模拟：项目血泪史 — 两个部门，一场纷争

人际管理三课的内在逻辑

协作（建立伙伴关系）→ 影响力（赢得承诺推动事情）→ 解决冲突（化分歧为共识）
三课形成从"建立关系"到"施加影响"再到"化解矛盾"的完整人际能力链。

领导力加速驱动 · 自我认知基座

Leadership Acceleration — 新经理转型必备的3门准备课程

DNA

领导力DNA

"领导力不是一种与生俱来的天赋，它是一种可以学习掌握，并能逐步提升的思维模式"
核心交付：解码领导力"基因图谱"，五个维度探秘个性"真相"，指导职业发展
课程要点：
1. 领导力DNA指标的理论来源 — 了解个性与自我认知
2. 个性与领导力发展：动力 vs 阻力
3. 领导力DNA指标解读 — 基于"大五"模型（51题测评），五个指标分别解读
4. DNA组合分析 — 不同组合的指标解读、个性与能力关系、个性与学习成效关系
核心工具：大五（Big Five）领导风格测试·51题 + DNA报告解读说明文案
LK

领导力启程

"新经理们，是否做好了转型的准备？角色转变带来诸多压力，做好准备对新晋经理至关重要"
核心交付：从聚焦于自己获得认可，转变为成就他人的成长型心态
课程要点：
1. 引言：领导力，从这里启程 — 领导力是后天可培养的能力
2. 从"院子"到"直升机"：提升你的领导者思维 — 阳台思维、树顶思维、直升机思维
3. 优秀领导者的自我修养 — 领导者应具备的特点及自我修炼
4. 领导力的三大维度 — 人际交往、业务管理、领导团队
LE

领导力精要

"了解作为领导者应该有的思维模式及自我修养，为后续课程学习做铺垫"
核心交付：不再仅仅是"专家"角色，需要更高视野与更强的团队带领能力
涵盖主题：经理人角色转变、领导力基本认知、如何与平级/老板/下属相处、拥抱新的团队领导角色

设计逻辑：先测评后认知，再进入核心课程

DNA（个性测评·知己）→ 启程（思维转型·明道）→ 精要（核心认知·筑基）
三门准备课程为后续16门核心能力课程建立自我认知基础和领导者思维框架。学员在清楚了解自身个性特质和发展方向后，有针对性地选择业务/团队/人际管理课程。

DDI的课程体系以单项能力为最小培训单位的模块化设计，使企业可以灵活组合课程。16门课程各自独立但形成体系，这种"积木式"产品架构值得参考。同时注意DDI侧重通用领导力，在出海/跨文化场景仅有"领导虚拟团队"一门，这是摩根差异化切入的空间。

DDI 英跃 · 竞争情报卡

DDI (Development Dimensions International) 成立于1970年，全球领先的领导力咨询机构。英跃是DDI在中国的数字化领导力学习品牌。

核心优势：

· 50年领导力研究与数据积累
· "行为模式法"将能力拆解为可操作的关键行动
· 基中层管理者覆盖面完整（16项能力）
· 四步学习闭环确保行为改善可衡量
· 大五模型测评与课程体系深度整合

产品局限：

· 强通用领导力，弱行业/场景定制
· 跨文化领导力仅有"虚拟团队"一门
· 缺乏出海/国际化人才培养专项方案
· 标准版纯线上，高级版才有混合式交付
· 央企/国企场景缺乏定制化适配

基本信息
中文名：智睿卓才企业咨询（上海）
产品线：英跃标准版/高级版/LMS
目标客群：中大型企业基中层
定价模式：线上直购+方案定制
总部：上海浦东前滩
竞争定位
品类：数字化领导力培训
对标：中欧、凯洛格、AMA
差异化：测评+课程一体化
对摩根的战略启示

1. 课程体系对标：DDI的16门能力课程覆盖了基中层管理者90%的通用场景，摩根可作为竞争参照，梳理自身课程的覆盖度和差异点。
2. 模块化设计参考："单项能力=最小培训单位"的产品思路，可应用于摩根课程体系设计。
3. 差异化空间：DDI在出海/跨文化/央企定制三个方向存在明显缺口，是摩根的蓝海机会。
4. 测评整合：DDI将大五测评与课程体系深度绑定的做法，摩根可参考将自身测评工具（如跨文化适应力评估）嵌入课程体系。
