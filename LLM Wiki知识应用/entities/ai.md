## 来源

## 核心架构

### 5-Agent内容生产线
```
Scout(选题侦察) → Writer(撰写+质检) → Publisher(发布)
                                       ↘  Feedback(数据回收→反哺选题)
Orchestrator(总编调度)
```

### 可迁移至爆红虾的能力

| 能力 | 迁移方式 |
|:----|---------|
| 选题评分公式 | 加权公式 `源质量^1.3×0.22 + 传播潜力×0.25 + 新鲜度×100×0.20` |
| 去AI腔正则库 | 建立内容风格检查规则集 |
| 文件系统即队列 | 用文件名约定作为任务状态管理 |
| 防死循环三层模型 | Agent协作的安全防护机制 |
