## 二、企业所得税（CIT）

| 项目 | 税率/规定 |
|------|----------|
| **标准税率** | **22%**（2022年起），满足条件可降至20% |
| **征税原则** | 居民企业全球所得征税；非居民仅印尼来源收入 |
| **全球最低税（支柱二）** | IIR和QDMTT自**2025年1月1日**起适用，UTPR自**2026年1月1日**起适用 |
| **亏损结转** | 可向前结转**5年**，不可向后追溯 |
| **境外税收抵免** | 有，对同一所得已缴境外税款可抵免 |
| **资本弱化** | 债务权益比**4:1**（特定行业另有规定） |

### 税收优惠（Incentives）

- **免税期（Tax Holiday）**：对新投资先锋行业（如炼油、金属冶炼、化工、制药等）给予**5至20年**企业所得税减免
- **免税津贴（Tax Allowance）**：特定行业和区域可享受**30%投资抵减**（分6年摊销）、加速折旧、延长亏损结转
- **经济特区（KEK）**：企业所得税可降至**0%-17%**，视投资规模而定
- **工业园区/保税区**：进口关税豁免、增值税优惠

## 三、个人所得税（PIT）

| 项目 | 内容 |
|------|------|
| **居民标准** | 在印尼停留**183天**或在12个月内居住意图 |
| **征税原则** | 居民全球征税；外籍人士前**4年**仅就印尼来源收入纳税 |
| **税率** | 累进税率：5%（≤6,000万IDR）至**35%**（>50亿IDR） |
| **申报截止** | 次年**3月31日** |

---

## 四、增值税（VAT/PPN）

| 项目 | 内容 |
|------|------|
| **标准税率** | **11%**（实际），法律税率12%（奢侈品适用） |
| **出口税率** | **0%** |
| **应税门槛** | 年营业额**4.8亿IDR** |
| **奢侈品销售税（LST/PPnBM）** | 10%-200%，取决于商品类别 |

## 五、预提税（Withholding Tax）

| 收入类型 | 居民企业→非居民企业 | 税收协定可降低 |
|---------|-------------------|--------------|
| **股息** | **20%** | ✅ |
| **利息** | **20%**（债券10%） | ✅ |
| **特许权使用费** | **20%** | ✅ |
| **技术服务费** | **20%** | ✅ |
| **国内技术服务费** | **2%** | — |
| **分支机构利润汇出税** | **20%**（可协定降低，再投资可豁免） | ✅ |

## 六、转让定价（Transfer Pricing）

- 关联交易须符合**独立交易原则（Arm's Length Principle）**
- 合并收入≥**1.1万亿IDR**须提交**国别报告（CbCR）**
- 可申请**预约定价安排（APA）**
- 同期文档（主文件、本地文件）准备要求

## 八、税收协定网络（DTT）

- 印尼已与**超过70个国家/地区**签署税收协定
- **BEPS多边公约（MLI）** 于2020年8月1日起在印尼生效
- 已签署**STTR多边公约**，预计影响36个现有税收协定

## 九、培训启示

### 外派税务合规培训要点

1. **183天规则陷阱**：外籍人员在印尼居住满183天即成为税收居民，需关注全球所得申报义务，但前4年可仅申报印尼来源收入
2. **社保登记义务**：在印尼工作≥6个月的外国人须参加社会保障计划（人力计划+医疗计划）
3. **预提税协定优惠**：股息/利息/特许权使用费支付时可依据税收协定申请降低税率，需确保**税收居民证明（COR）** 完备
4. **增值税登记门槛**：年营业额超4.8亿IDR须登记，外籍电商平台须关注远程服务的增值税义务

### 薪资架构设计启示

- 外籍人士前4年仅就印尼来源收入纳税，境外收入（如母国工资、股票期权）可做税务规划
- 社保缴费上限和比例需纳入总用工成本计算
- 分支机构利润汇出税20%，可通过再投资规避
