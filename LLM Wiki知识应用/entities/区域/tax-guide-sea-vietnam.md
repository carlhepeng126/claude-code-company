## 二、企业所得税（CIT）

| 项目 | 税率/规定 |
|------|----------|
| **标准税率** | **20%** |
| **中小企业优惠** | 年收入≤30亿VND → **15%**；30亿-500亿VND → **17%** |
| **石油天然气** | **25%-50%**（视项目类型） |
| **征税原则** | 居民企业全球所得；非居民仅越南来源收入；分公司与子公司同等待遇 |
| **全球最低税（支柱二）** | **IIR和QDMTT自2024年1月1日起适用**；UTPR暂未引入 |
| **亏损结转** | 连续**5年**，不可向后追溯 |
| **境外税收抵免** | 有，以越南应纳税额为上限 |

### 税收优惠（Incentives）

- **优惠税率10%**（最长15年，可延至30年）— 适用于激励行业/地区项目
- **优惠税率17%**（10年）
- **免税期政策**：**"四免九减半"**（4年免税+9年减半）或**"四免五减半"**
- **社会化领域**（教育、医疗、文化、环保）：全周期**10%**
- **高科技/研发/半导体/AI特大型项目**：
  - 优惠税率**5%-10%**，期限15-22.5年
  - 免税**4-6年**+减半**9-13年**
- **新增/扩建投资**：符合条件也可享受税收优惠

## 三、个人所得税（PIT）

| 项目 | 内容 |
|------|------|
| **居民标准** | (1) 在12个月内在越停留**≥183天**；(2) 在越南有住所；(3) 一年内租用房≥183天 |
| **征税原则** | 居民全球所得；非居民仅越南来源收入 |
| **居民税率（就业收入）** | 累进税率**5%-35%** |
| **非居民税率（就业收入）** | 统一**20%** |
| **高科技人才免税** | 数字技术、半导体、AI等领域研发人员就业收入**5年免税** |
| **个人基本扣除** | 纳税人每月**1,550万VND**；抚养人每人**620万VND**/月 |

### 应税收入范围

- 工资薪金（含大部分现金/实物福利）
- 股息、利息（政府债券/银行存款/人寿保险利息除外）
- 证券转让资本利得
- 年收入超过5亿VND的个体经营所得
- 超2,000万VND的特许经营、继承、赠与、土地使用权转让、中奖等
- 数字资产、碳排放权、车牌转让等新型收入

## 四、增值税（VAT）

| 项目 | 内容 |
|------|------|
| **标准税率** | **10%**（2025年6月30日发布降税令，部分商品/服务降至**8%**至2026年12月31日） |
| **优惠税率** | **5%**（基本民生商品/服务） |
| **零税率** | **0%**（出口商品和服务） |
| **登记要求** | 营业执照颁发后**10日内**完成VAT登记 |
| **申报频率** | 月度（次月20日）或季度（下季度30日） |
| **电子发票** | 强制要求 |
| **海外供应商** | 通过电商/数字平台向越南提供服务须自行或委托代表在越登记 |

## 五、预提税（Withholding Tax）

| 收入类型 | 支付给非居民公司 | 支付给居民公司 | 税收协定可降低 |
|---------|-----------------|--------------|--------------|
| **股息** | **0%**（不征税） | 0% | — |
| **利息** | **5%** | 0%（豁免） | ✅ |
| **特许权使用费** | **10%** | 0%（豁免） | ✅ |
| **技术服务费** | **5%** | 0%（豁免） | ✅ |
| **分支机构利润汇出税** | **无** | — | — |

> **特别提示**：非居民个人取得股息/利息/特许权使用费，预提税率统一为**5%**，税收协定可从优。

### 外国承包商预提税（FCWT）

非居民实体在越提供商品与服务的收入，须缴纳**企业所得税+增值税**的组合预提税，综合税率**0.1%-15%**。

## 六、转让定价与反避税

| 项目 | 规定 |
|------|------|
| **转让定价方法** | 优先使用可比非受控价格法（CUP）；OECD指南基本接轨 |
| **同期资料** | 需准备本地文件、主文件、国别报告（符合BEPS第13项行动计划） |
| **利息扣除限制** | 净利息费用上限为**EBITDA的30%**（含关联和非关联交易），不可扣除利息可结转**5年** |
| **实质重于形式** | 不具经济实质的交易可被调整或拒绝协定优惠 |
| **APA** | 可申请预约定价安排 |
| **CbCR** | 特定跨国企业须提交国别报告 |

## 七、其他企业税

| 税种 | 说明 |
|------|------|
| **社会保障（雇主）** | 越籍：SI 17% + HI 3% + UI 1% + 工伤0.5% = **21.5%**；外籍：HI 3% + SI 17.5%（部分情况） |
| **社会保障（雇员）** | 越籍：SI 8% + HI 1.5% + UI 1% = **10.5%**；外籍：HI 1.5% + SI 8%（部分情况） |
| **印花税** | 对特定财产（含不动产）征收**0.5%-15%** |
| **不动产税** | 市政当局征收土地租金、土地使用费等 |

## 八、合规申报周期

| 税种 | 申报周期 | 截止日期 |
|------|---------|---------|
| **企业所得税（预缴）** | 季度 | 按预估**季度预缴**，四季度总额须达年度CIT的**80%** |
| **企业所得税（年度决算）** | 年度 | 财年结束后**第3个月最后一天** |
| **个人所得税（月度/季度）** | 月度/季度 | 雇主代扣代缴，年度最终申报：次年**3月最后一天** |
| **个人所得税（个人直接申报）** | 年度 | 次年**第4个月最后一天** |
| **增值税** | 月度/季度 | 月度：次月**20日**；季度：下季度**30日** |

## 九、税收协定网络（DTT）

- 越南与**超过80个国家和地区**签署税收协定
- 与中国税收协定：股息10%、利息10%、特许权使用费10%
- BEPS多边公约（MLI）已对越南生效
- 税收征管高压：欠税个人可被**限制出境**（第49/2025/ND-CP号议定）

## 十、培训启示

### 外派税务合规培训要点

1. **183天双重测试**：越南的居民判定采用"12个月滚动183天"规则，外派人员入境日期开始计算，需精确记录出入境日期
2. **税收征管高压**：越南已实施欠税个人**限制出境**措施，外籍人员务必按季度核对申报记录，避免因信息滞后导致出境受限
3. **高科技人才5年免税**：半导体、AI、数字技术领域的外籍人才可享受5年工资薪金免税，对外派政策有重大影响
4. **FCWT制度**：外国承包商预提税涵盖CIT+VAT双重税负，跨境服务合同中需明确税费承担方
5. **外籍社保**：外籍员工在特定情况下须缴纳HI 1.5%+SI 8%（退休基金），计入用工成本

### 薪资架构设计启示

- 居民累进税率最高35% vs 非居民统一20%——控制居民身份对税负有显著影响
- 个人基本扣除1,550万VND/月、抚养人扣除620万VND/月，薪资设计中应考虑抵扣项安排
- 股票期权、限制性股票等股权激励收入按劳动收入征税，适用累进税率
- 搬迁补贴、探亲机票、学费等特定福利可**免税**——外派薪酬包设计时优先纳入
- 年度最终申报由**雇主**负责提交（截止日期为3月底），HR需提前准备全年数据
