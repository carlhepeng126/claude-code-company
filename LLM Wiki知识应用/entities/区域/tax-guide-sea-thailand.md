## 二、企业所得税（CIT）

| 项目 | 税率/规定 |
|------|----------|
| **标准税率** | **20%** |
| **中小企业优惠** | 净利润一定范围内适用累进税率**0%/15%/20%** |
| **国际运输** | 在泰有办事处的外国公司按总收入**3%**征税 |
| **征税原则** | 居民企业全球所得；非居民仅泰国来源收入 |
| **全球最低税（支柱二）** | 《补足税紧急法令》自**2025年1月1日**生效，适用于合并收入≥**7.5亿欧元**的跨国集团；QDMTT、IIR、UTPR视情况适用 |
| **亏损结转** | 最多**5年**，不可向后追溯 |
| **境外税收抵免** | 有，以泰国应纳税额为上限 |

### 税收优惠（Incentives）

- **BOI投资促进**：依据行业类别给予**3至13年**免税期，特定措施可额外延长
- **国际商务中心（IBC）**：
  - 年运营支出≥6,000万THB → CIT降至**8%**
  - 年运营支出≥3亿THB → CIT降至**5%**
  - 年运营支出≥6亿THB → CIT降至**3%**
  - IBC的外籍员工享**15%**固定PIT税率
- **特别经济区/边境经济特区**：额外税收优惠

## 三、个人所得税（PIT）

| 项目 | 内容 |
|------|------|
| **居民标准** | 一个日历年度内在泰国停留**180天或以上** |
| **征税原则** | 居民仅就**泰国来源收入**纳税；境外所得**汇入泰国时**纳税 |
| **税率** | 累进税率最高**35%**（>500万THB） |
| **申报截止** | 次年**3月31日**（电子申报可延8天） |

---

## 四、增值税（VAT）

| 项目 | 内容 |
|------|------|
| **标准税率** | 法律**10%**，实际执行已降至**7%** |
| **出口** | **0%** |
| **登记门槛** | 年营业额超过**180万THB** |
| **电子服务** | 向泰国非VAT登记用户提供电子服务的海外平台，年收入超180万THB须登记并按月申报 |

## 六、其他企业税

| 税种 | 说明 |
|------|------|
| **土地与建筑税** | 农业0.01%-0.1%、住宅0.02%-0.1%、商业0.3%-0.7%、荒地0.3%-3% |
| **特别营业税（SBT）** | 银行/金融交易3%、不动产销售3%，另加10%地方税 |
| **印花税** | 租赁/股权转让0.1%、贷款合同0.05%（上限10,000THB） |
| **社会保障** | 雇主和雇员各按月薪**5%**（有上限） |

## 七、转让定价（TP）

- 关联交易须符合**独立交易原则**
- 年收入≥**2亿THB**的企业须提交关联交易披露表和转让定价文档
- 可申请**预约定价安排（APA）**
- 未按规定提交，罚款最高**20万THB**
- CbCR适用于2021年1月1日后开始的财政年度

## 九、税收协定网络（DTT）

- 泰国与**超过60个国家和地区**签署了税收协定
- 普遍适用**OECD税收协定范本**
- 对中国税收协定：股息10%（持股≥25%可降至5%）、利息10%、特许权使用费15%
- BEPS MLI已对泰国生效

## 十、培训启示

### 外派税务合规培训要点

1. **180天居民测试**：在泰停留满180天即为税收居民，但泰国采取**来源地征税+汇入原则**——境外所得只有在汇入泰国时才需纳税，这为外派人员提供了规划空间
2. **IBC外籍员工15%固定税率**：在IBC工作可大幅降低税负（正常最高35%），是区域总部设立的重要考量
3. **BOI优惠政策管理**：需监控免税期结束后的利润分配及分支机构利润汇出税（10%）
4. **半年预缴制度**：泰国企业须提前预估半年利润并预缴所得税，低估超过25%将面临最高20%附加费

### 薪资架构设计启示

- 境外收入**不汇入不征税**——可考虑境外支付部分薪酬以合法降低税负
- 股票期权、股权激励在未变现/未汇入前可能不计入应税收入
- IBC外籍员工可适用15%优惠税率，显著降低高端人才税负
- 雇主须每月代扣代缴工资税（薪酬税），社保缴款按月薪5%计算
