# 深度学习萃取 — 跨素材合成报告

## 核心内容

本报告是10套知识素材（华为SDBE闭环战略管理、麦肯锡结构化战略思维、TTT内训师宝典、一分钟面对面识人、组织发展与构建、跨文化管理精品案例、8本职场沟通书籍、B2B Professional Sales、天元鸿鼎、英耐英语）的跨素材关联分析。核心发现包括：

1. **六本书全景图谱**：华为SDBE（战略落地）→ 麦肯锡结构化（思维方式）→ TTT（教学技能）→ 一分钟识人（人才识别）→ OD（组织能力）→ 跨文化案例（中国企业出海），形成"战略→思维→工具→沟通→案例"的完整培训产品链。
2. **涉外沟通胜任力全栈**：从心理底层到谈判上层的4层架构（自信开口↔结构化表达↔专业演讲↔谈判实战）。
3. **社交风格矩阵**：独立于DISC的B2B销售场景框架（Analytical/Amiable/Driver/Expressive），含16子象限+压力降级行为。
4. **天元鸿鼎竞品画像**：建工行业人才培养全案，含项目经理五步法、商务三步走、钻石模型等核心方法论。
5. **英耐英语竞品分析**：T+C+E（Teaching+Coaching+Extension）通用框架、CEFR分级、1对1私教+APP模式。

素材之间存在强互补关系：8本职场沟通书籍填补了R.I.S.E.体系的学术背书空白；Feedback Toolkit的16个工具可重构I层（Interact）模块；B2B销售与8本书涉外沟通全栈的社交感知层互补，可合成"中企出海B2B销售训练营"。

## 与Carl业务的关系

- **课程设计**：提供10套素材的关联索引，帮助设计"战略→思维→工具→沟通→案例"完整培训产品链
- **赢单**：跨素材关联矩阵可作为售前诊断工具，展示Carl团队的知识深度和系统性思维
- **服务客户**：可根据客户需求，从10套素材中灵活组合定制化培训方案（如出海企业选"华为SDBE+跨文化案例+8本职场沟通"组合）
- **差异化竞争**：合成报告本身即专业壁垒，证明Carl团队不仅做培训，更做"知识工程"
