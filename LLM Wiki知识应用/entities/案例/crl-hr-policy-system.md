## 二、经理人全生命周期管理

### 2.1 经理人12条标准（素质模型）

| 维度 | 标准 | 备注 |
|------|------|------|
| 激情 | 事业心、进取心 | |
| 学习 | 学习能力、自我提升 | |
| 团队 | 团队建设、协作 | |
| 诚信 | 正直、可信赖 | |
| 创新 | 变革、突破 | |
| 决断 | 果断、决策力 | 谈判和争辩能力强 |
| 学识 | 专业知识、管理知识 | |
| 经历 | 多岗位、多行业历练 | |
| 智力 | 分析判断、问题解决 | 包含谈判能力 |
| 表达 | 沟通技巧、感召力 | "善于鼓动，感召力强" |
| 体魄 | 身心健康 | |
| 环境 | 环境的适应、塑造能力 | 包含跨文化适应 |

### 2.2 一把手标准
> "诚信的人、反思的人、带队的人、成事的人"

### 2.3 经理人五维历练模型
> 精神 → 修养 → 品格 → 能力 → 经历

### 2.4 核心管理机制
- **滚动评价** — 360度滚动评价机制
- **后备比例** — 主要负责人:后备 = 1:2；管理团队:后备 = 1:1
- **强制轮岗** — 集团领导层须有多行业、多岗位体验
- **强制休假** — 领导层/执行层 ≥ 10天/年
- **专业双轨制** — 管理序列+专业序列，淡化行政级别

## 三、培训体系特色

### 3.1 行动学习——华润最具辨识度的培训创新

行动学习在华润已从培训工具升华为**战略执行手段**：
- 每年10月召开"行动学习"经验分享会
- 经理人年会评出上年度最佳实践单位
- 用行动学习的方式推动利润中心打造行业优势
- 以解决迫切的业务问题为载体

### 3.2 内部培训师四级认证体系

| 级别 | 角色定位 | 关键要求 |
|------|---------|---------|
| 初级培训师 | 基础授课 | 能讲标准课程 |
| 中级培训师 | 催化引导 | 能做行动学习催化师 |
| 高级培训师 | 课程开发+教练 | 能开发课程、做教练 |
| 首席培训师 | 体系设计 | 能搭建培训体系 |

> 华润内部培训师必须是 **"催化师+教练"** 双角色——这大幅提升了培训师的专业门槛和价值定位。

### 3.3 利润中心必做培训项目
1. 新员工入职培训
2. 管理岗位晋升培训
3. 专业岗位晋级培训
4. 配合集团年度战略重点的培训

### 3.4 关键数据点

| 项目 | 标准 | 出处 |
|------|------|------|
| 关注层经理人培训 | ≥ 100小时/年 | 第28条 |
| 主管谈话时长 | ≥ 3小时/年 | 第26条 |
| 后备比例 | 主要负责人1:2 | 第6条 |
| HR审计 | 每两年一次 | 第3条 |
| 战略规划滚动 | 三年制定/研讨一次 | 第7条 |

## 四、薪酬与激励体系

### 4.1 薪酬结构四大要素
1. 基本薪酬
2. 绩效奖金
3. 中长期激励
4. 福利保障

### 4.2 四大奖项
- **终身成就奖** — 30万港币/年，服务≥15年+3次集团级奖项
- 年度优秀奖
- 专项奖
- 团队奖

### 4.3 薪酬原则
- "必需实现主营行业关键岗位薪酬水平有足够的竞争力，总体人力成本低于竞争对手"
- "薪酬尽量减少福利功能，突出激励功能"
- 市场化导向，经理人薪酬体现市场价值

## 五、HR数字化监控体系

### 5.1 13项监控指标（示例）

| 类别 | 指标 |
|------|------|
| 态度类 | 员工满意度、敬业度 |
| 经理人类 | 经理人滚动评价指数、薪酬竞争指数、健康指数、流失率 |
| 运营类 | 人均销售额、人均利润、人力成本率 |
| 风险类 | 关键岗位空缺率、劳动争议率 |

### 5.2 数字化要求
- 统一标准、统一平台、统一语言
- 每两年一次薪酬调查（可与外部咨询公司合作）
- HR从"事务管理"向"数据驱动的战略型"转型

## 七、行业趋势意义

1. **多元化控股HR模式价值凸显** — 统一性与行业适配性兼具
2. **HR从行政事务向战略驱动转型** — 数据驱动人才决策
3. **市场化选人用人深化** — 市场化测评、市场化薪酬
4. **数字化HR监控体系成行业大势** — 用数据说话
5. **学习型组织建设** — 创建国际领先的学习型组织
