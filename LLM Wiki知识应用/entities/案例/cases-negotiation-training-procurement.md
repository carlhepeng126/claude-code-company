# 涉外商务谈判-培训采购篇

> 从培训供应商（乙方）视角看央企/出海企业的培训采购谈判。这是Carl的"实战手册"——因为客户咨询的三大问题之一就是"如何建立国际化人才培养体系"，而体系建立的第一步通常是从培训采购谈判开始的。

## 一、央企培训采购的核心痛点

| 痛点 | 严重度 | 真实场景 |
|------|--------|---------|
| 供应商"学术派多、实战派少" | ★★★★★ | 客户对国内培训机构的普遍不满 |
| 内容与业务场景脱节 | ★★★★☆ | 课程理论化，无法直接应用到海外项目 |
| 供应商"服务省心"能力不足 | ★★★★☆ | 交付协调/数据管理/跟进机制缺失 |
| 供应商选择困难 | ★★★★☆ | 学术派vs实战派怎么选 |

> **关键洞察**：客户（能建国际、中交海外等）普遍反映国内供应商"学术派偏多、实战派偏少"——这正是摩根世纪强调实战性的差异化突破口。

## 二、培训采购谈判的三大确认原则

谈判前必须确认：

1. **人员构成**：资深人员 vs 储备人员的比例
2. **业务阶段**：市场开拓期 vs 项目执行期
3. **真实目的**：人才培养 / 引入外脑 / 改变理念

> "客户给你的自由度越大，其实对你的考验越高"

## 三、供应商更替的真实案例

### 案例：中交路建海外-供应商更替

**背景**：中交路建此前长期与基业长青合作，反馈"学术派偏多，实战派偏少"，正在寻找新供应商。

**谈判过程**：摩根团队通过主动接触，强调"实战性+服务省心"核心差异化，成功获取客户信任。

**关键点**：摸清竞品不满点 → 找到自身差异化定位 → 完成供应商更替。

### 案例：能建国际-管理层在线英语培训方案谈判

**核心内容**：涉外商务谈判基础（B1+至C2，380+节次）+ 涉外商务谈判进阶

**谈判要点**：学员级别分层、课程定制化程度、考核机制设计

## 四、培训交付中的信任危机与解决方案

### 数据不一致引发的信任危机

**问题**：系统数据与给客户的报告数据不一致，被客户发现。

**根本原因**：数据管理缺乏严谨性和确认机制；系统数据与对外报告数据缺乏同步验证。

**解决方案**：
- 建立数据确认SOP
- 设置数据同步检查点
- 定期数据审计

**谈判要点**：主动承认问题 → 提出系统性解决方案 → 重建信任

## 五、培训预算谈判策略

### 价值定价，而非成本定价

央企客户关注的**不是价格最低**，而是"实战性+服务省心+可量化效果"。

### 分层报价策略

| 方案 | 内容侧重 | 适用客户 |
|------|---------|---------|
| 商务导向型 | 商务内容≤2天，其余项目管理与风险 | 项目经理为主 |
| 项目管理导向型 | 商务1天，项目管理2-3天 | 综合管理团队 |

### 渐进式合作模式

> "先拿小单再扩展，不要让客户感觉你在钓鱼"

从英语能力提升班开始 → 建立信任 → 扩大到面授项目经理培训 → 更大项目

## 六、培训效果承诺（SLA）

### 可量化的交付标准

- 课程完成率 ≥ 85%
- 学员满意度 ≥ 90%
- 英语水平至少提升1个摩根级别（可量化）
- 每个学员输出实战应用方案（可验证）
- 培训后1个月跟踪（可持续）

### 服务响应标准

| 服务项 | SLA标准 |
|--------|--------|
| 问题响应 | 24小时内 |
| 技术故障处理 | 48小时内给方案 |
| 内容调整 | 3个工作日内反馈 |
| 学员投诉 | 立即响应+跟踪解决 |

## 七、供应商管理谈判要点

### 供应商更替的触发信号

- 客户反映"学术派偏多，实战派偏少"
- 数据管理缺乏严谨性（信任危机）
- 课程内容与学员级别不匹配
- 缺乏属地化管理经验和方法论

### 新供应商信任建立路径

**诊断先行 → 小单验证 → 逐步扩展**

## 八、小红书可用选题

1. **"央企换培训供应商，90%的人搞错了第一步"**
   - 摸清竞品不满点才是突破口
   - 中交路建从基业长青换到摩根的案例

2. **"给央企报培训方案，为什么不能一上来就报价格？"**
   - 先诊断：人员构成/业务阶段/真实目的
   - 三大确认原则展开

3. **"培训效果怎么承诺才不算虚假宣传？"**
   - 可量化的SLA标准
   - 对HR来说很实用的谈判参考

4. **"央企培训采购谈判避坑指南"**
   - 预算谈判/合同条款/交付保障
   - 乙方视角告诉甲方怎么避坑
