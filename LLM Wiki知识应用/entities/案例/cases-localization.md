# 属地化融合案例

## 已验证案例

### 比亚迪巴西卡马萨里
- 来源：ESG报告 + 36氪
- 规模：800人
- 效果：上岗合格率60%→92%
- 用途：汽车出海培训参考

### 奇瑞西班牙巴塞罗那
- 来源：虎嗅 + 霞光社
- 规模：300人
- 效果：提前1月达产，合格率98%
- 用途：欧洲出海培训参考

### 蔚来阿联酋
- 来源：亿欧2025年9月
- 规模：200人
- 效果：NPS 4.8/5
- 用途：**销售线索** — 中东拓展客户

### 海尔埃及三段式 ★
- 来源：官网 + 亿欧
- 规模：450人
- 效果：三段式模型（认知→浸润→实战）
- 用途：可直接融入R.I.S.E.课程体系

### TikTok印尼
- 来源：36氪 + 霞光社
- 规模：1200人
- 效果：审核误判8%→3.2%

## 新增：《跨文化管理精品案例》本土化案例补充

以下案例来自魏小军《跨文化管理精品案例》，提供了中外合资企业本土化管理的深度实例。

### 法波坚石 — 完全本土化最佳实践
- **模式**：完全本土化管理 — 2500名中国员工，全部中高层由中方担任
- **外方角色**：意方仅早期派驻财务控制人员，3-4年轮换
- **品牌策略**：意大利品牌+中国表达（"法波电子，创立于1936年，来自佛罗伦萨"）
- **培训机制**：中方员工到意大利总部"回顾性比较"培训
- **HR成果**：员工留任率从50%提升到75%（职业生涯规划+企业文化+公平报酬）
- **对标概念**：[[localization-complete-vs-hybrid]]

### BC×CC — 混合管理失败教训
- **外方派驻困境**：英方无法长期派驻管理者，造成"权力真空"
- **语言障碍**：维特代总经理不会普通话，无法直接沟通
- **制度冲突**：中方"灰色操作"（采购回扣、假发票）与英方制度的直接冲突
- **教训**：混合管理需明确边界，外派人员语言能力是前提
- **对标概念**：[[localization-complete-vs-hybrid]]

### 其他本土化案例
- **德国五矿**：中资在德全资子公司的"四阶段"发展框架，德籍高管管理
- **上工申贝/德国DA**：中资收购后的技术+品牌+工会三维度本土化整合
- **小肥羊加拿大**：中餐出海的本土化适应（菜单调整、供应链重建）
