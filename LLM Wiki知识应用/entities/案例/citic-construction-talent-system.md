## "四苗工程"方法论框架

**"四苗工程"**是2025年推出的系统化年轻干部培养框架，荣获第五届党建创新成果展示交流活动示范案例（为所在组别唯一入选的国有企业案例）。

### 1. 强基"育苗" — 把稳方向盘
- **思想淬炼"铸魂"**：线上"中信建设培训学苑"自主选学，线下理论学习中心组、"第一议题"制度、青年理论学习小组
- **党性锤炼"修心"**："三会一课"、主题党日，"身在境外、党在心中"等主题实践
- **帮带历练"赋能"**：选配导师与1-2名年轻干部结对，政治上引导、业务上传授、纪律上提醒

### 2. 一线"墩苗" — 练就铁肩膀
- **多岗位锻炼**：总部与项目一线轮岗、不同国家/区域之间轮岗、管理与专业技术岗位之间轮岗
- **多任务磨炼**：市场开拓、国际商务、投资融资、项目攻坚、风险化解等作为"练兵场"
- **多难题历练**：围绕海外投建营评估、投融资模式探索、风险防控等课题组织攻关小组

### 3. 机制"优苗" — 激发动力源
- **能上能下机制**：完善领导干部选拔任用和考核评价机制
- **正向激励机制**：境外一线薪酬倾斜、艰苦地区补贴、优先提拔使用境外干部
- **创先争优机制**："十佳共产党员""优秀员工"评选，设立"青年突击队""党员先锋队"

### 4. 暖心"护苗" — 撑起平安伞
- **日常监管**：境外事业部领导班子配备"廉洁专员"，项目部设置"廉洁监督员"
- **人文关怀**：海外员工家属探亲安排、商业保险多重保障、与国际SOS合作提供全球24小时援助
- **容错纠错**：非主观过错具体分析，坚持"出于公心、尽职尽责"容错底线

> 来源：《中信建设精心实施"四苗工程" 锻造堪当共建"一带一路"重任的年轻干部队伍》（2025年11月）

## 数字化学习平台

### 双平台架构
| 平台 | 定位 |
|------|------|
| **中信建设智汇堂** | 移动学习平台，随时随地随身学习理论知识、提升业务技能、感悟企业文化 |
| **中信建设培训学苑** | 正式培训管理平台，含必修课、选修课、直播课、微课、考试等功能 |

### 平台特色
- 确保海内外员工"零时差"参加政治学习和业务培训
- 员工自行制作规章制度解读微课（2016年启动）
- 涵盖各业务领域、共116人的**内训师队伍**
- 持续更新的必修课+选修课内容体系

> 来源：《中信建设坚持走人才自主培养之路》（2023年5月）

## 内训师体系建设

中信建设从2016年开始组织员工自行制作微课，2018年以"中信建设培训学苑"上线为契机，逐步建立起涵盖各业务领域的**116人内训师队伍**，采用线上与线下相结合的方式为广大员工培训授课。

## 青训营品牌方法论（详版）

青训营作为核心品牌项目，其方法论可抽象为以下模型：

### 四层次价值闭环模型
```
┌─────────────────────────────────────────────────────────────────┐
│                   价值层：党建凝魂，筑牢理想信念                   │
│     （十九届六中全会、二十大精神、井冈山精神、专题辅导）          │
├─────────────────────────────────────────────────────────────────┤
│                   理论层：顶级师资，锚定能力短板                   │
│   （DDI英跃领导力、TechMark商战模拟、海外沙盘、跨界参访）        │
├─────────────────────────────────────────────────────────────────┤
│                   实战层：行动学习，引领攻坚克难                   │
│  （公司首次引入行动学习，课题攻关、案例复盘、沙盘推演）           │
├─────────────────────────────────────────────────────────────────┤
│                   提升层：自我认知，制定职业规划                   │
│     （进阶版导师制、公司领导座谈、人才测评、职业发展通道）        │
└─────────────────────────────────────────────────────────────────┘
```

### 行动学习机制
青训营在公司**首次引入行动学习**，流程如下：
1. 行动学习导师+人力资源部+学员共同确定课题（基于公司战略和重难点工作）
2. 特邀公司内部相关领域中青年骨干作为专业导师
3. 时间跨度近一年的持续团队协作
4. 向公司董事长、总经理等领导汇报课题成果
5. 评估并纳入公司决策参考

### 导师制进阶版
- **职业生涯导师**：公司领导担任，指导中长期职业发展
- **行动学习专业导师**：业务中青年骨干担任，指导课题攻关
- 结营时导师为学员颁发青训营专属徽章及"知行录"

## 以赛带训机制

### 三大品牌赛事
1. **外语考试** — 每年一次，9语种，全球云考试，配套津贴激励
2. **英语竞赛** — 每年一届，已连续7年，品牌化运营
3. **微课大赛** — 已举办两届，鼓励员工知识沉淀与分享

核心逻辑：**"以比赛培训人才，靠比赛发掘人才"**

> 来源：《中信建设坚持走人才自主培养之路》（2023年5月）

## 激励保障机制

### 薪酬激励
- 项目一线人员重点倾斜
- 卓越贡献人员专项奖励
- 专业技术人员收入涨幅高于中高层管理人员
- 技术技能津贴、外语津贴
- 科技研发人员激励投入

### 容错机制
研究出台容错机制，全面营造勇于创新、敢于担当的干事创业氛围。

### 人才评价
- 2020年按标准组织超1,700人次民主测评和360度访谈
- 建立人才考察数据库
- 公开竞聘选拔干部（2021年实施两个事业部总经理岗位"举手"竞聘）
- 全年提聘各级干部144人次

> 来源：《坚持走人才自主培养之路，打造集团新型城镇化板块人才高地》（2022年6月）

## 培养成效

- **青训营**：累计95+名青年学员参训（两期），100%学员获得不同程度提升，部分学员获得职务或职级晋升
- **内训师队伍**：116人，覆盖各业务领域
- **外语考试**：连续6年，9语种，覆盖全球项目所在地
- **英语竞赛**：连续7年，2024年参赛956人创历史新高
- **人才储备**：快速搭建"战略预备队、业务先遣队、智囊储备队"
- **行业认可**：青训营获评"人才强企工程·品牌学习项目"，《培训》杂志2024年第1期刊发专题文章
