# Communication Skills Training（Robert Moore）

## 简介
Robert Moore著《Communication Skills Training》（约1000行企业适用正文），一套面向个人自我提升的沟通技能方法论体系。本书以「自信是一切沟通技术的地基」为核心观点，从自信建设→言语沟通→非言语沟通→身体语言→眼神→步态→姿势→手势逐层展开。后半部分（Preview章节）为PUA/搭讪内容，不建议纳入企业培训素材库。

## 核心方法论

### 自信三角形（Confidence Triangle）
```
        Self-confidence
        /              \
  Self-efficacy    Self-esteem
    ↑               ↑
 能做好            配得上
```
> 自信 = 我相信我能做好（self-efficacy）+ 我相信我配得上（self-esteem）

### 自信建设三阶段
| 阶段 | 活动 | 工具 |
|------|------|------|
| Preparation | 盘点→列成就清单→SWOT→SMART目标→100%承诺 | 成就清单/SWOT/目标承诺书 |
| Coming Out | 获取知识→打基础→追求卓越而非完美→从小目标开始 | 学习路径/基础课程/里程碑 |
| Acceleration | 接受更大挑战→保持谦逊→拒绝傲慢 | 挑战分级/导师反馈 |

### 发音清晰度5法
1. **LTTT练习**：快速重复「The lips, the teeth, the tip of the tongue」
2. **A2E练习**：逐音节说「e-nun-ci-ate, ar-tic-u-late, ex-agg-er-ate」
3. **阅读朗读**：大声读文章并强制张大嘴巴（至少一根手指宽度）
4. **Pencil Mouth**：横咬铅笔唱歌/说话
5. **录音自评**：录制并回看口腔开合度

### 非言语沟通4大功能
- **重复**：非言语动作强化言语信息（说「我最有资格」时站得更直）
- **补充**：非言语动作补充言语信息（说「我想你」+拥抱）
- **替代**：非言语动作取代言语信息（你看到的表情就知道不用问了）
- **强调**：非言语动作凸显信息（说「我生气了」+摔杯子）

### 身体语言7大组成部分
1. 面部表情（≈Paul Ekman教授的微表情理论）
2. 姿势与身体移动（站姿/坐姿/头部位置）
3. 手势（开放型/闭合型/塔尖型）
4. 眼神（30/70法则+看周围区域而非直视+文化差异）
5. 触摸（握手的力度=自信程度）
6. 个人空间（文化差异极大→1.2m/0.5m/1.5m）
7. 语调（同一句话的不同语调=完全不同意思）

### 自信自评5维度
| 自信的表现 | 不自信的表现 |
|-----------|-------------|
| 做对的事→即使被批评 | 行为由他人看法决定 |
| 敢于冒险→多做 | 最小化风险→安全区 |
| 承认错误→学习 | 否认错误→掩盖 |
| 等人认可 | 自吹自擂 |
| 欣然接受赞美 | 否认/轻视成就 |

## 注意点
- **企业使用需过滤**：本书后半部分含PUA/搭讪/性别刻板印象内容，建议仅萃取前10章
- **与经典理论差异**：本书称「10%言语+90%非言语」vs 经典的Mehrabian「7-38-55法则」——略有偏差但方向一致
