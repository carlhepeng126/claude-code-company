# 雷·克洛克 — 麦当劳创始人

## 核心标签
- 52岁创业 | 标准化人才复制 | 特许经营模式 | 汉堡大学

## 关键履历
- **1902年** 出生，芝加哥人
- **1922-1954年** 纸杯推销员（17年），后转做多头搅拌机推销
- **1954年** 52岁时在圣伯纳迪诺遇到麦当劳兄弟，签下特许经营合同
- **1955年** 在德斯普兰斯开出第一家自己的麦当劳餐厅（首日销售额$366.12）
- **1961年** 以270万美元买断麦当劳兄弟全部股权；同年建立汉堡大学
- **1965年** 麦当劳上市（IPO价$22.5→当日$30）
- **1974年** 买下圣地亚哥教士棒球队（$4200万+15年分期）
- **1984年** 去世，享年81岁

## 对Carl培训业务的核心价值

### 1. 标准化体系的教科书
克洛克不是"发明了汉堡"，而是"发明了把人培训成标准机器的系统"。QSCV+营运手册+汉堡大学+持牌人筛选 → 就是一套完整的企业人才复制系统。

### 2. 52岁创业的叙事力
任何场合讲这个故事都是王炸。Carl给央企客户讲培训时，这个案例比任何培训理论都强。

### 3. 七个可复用的方法论
- **持牌人筛选漏斗** → 属地化人才认证
- **QSCV体系** → 课程标准化SOP
- **汉堡大学** → 企业大学搭建咨询
- **边缘和角落理论** → 新兴市场进入策略
- **经营在你，管理在我** → 总部-属地分权的管理哲学
- **持牌人驱动创新** → 培训ROI量化案例
- **抢话筒事件** → 涉外演讲的真实感训练

## 雷·克洛克的核心能力: 30年推销员生涯的谈判洞察

克洛克在52岁前做过30年推销员——纸杯（17年）、Prince Castle多头奶昔机（13年）。这段经历塑造了他独特的谈判/说服风格：

### 推销员的五条法则
1. **先观察，后说话** — 1954年第一次去麦当劳餐厅，不急着谈生意而是站在停车场观察了2小时
2. **用提问代替推销** — "如果不介意的话，能告诉我你多久来这里一次吗？"→用市场调研的幌子了解客户
3. **把自己放在对方的立场** — 跟Sonneborn谈判时："我付不起薪水"→对方算出底线→双方满意
4. **让对方先出价** — 收购麦当劳兄弟时让兄弟先开价→拿到了对方的心理价位
5. **用数据代替情绪** — Fred Turner涨薪谈判时逐项算差额成本

### 对Carl业务的启发
- 商务谈判课程应该包含"推销员思维"——谈判本质是"帮对方看到他没看到的选项"
- 涉外沟通中，提问比陈述更有力量——克洛克用市场调研的方式探测客户需求
