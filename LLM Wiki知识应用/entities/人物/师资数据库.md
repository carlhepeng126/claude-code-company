# 师资数据库

> 状态：共创中（2026-05-16 启动）
> 用途：product-expert / project-delivery Agent 核心数据源
> 关联B卡：B001-教师规范管理

## 字段说明

| 字段 | 说明 |
|------|------|
| 姓名/代号 | 中文名或英文名 |
| 角色 | 主讲 / 行业专家 / 外教 / 方法论顾问 / PM |
| 擅长领域 | 核电英语、商务谈判、跨文化、领导力、B2B营销、工程英语等 |
| 行业经验 | 服务过的行业（核电/工程/能源/汽车/零售等） |
| 已交付项目 | 客户名+项目类型 |
| 学员反馈 | 好评 / 中评 / 待观察 |
| 报价档位 | 高 / 中 / 低 |
| 可用性 | 活跃 / 偶尔可用 / 已停用 |
| 备注 | 特殊优势或注意事项 |

## 角色配置模式（来自B025）

| 模式 | 适用 | 角色构成 |
|------|------|---------|
| 单人专家 | 小型培训 | 主讲+项目统筹 |
| 双人组合 | 中型培训/轻咨询 | 方法论专家+行业专家 |
| 三人项目组 | 组织咨询 | PM+方法论+行业专家 |
| 五人项目组 | 大型混合 | PM+方法论+行业+实施顾问+客户对接 |
