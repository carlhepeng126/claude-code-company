# 财经杂志素材索引

## 概述

《财经》杂志为综合财经类核心期刊，8个文件按年度合集组织（2022-2025年），每个合集约2万行。内容涵盖宏观经济、资本市场、政策分析、产业报道等。

## 内容特征（基于2025年合集采样）

2025年第10期封面为"中国资产重估——关税战引发新的不确定性"，典型内容分布：

| 板块 | 说明 |
|------|------|
| 封面文章 | 宏观经济与资本市场深度分析 |
| 社评 | 政策评论（如"服务消费再发力"） |
| 经济全局 | 各省经济数据、政策效果分析 |
| 财经特稿 | 深度报道（如"面对关税战，最该做什么？"） |
| 特别策划 | 行业/企业专题 |
| 理性投资 | 投资策略与市场分析 |

## 与出海培训的关联点

- **关税战/贸易政策**专题可为企业出海提供宏观背景理解
- 企业国际化案例报道可能出现在特别策划中
- 但整体内容为**综合财经方向**，与央企国际化人才培养/商务培训的直接交集有限

## 利用建议

- 可作为出海培训方案的**宏观背景素材**（如关税战、供应链重构等议题）
- 不建议作为核心知识萃取来源
- 如需跟踪出海相关政策走向，可选择性关注特稿和封面文章
