# 国企杂志素材索引

## 概述

《国企》是全国党建研究会国有企业党建研究专业委员会指定刊物。内容以**国有企业党建**为核心导向，涵盖党建论坛、特别策划、国企改革、工作研究、经验交流等板块。

## 内容特征

| 板块 | 说明 |
|------|------|
| 卷首语 | 党建/政策评论 |
| 党建论坛 | 基层党组织建设经验 |
| 特别策划/封面聚焦 | 典型央企党建实践专题 |
| 工作研究 | 政治巡视、干部管理、人才培养 |
| 经验交流 | 具体企业党建融入业务实践 |
| 一线报道/党员先锋 | 基层党建案例和人物 |

## 相关性评估

已通过深度抽样（3期）确认：

- **2023-10期**: 卷首语《依靠学习走向未来》涉及培训理念，但整体为党建主题
- **2024-15期**: "青马工程"专题——青年干部培训，间接相关
- **2026合集**: "投资于人"讨论——与培训投入直接相关，但仅占极小篇幅

**多数内容为党建方向，与"国际化/商务培训"的直接交集有限。**

## 利用建议

- 可作为了解央企党建生态和培训投入话语体系的**背景参考**
- 如需撰写面向央企党建口的培训方案，可提取"青马工程""人才兴企"等话语体系
- **不建议作为核心训练语料或知识萃取来源**
