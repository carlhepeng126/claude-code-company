## 框架模型

### 1. 出海人力资源"三条链路"框架
```
数据篇 ──► 看清趋势 ── 中国ODI数据与全球投资格局
政策篇 ──► 守住合规 ── 税收指引 + 数据合规 + 各国劳动法
服务篇 ──► 做强组织 ── EOR + 招聘 + 培训 + 薪酬 + 雇主品牌 + 法务
```

### 2. 名义雇主（EOR）运营模型
```
核心三角关系：
┌──────────┐     法定雇主关系      ┌──────────┐
│  客户企业 │ ◄──────────────────► │  EOR服务商 │
│ (用工管理) │                      │ (法定雇主) │
└────┬─────┘                      └────┬─────┘
     │ 实际管理/业务指挥                │ 雇佣合规责任
     ▼                                  ▼
┌─────────────────────────────────────────────┐
│                  员工                         │
│    (入职 → 在职管理 → 离职)                   │
└─────────────────────────────────────────────┘
```
EOR关键管控维度：
- **雇佣合规与责任边界**：合同条款、工时休假、解雇程序、工会协议
- **雇主成本可控**：对账、审计、预算管理
- **员工体验与跨文化一致性**：入职沟通、福利解释、政策告知
- **数据隐私与知识产权保护**：跨境数据合规
- **交付稳定性与可规模化运营**：多国并行、跨时区协作

### 3. 全球人才招聘"三维一体"解决方案模型
```
战略洞察层 ── 定制区域人才画像模型（精准度达行业平均1.8倍）
生态链接层 ── "当地顶级猎企 + 华人商会 + 高校智库"寻访网络
组织赋能层 ── "沉浸式合规渗透"培养海外雇佣合规意识
```

### 4. 全球用工模式选择矩阵
| 企业状态 | 推荐模式 | 核心价值 |
|---------|---------|---------|
| 无海外实体，快速启动 | **EOR（名义雇主）** | 无需设立实体，快速合规搭建团队 |
| 已有海外实体，需深度运营 | **PEO（专业雇主组织）** | 分散行政审计风险 |
| 灵活用工/项目制 | **COR（承包商管理）** | 灵活用工，多币种支付 |
| 成熟市场，本地化运营 | **本地雇佣/自建实体** | 属地化社保与税收优化 |

### 5. 学习发展（L&D）全球化框架（aTalent模型）
```
四大场景：
├── 内部人才发展 ── 个性化学习路径 + 社交性学习 + 定制课程
├── 销售及经销商培训 ── 模拟场景 + 案例研究 + 实战训练
├── 客户支持与教育 ── 产品知识 + 品牌一致性传递
└── 外部生态培训 ── 多语言集成 + 全球学习数据统一
关键能力：跨文化适配 + 数据合规（FDA/PIPL）+ 多HRIS系统对接
```

## 关联概念

| 概念 | 说明 | 关联度 |
|------|------|--------|
| **名义雇主（EOR）** | 企业无需在目标国设立实体，由EOR服务商作为法定雇主完成合规雇佣 | 核心 |
| **专业雇主组织（PEO）** | 企业已有海外实体，将HR职能外包给PEO进行共雇佣(co-employment) | 高 |
| **全球本土化（Glocalization）** | "全球思考，本地行动"——用工策略从总部派驻转向本地化雇佣 | 高 |
| **跨境数据合规** | GDPR、PIPL、CPRA等数据隐私法规对跨境用工的合规要求 | 高 |
| **跨文化管理（Cross-cultural Management）** | 解决文化差异带来的管理摩擦，提升全球团队协作效率 | 高 |
| **全球人才招聘（Global Talent Acquisition）** | 跨国人才搜寻、筛选、录用的完整流程体系 | 高 |
| **ODI（对外直接投资）** | 中国企业海外投资流量、存量及全球位次数据 | 中 |
| **OECD全球最低税率** | 15%全球最低企业税率协议对出海企业税务架构的影响 | 中 |
| **常设机构（PE）风险** | 跨境用工中承包商模式可能触发税务常设机构认定 | 中 |
| **雇主品牌全球化** | 在全球范围内塑造统一且有吸引力的雇主品牌形象 | 中 |
| **灵活用工/远程办公** | 后疫情时代远程办公常态化对全球劳动力市场的影响 | 中 |
| **HR SaaS/HR数字化** | 全球一体化人力资源软件或信息系统（如eRoad、ADP） | 中 |
| **合规成本管理** | 用工合规成本占人力总支出25%-35%的结构性攀升趋势 | 中 |
| **出海人才画像** | 根据目标市场定制的国际化人才能力模型和招聘标准 | 中 |
| **外派人员管理** | 海外派驻员工的签证、税务、社保、文化融入等全周期管理 | 中 |
| **名义雇主/数据合规双重治理** | 跨国医药/金融行业中EOR服务与数据合规的协同实践 | 中 |

## 来源与服务商概览

本指南由**HRflag（上海众旗信息咨询有限公司）**编撰，数据来源包括：
- 中华人民共和国商务部、国家统计局、国家外汇管理局《2024年度中国对外直接投资统计公报》
- 联合国贸发会议《2025世界投资报告》
- 世界经济论坛2025报告、国际劳工组织(ILO)、麦肯锡全球研究所(MGI)
- 德勤(Deloitte) 2025全球税务报告、PwC 2025全球合规调查、EY全球劳动力报告

**核心服务商**：
- **外企德科（FESCO Adecco）**：覆盖全球60+国家，服务客户26,000+家，全国服务人数200万+
- **aTalent**：学习发展解决方案领导者，支持10+语言、跨文化多区域实施
- **万领钧（Knit People）**：EOR/PEO/COR模式，覆盖187个国家/地区
- **易路（eRoad）**：AI HR SaaS平台，覆盖180+国家，服务近1000家客户
- **舒仕福（Safeguard Global）**：全球EOR先驱，187个国家/地区
- **必博人力（BIPO）**：60+直营主体，覆盖172国家/地区
- **安德普翰（ADP）**：全球薪酬与HR管理领导者
