# 跨文化商业情报系统

> 223个国家的结构化商务文化情报 → 直接服务于企业出海培训方案设计与课程开发

---

## 系统架构

```
┌─────────────────────────────────────────────────────┐
│                  数据层（raw/文化指南）                │
│  223国 × 20篇/国 = 3,615篇结构化文章                 │
│  4大类（印象/关系/会议/协作）                         │
└──────────────────────┬──────────────────────────────┘
                       ↓
┌──────────────────────┴──────────────────────────────┐
│                  分析层（跨文化模式识别）              │
│  区域聚类（中东/东南亚/非洲/拉美/中亚/欧洲）          │
│  主题横评（决策×文化维度/谈判×文化维度）             │
└──────────────────────┬──────────────────────────────┘
                       ↓
┌──────────────────────┴──────────────────────────────┐
│                  应用层（R.I.S.E.映射）               │
│  R模块：跨文化认知 + 合规案例                        │
│  I模块：跨文化互动 + 关系建立                        │
│  S模块：跨文化表达 + 商务场景话术                    │
│  E模块：跨文化决策 + 沙盘推演                        │
└─────────────────────────────────────────────────────┘
```

---

## 核心价值

### 对销售提案的价值
- 每个S级客户国家都有**现成的文化深度分析**可直接引用
- 巴拿马文化→中交合规培训提案话术
- 伊拉克文化→北方工业安全培训逻辑
- 坦桑尼亚文化→坦赞铁路人才培养方案依据

### 对课程开发的价值
- **20个标准化话题**（Dress Code/DM/Negotiating等）可直接转化为课程模块
- 每个话题都有223个国家的对比素材
- 可设计"国家对比工作坊"：同一话题在A国vsB国的差异

### 对R.I.S.E.体系的价值
| R.I.S.E.模块 | 文化指南素材支撑 | 应用方式 |
|:-----------:|----------------|---------|
| R - Recognize | Making a Good Impression全系列 | 文化差异认知基础课 |
| I - Interact | Managing Relationships全系列 | 跨文化互动沙盘 |
| S - Speak | Meetings & Presentations系列 | 商务展示/谈判话术 |
| E - Execute | Working Together系列 | 跨文化决策推演 |

## 与100家央企机会的交叉关联

### 决策模式分类（直接影响海外项目推进）

| 决策模式 | 典型国家 | 对央企的启示 |
|---------|---------|------------|
| **共识协商型** | 印尼/马来西亚/坦桑尼亚 | 项目决策需要预留额外沟通时间 |
| **层级执行型** | 沙特/阿联酋/安哥拉 | 高层关系比方案质量更重要 |
| **关系驱动型** | 伊拉克/黎巴嫩 | 政治派别/部落网络影响决策 |
| **混合适应型** | 巴拿马/墨西哥 | 城市快节奏 vs 乡村慢节奏需区分 |

### 谈判模式分类（直接影响商务谈判培训）

| 谈判模式 | 典型国家 | 培训要点 |
|---------|---------|---------|
| **关系优先** | 沙特/阿联酋/伊拉克 | 先建立信任再谈条款 |
| **面子敏感** | 东南亚（马/印/泰/越） | 避免公开逼YES/NO |
| **尊严保护** | 北非（阿尔及利亚/摩洛哥） | 给对方留退路 |
| **务实直接** | 拉美（巴拿马/巴西） | 准备谈运营可行性而非纯理论 |

## 使用方法

**在飞书对话中查某国文化信息时：**
→ 访问 `raw/文化指南/{国家}/` 下的20篇文章
→ 优先读取：Decision-Making / Negotiating / Business Meeting Culture
→ 结合100家央企对应企业信息输出

**在课程设计中使用时：**
→ 选取目标国家的对比文章（如中国 vs 沙特 Negotiating）
→ 提取文化逻辑差异点 → 设计课堂讨论题
→ 引用本地人视角增加说服力
