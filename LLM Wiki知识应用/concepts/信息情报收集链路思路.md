# B000情报搜集全链路设计

情报的收集主要有如下的难点：
1、信息源怎么来？
2、信息源的基础上如何来丰富？
3、如何通过技术实现自动化？
4、信息搜集后如何分析萃取知识？（情报的搭建逻辑）

## 核心方案

已建成完整的三层情报管道架构，详见完整文档：

> 📄 **`[[LLM Wiki知识应用/concepts/intelligence-collection-pipeline|情报搜集链路设计]]`**

### 链路口诀

```
三层管道：采集 → 处理 → 输出
四级源：RSS → HTML → 本地 → 人工
六路输出：日报 → 周报 → Dashboard → 简报页 → 知识库 → 核心卡
日常维护：健康检查 → 排坑 → 格式同步 → 知识萃取
```
本地主要通过：
Opencli+Qclaw+Obsidian+ AI实现自动化
