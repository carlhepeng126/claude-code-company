# 🦐 深度学习萃取方法论

> 当 Carl 说「帮我深度学习萃取」时调用的标准化流程。
> 任何素材（书籍/PDF/文章/文档）→ 情报虾 + 爆红虾 + 知识库 + 培训应用四个层面同时萃取。

## 触发条件

- 「帮我深度学习萃取」
- 「帮我消化一下这个」
- 「情报虾和爆红虾都深度学一下」
- 「让我的情报虾/爆红虾深度学习 XX」

## 完整萃取流程（四层）

### 第0层 — 前置准备

**身份验证：** 先确认读的是什么（书名/作者/类型），防止错误识别

**类型识别（决定各层权重）：**
- 📖 **书籍型**（传记/商业案例）→ 情报虾偏⑤海外项目+④演讲谈判；爆红虾偏故事型痛点
- 📄 **制度型**（政策/规划）→ 情报虾偏⑥央企动态+①外语培训；爆红虾偏数据型痛点
- 📊 **报告型**（调研/白皮书）→ 情报虾偏⑦行业趋势+②跨文化；爆红虾偏诊断工具型

**文件格式识别：**
- `.pdf` → pdftotext 优先
- `.pdf` 扫描件 → tesseract OCR (chi_sim+eng)
- 图片型 Markdown（公众号）→ OCR 流水线
- 混合型 Markdown（有正文也有图片）→ 直接读取正文

**内容安全过滤：** 检查 PUA/搭讪/性别刻板印象等不适合企业培训的内容，整章跳过

### 第1层 — 情报虾深度萃取（7大方向）

| # | 方向 | 输出要求 |
|---|------|---------|
| ① | 企业外语培训 | 可转化为培训方法论/案例/工具 |
| ② | 跨文化 | 国际化/跨文化元素 + 课程案例 |
| ③ | 属地化 | 当地化人才培养启示 |
| ④ | 涉外演讲与谈判 | 实战案例 + 可拆解的方法 |
| ⑤ | 海外项目 | 项目管理/扩张策略 → 培训模型 |
| ⑥ | 央企动态 | 与央企出海人才培训的对接 |
| ⑦ | 行业趋势 | 与 L&D 趋势的关联 |

每方向至少 3 条可落地洞察，格式：素材 X → Carl 业务 Y → 具体行动 Z

### 第2层 — 爆红虾深度萃取（3篇帖子）

每篇公式：**普遍痛点 → 诊断工具 → 方案 → CTA**

3 篇差异化定位：
| 帖子 | 目标 | 核心痛点 |
|------|------|---------|
| 帖子一 | 培训负责人 | 培训落地难（TOP5 #033） |
| 帖子二 | 企业高管 | 培训 ROI（TOP5 #013 虚假自信） |
| 帖子三 | 培训从业者 | 职业焦虑（TOP5 #007 验收说不出） |

### 第3层 — 知识库补充

- 已有实体 → 做 diff，只补增量
- 新概念 → 创建 concept 卡片
- 更新 log.md + index.md
- 跨素材关联检查

### 第4层 — 培训应用补充

输出 R.I.S.E. 框架映射表，识别 P0 级案例（可直接进入标准版课程的素材），设计服务产品。

## 关键约束

### 子代理超时容错
- 子代理连续 2 次超时 → 立即切换手动直写，不继续重试
- 3000+ 行文本 → 直接手动，不试子代理
- 手动模式：主代理直接 `write_file` 5 个文件到 `/tmp/`

### 叙事型 vs 知识型素材
- **叙事型**（传记）：先通读全书 → 理解故事弧线 → 再系统萃取
- **知识型**（报告/白皮书）：可边读边打标分桶

### 跨素材合成
- 每本书独立产出 5 个文件（buckets/qingbao/baohong/knowledge/training）
- 全部完成后做一次跨素材关联分析
- 发现「全栈结构」（各素材覆盖沟通/培训的不同层级）→ 设计体系化服务产品

### 打标分桶（关键中间步骤）
- 读完文本后，先不写产出，先把所有可用的素材按 7 大方向分桶
- 每桶至少 3 条素材才出洞察，不足则精简

> ⚠️ **陷阱：** 跳过打标分桶直接写产出 → 遗漏关键素材。读完全文后必须先做这一步。

## 多集群编排

适用于 50+ 篇批量素材的批量萃取：

1. **Phase 0 分群** — 按主题分成 3-6 个集群（每集群 8-15 篇）
2. **Phase 1 逐集群萃取** — 每集群走完整四层
3. **Phase 2 并行小集群** — ≤8 篇的用 delegate_task 并行
4. **Phase 3 全集群合成** — 输出跨集群合成报告
5. **Phase 4 memory 维护** — 集群边界清理 memory

## 验收标准

完成后自问：
- ✅ 素材类型识别正确？各层权重调整了？
- ✅ 做了打标分桶？
- ✅ 7 大方向每个都有洞察？
- ✅ 3 篇帖子有完整痛点+方案+CTA？
- ✅ 知识库无遗漏维度？log 更新了？跨素材关联做了？
- ✅ 培训应用有新案例/数据/金句/服务产品？
- ✅ **Carl 能用这个做什么？**（答不上 → 重做）
