# 情报虾 · 情报验证体系

## 核心原则
每条情报必须可追溯、可验证、可落地。不允许AI编造数据。

## 验证标准
1. 来源可追溯到具体媒体/官网报道
2. 含明确的企业名+时间+规模+数据
3. 排除AI编造和通用趋势描述
4. 每条标注可用场景（提案/销售工具/销售线索/课程设计/竞品对标）
