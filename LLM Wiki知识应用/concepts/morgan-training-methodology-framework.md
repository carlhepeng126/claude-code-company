# 摩根世纪核心方法论框架

> Carl/Morgan世纪在多个客户项目中沉淀的9个核心方法论。可直接复用于新客户方案设计、产品开发、项目交付。

---

## 一、项目级方法论

### 1. OMO混合学习模式
**出处**：中交海外公司（第三期项目）、AXA集团项目
**核心**：线上预习 → 线下实战 → 线上巩固 → 实战考核
**关键设计**：
- 1对1外教约课（高频小班/1v1，每次25min）
- 在线课程（自主学习，AI推荐路径）
- 班主任/教练跟进（过程管理）
- **优势**：灵活适配不同岗位、不同时区的学员

### 2. 三层目标拆解法
**出处**：项目背景分析与方案设计SOP
**核心**：战略层 → 业务层 → 结果层
**应用**：从客户企业战略出发，逐层拆解培训需求，确保培训设计与企业战略对齐
**关键问题链**：
- 战略层：企业当前最大挑战是什么？国际化处于什么阶段？
- 业务层：哪些业务节点因沟通力不足而卡住？
- 结果层：培训后什么行为/指标的改变代表成功？

### 3. 五步深度定制法
**出处**：AXA集团项目、设计背景总纲
**核心步骤**：场景萃取 → 内容设计 → 课程开发 → 训练设计 → 考核设计
**特点**：不做通用课，每门课从客户真实业务场景出发

### 4. 项目方案设计SOP（4轮16问）
**出处**：项目背景分析与方案设计SOP_v2.4
**流程**：
1. 信息收集（4问：客户/背景/预算/目标）
2. 需求分析（4问：人群/场景/水平/痛点）
3. 方案设计（4问：模式/内容/师资/考核）
4. 输出物（3模块+19项质检查）

### 5. 内部行文格式规范
**出处**：摩根英语项目方案内部行文格式规范
**核心**：项目方案的标准结构、标题层级、字体规范、附件格式

## 二、内容设计方法论

### 6. 场景驱动术语（Scenariao-Driven Terminology）
**核心发现**：术语培训单独效果有限（遗忘率70-80%），必须嵌入场景培训才有效
**正确方式**：不是先学术语再学场景，而是在场景中学
**实现路径**：术语作为"知识胶囊"嵌入场景课程，不做独立术语课
**数据支撑**：场景模拟+AI陪练（★★★★★）> 混合式（★★★★）> 微学习（★★★★）> 纯术语培训（★★）

### 7. IAI框架（融合与归化指数）
**出处**：海外建厂隐性壁垒方法论
**核心**：量化评估跨文化沟通与合规风险
**应用场景**：企业出海前的文化风险诊断、跨文化培训需求评估

### 8. 审计问答"肌肉记忆"训练
**出处**：医药企业欧美审计合规沟通方案
**核心**：问题识别 → 分级标准 → 调查依据 → 纠正措施 → 有效性验证 → 记录位置
**训练方式**：1对1外教高压角色模拟（模拟审计官连续追问/打断/换角度提问）
**效果**：回答稳定性提升、证据导向增强、表达具体化

## 三、战略层框架

### 9. 设计背景总纲（五大结构性障碍）
**出处**：设计背景-从产品出海到全球运营的战略赋能
**五大障碍**：
1. 语言能力断层（能读不能讲）
2. 跨文化盲区（听不懂"弦外之音"）
3. 组织协同失灵（总部指令无法落地）
4. 属地化人才缺失（"空降兵"文化冲突）
5. 合规风险认知不足

## 四、方法论选用指南

| 场景 | 推荐方法论 | 说明 |
|------|-----------|------|
| 新客户方案设计 | SOP（4轮16问）+ 三层目标 | 快速对齐需求 |
| 课程开发 | 五步深度定制 + 场景驱动术语 | 内容精准匹配 |
| 混合式项目交付 | OMO模式 + 过程考核 | 适配多时区团队 |
| 跨文化诊断 | IAI框架 | 量化风险 |
| 合规审计培训 | 肌肉记忆训练 | 高压场景专项 |
| 方案汇报 | 设计背景总纲 | 客户高层沟通 |
