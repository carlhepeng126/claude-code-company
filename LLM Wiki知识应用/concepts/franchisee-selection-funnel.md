# 🧪 持牌人选拔漏斗模型

## 定义
麦当劳持牌人选拔的完整流程，以"意愿筛查优先于能力评估"为核心理念的五步漏斗。**关键洞察：把70%的筛选工作前置到"让人自己退出"**，而非面试考核。

## 漏斗流程

```
              申请（海量）
                 ↓
    ┌────── 附近餐厅试工（夜班+周末）──────┐
    │    ← 体验真实运营强度，筛掉叶公好龙者    │
    └──────────────────────────────────┘
                 ↓
        ┌───── 4000美元押金──────┐
        │ ← 经济付出感，筛掉犹豫者  │
        └─────────────────────┘
                 ↓
        ┌──────── 排队等待（最长2年）───────┐
        │ ← 时间检验，筛掉一时冲动者，留在最坚定者 │
        └────────────────────────────────┘
                 ↓
         ┌─── 看地点 + 500小时实操 ────┐
         │ ← 真实环境下的能力验证    │
         └────────────────────────┘
                 ↓
         ┌── 汉堡大学高级课程 ──┐
         │ ← 系统化最终考核    │
         └──────────────────┘
                 ↓
               开业
```

## 对Carl的价值
**核心卖点**：给客户做外派人员选拔方案时，把"测评/面试/背景调查"这套传统流程换成**"意愿驱动的漏斗"**——让候选人自己筛选自己，而非HR坐在那里猜谁适合。一个"海外体验周"就能筛掉50%的不适合者。

## 关键设计原理
1. **意愿 > 能力** — 能力可以培训，意愿无法注入
2. **真实体验 > 面试评价** — "做一周"比"聊一小时"准确10倍
3. **沉没成本筛选** — 投入越多的人越认真，越有风险就越谨慎

## 应用场景
- 外派人员选拔体系设计
- 海外合伙人/本地管理者招聘流程
- 高管海外轮岗筛选
- 国际化人才培养项目入口设计
