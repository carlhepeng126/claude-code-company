# 项目背景分析与方案设计SOP（问答式 v2.4-2.5）

## 框架概述

针对国际外语培训机构新项目场景设计的标准化方案设计流程。通过 4 轮 16 个问题（v2.4 为 16 题 → v2.5 优化精简版）引导新人/顾问/销售系统化收集客户信息，AI 自动搜索客户背景，输出结构化方案。覆盖从战略推导到场景定制的全流程，核心输出 3 个模块：**客户目标拆解 + 培训模式设计 + 场景定制设计**，附加 19 项新人检查清单。

适用对象：新手策划人员、顾问、销售
适用场景：国际外语培训机构新项目方案设计、客户培训需求调研和分析、跨文化培训项目设计

## 核心维度和步骤

### 1. 信息收集阶段 — 4 轮 16 个问题

| 轮次 | 主题 | 问题数 | 核心产出 |
|------|------|--------|----------|
| 第一轮 | 项目背景 | 9 | 客户名称/行业/阶段 + AI搜索战略→关键节点→培训需求 + 挑战梳理 |
| 第二轮 | 客户目标拆解 | 6 | 培训目标 + 三层目标拆分（战略层/业务层/结果层）+ 五步深度定制法 |
| 第三轮 | 培训模式设计 | 4（v2.4）→3（v2.5） | 线上/线下/OMO + 内容类型 + 时长 |
| 第四轮 | 场景定制 | 1+灵活提问 | 岗位-场景映射 + 贯穿场景 + 实战任务 + 特殊定制 |

**AI 主动搜索机制**（核心特色）：
- 第一轮问题 5-8：AI 根据客户名称自动搜索企业战略方向、关键节点、培训需求相关动态
- 搜索引用必须标注具体来源（如：来源：xx公司官网2024年年报），不得编造
- 新人只需确认或修正搜索结果，无需自行收集信息

### 2. 方案输出阶段 — 3 个核心模块

#### 模块 1：客户目标拆解
- **三层目标结构**：战略层（支撑什么战略）、业务层（提升什么业务能力）、结果层（达成什么具体结果）
- **五步深度定制法**：场景萃取 → 内容设计 → 课程开发 → 训练设计 → 考核设计
- **成功标准**：知识掌握 → 技能提升 → 行为改变 → 业务影响，四层次评估

#### 模块 2：培训模式设计
- 形式选择：线上 / 线下 / 线上线下结合（OMO）
- 线上内容：课件学习 / 一对一辅导 / AI 陪练 / 其他
- 线下形式：专题培训 / 集训 / 工作坊 / 模拟实战 / 其他
- OMO 典型流程：线上预习 → 线下实战 → 线上巩固 → 实战考核
- 输出：培训路径图（阶段-时间-内容-方式-交付物）+ 资源投入清单

#### 模块 3：场景定制设计
- **岗位-场景映射表**：岗位层级 → 场景类型 → 具体场景 → 培训重点
- **贯穿性场景设计**：跨培训阶段的场景主线（如 HSE 安全场景贯穿语言筑基期→融合实战期）
- **实战任务设计**：任务描述 + 目标 + 完成方式 + 交付物 + 评估标准
- 客户明确场景 ≥3 个；如客户未提供，基于岗位主动推断

#### 模块 4：特殊定制要求（可选）
- 跨文化协作机制设计
- AI 工具赋能场景
- 其他客户特殊需求

### 3. 新人检查清单 — 19 项
- 基础信息（5 项）：公司名称/行业正确、项目阶段明确、战略推导路径清晰、机遇窗口期、来源标注
- 目标拆解（5 项）：目标清晰、三层结构完整、KPI 可衡量、五步法说明、成功标准
- 培训模式（4 项）：形式明确、线上内容、线下形式、时长合理
- 场景定制（5 项）：场景映射表完整、场景数量 ≥3、贯穿场景、实战任务、来源可追溯

### 4. 信息收集表（可复用的客户信息采集模板）
统一的表格模板，涵盖：客户基本信息、战略推导与机遇识别、客户目标与三层拆解、培训模式设计、场景定制设计、特殊定制要求。

## 关键方法论

### 战略推导路径
```
企业战略目标 → 关键里程碑 → 培训需求
```
基于 AI 搜索获取客户公开信息，推导出"为什么需要培训，如何支撑战略目标"的逻辑链。引用来源必须可追溯。

### 三层目标拆解框架
| 层级 | 关注点 | 示例 |
|------|--------|------|
| 战略层 | 支撑什么战略、价值贡献 | 支撑芬兰基地投产与"产能出海"战略 |
| 业务层 | 提升什么业务能力 | 工作英语、跨文化协作、HSE 合规 |
| 结果层 | 达成什么具体结果、KPI | 中方团队能独立与芬方技术交流 |

### 五步深度定制法
1. **场景萃取** → 提炼岗位真实工作场景
2. **内容设计** → 根据场景设计培训内容
3. **课程开发** → 开发场景化课程
4. **训练设计** → 设计实战训练方式
5. **考核设计** → 设计场景化考核标准

### OMO 模式设计框架
```
线上预习 → 线下实战 → 线上巩固 → 实战考核
```

## 对 Carl 团队的应用建议

1. **作为新人培训标准教材**：此 SOP 可直接用于新人策划人员/顾问/销售的 onboarding 培训，新人拿到后即可按流程产出方案
2. **嵌入项目交付虾的工作流**：可将此 SOP 的 4 轮 16 问作为项目交付虾与客户沟通的标准化提问框架，确保每个项目都覆盖战略推导、目标拆解、模式设计、场景定制四个维度
3. **AI 搜索机制接入**：SOP 中的"AI 主动搜索+新人确认"模式可直接复用到项目交付虾的自动化流程中——AI 收到客户名后先搜索企业战略/关键节点/培训需求，再引导对话
4. **作为客户方案模板参考**：模块 1-3 的输出格式（含表格、路径图、清单）可直接作为方案文档的标准模板
5. **持续迭代机制**：SOP 已到 v2.4-2.5 版本，可继续在指导库中维护版本记录，每次迭代后更新此卡片

## 最佳实践举例

**当升科技芬兰基地培训项目（SOP 自带案例）**
- 场景：芬兰基地团队融合培训项目，需求调研阶段
- AI 搜索结果：当升科技实施"产能出海"战略 → 芬兰基地预计 2026Q4 投产 → 培训需求为语言沟通、文化融合、HSE 合规
- 三层目标分拆：战略层支撑产能出海 → 业务层提升英语/跨文化/HSE → 结果层中方团队能独立与芬方协作
- 五步定制法：从芬兰基地真实岗位场景出发，贯穿语言筑基→融合实战两阶段
