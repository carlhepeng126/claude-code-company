# 企业涉外商务谈判能力框架

> 整合自客户洞察知识库：出海相关报告、国际观察、东南亚区域、企业培训杂志、R.I.S.E.解决方案等。
> 核心结论：出海竞争正在从「产品力之争」转向「沟通力之争」。

## 一、涉外商务谈判六大问题层次

### 1.1 信息与情报层面
| 问题 | 表现 | 来源 |
|------|------|------|
| 信息搜集弱 | 不掌握六大情报渠道（公开情报、人脉网络、在地调研、社交媒体、第三方数据库、内部资源） | 国际市场开发培训 |
| 市场分析弱 | PESTEL、波特五力、SWOT工具使用不达标 | 中原对外工程培训方案 |
| 竞品动态滞后 | 难以预判竞争对手投标策略 | 吉林乙醇直供日韩案例 |
| 情报网络缺失 | 未在目标国建立在地情报网络 | 石悦华中东实战 |

### 1.2 战略与策划层面
- 出海战略顶层设计三问缺失：「为什么出海？」「去哪里？」「怎么去？」
- 混淆四种出海模式适用条件（贸易出口/技术授权/合资公司/独资并购）
- 客户需求停留在显性需求，无法挖掘隐性需求和个人诉求（尤其在中东、东南亚市场）
- 缺乏竞争屏蔽策略（引导采购标准、捆绑独家资源等）

### 1.3 合同与法律层面
- FIDIC合同条款（红皮书/黄皮书/银皮书）风险分配机制掌握不足
- 索赔意识薄弱（工期延误/费用增加时未及时发出通知）
- 投标报价策略运用不当
- 国别风险、汇率风险、合规风险识别不充分

### 1.4 沟通与协调层面
- 结构化汇报能力弱，无法在高层面前清晰表达战略方案
- 市场人员与技术、法务、财务部门间协调困难
- 对突发情况（签证被拒、联系人离职、政策调整）缺乏预案

### 1.5 跨文化谈判层面
- 中东场景：Wasta信任体系、斋月影响、沙特IKTVA本地化率要求
- 场景差异：央企与政府间的谈判模式 vs 与私企的谈判模式
- 团队分工：谁负责价格、谁负责关系、谁负责合同条款

### 1.6 人才培养与培训采购层面
- 供应商选择困难（央企对培训供应商的评估标准模糊）
- 需求与供给错配（"想要国外名师但预算只有国际机票钱"）
- 培训效果难量化（"培训了3个月，到底有没有用？"）
- 跨文化培训效果验证困难

## 二、HR场景九大谈判核心场景

| 场景 | 核心挑战 | 代表案例 |
|------|---------|---------|
| 海外人才招聘谈判 | 薪酬标准难对齐、招聘渠道有限、外派谈判复杂 | 瑞幸分群体精准招聘、大众汽车法国创意招聘 |
| 跨文化团队管理冲突 | 语言障碍、价值观冲突、信任缺失 | M铜矿中厄跨文化融合（中籍181人+厄籍1176人） |
| 外籍员工沟通协作 | 沟通方式差异、文化敏感性不足、隐私边界 | 华为HRSS全球统一管理平台 |
| 跨国绩效考核与激励 | 绩效标准差异、远程评估难、激励效果差异 | 华为全球绩效管理体系（SMART原则定制） |
| 海外派遣与薪酬福利 | 派遣成本高、税务合规复杂 | — |
| 劳动合规与解雇谈判 | 不同国家解雇保护法差异大、遣散成本不可预测 | 各国解雇保护法对比 |
| 内部人才发展沟通 | 晋升通道模糊、轮岗计划沟通不畅 | 华为"之"字形人才培养路径 |
| 劳资关系与工会谈判 | 海外工会力量远超国内经验、罢工风险 | 西班牙/马耳他工会案例 |
| 外部培训采购谈判 | 供应商选型难、效果难量化、长期合作建立 | — |

## 三、培训采购谈判三大痛点

| 痛点 | 严重程度 | 典型场景 |
|------|:--------:|---------|
| 供应商选择困难 | ⭐⭐⭐⭐ | 央企采购流程冗长，培训供应商评估标准模糊 |
| 需求与供给错配 | ⭐⭐⭐⭐⭐ | 预算与期望不匹配（"万元预算提出百万级别需求"） |
| 培训效果难量化 | ⭐⭐⭐⭐ | 短期培训效果难以量化，ROI论证困难 |

## 四、Carl业务价值

- **谈判能力框架可做诊断工具**：帮助客户识别在6个问题层次中哪个是短板，对应推出培训方案
- **9大HR谈判场景可直接课程化**：每个场景都是半天工作坊的天然素材
- **跨文化沟通切入点**：中东/东南亚/欧洲三个典型场景对应不同的文化沟通策略
- **培训采购谈判痛点**：Carl在做客户咨询时，可以精准预判对方对培训采购的核心顾虑（效果量化、供应商选型）
- **组合产品**：涉外商务谈判能力框架 + R.I.S.E.解决方案 = 涉外沟通力完整能力栈
