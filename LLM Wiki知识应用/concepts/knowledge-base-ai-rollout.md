# 知识库建设 + AI工具推广

## 核心观点
以知识库作底座 + AI工具作抓手，通过三层次递进建设 + AI竞赛机制驱动双团队效能提升。

## 三层次递进模型

### 第1层：飞书Wiki（第1-2周）
- 5大目录：项目交付、培训体系、客户洞察、团队工具、情报采集
- 每人认领1-2个目录，1周内3-5篇文档
- 强制周报：每人每周五写工作笔记归档

### 第2层：多维表格（第2-4周）
- 项目看板 / 客户漏斗 / 培训排期 / 问题库 / AI工具日志
- 关联Wiki文档，日志自动触达Bot推送

### 第3层：AI赋能（第4-8周）
- 指挥虾自然语言查Wiki
- 情报虾自动采集写入
- AI摘要群聊，提炼写入知识库

## AI工具推广：竞赛驱动
红队（Carl团队）vs 蓝队（小卡团队）积分赛：
- 用AI完成工作 +2分
- 提交工具日志 +1分
- 教同事 +3分
- 出模板 +5分
每周五交叉分享会，月冠军请奶茶

## 工具选型清单
| 工具 | 用途 | 优先级 |
|------|------|--------|
| 飞书（Wiki+多维表格+IM） | 协作底座 | 已有 |
| 指挥虾Bot | AI能力入口 | 已有 |
| 小卡助手Bot | 另一团队AI入口 | 已有 |
| NotebookLM/Obsidian | 个人知识管理 | 推荐 |
| HeyGen | AI视频/课程 | 推荐 |
| DeepL/火山翻译 | 翻译 | 推荐 |

## 双团队协调机制
- 虾兵蟹将群：Bot+全员互通
- 指挥虾不进群，Carl@时进入
- 每周五17:00推送双团队周报

## 落地时间线
第1周→建Wiki+AI培训+挑战
第2周→填内容+建表格+启动竞赛
第3-4周→竞赛+分享会+颁奖
第5-8周→AI赋能+制度化+季度盘点
