## 二、FIDIC合同管理（风险条款深度拆解）

### 2.1 FIDIC 2017版关键变化
- 工程师中立性加强
- DAB→DAAB（争议避免/裁决委员会，强调事先避免争议）
- 索赔通知28天→21天（2017版黄皮书）
- 扩大承包商暂停权和终止权

### 2.2 合同关键人员（FIDIC黄皮书6.12款）

| 岗位 | FIDIC称谓 | 核心职能 |
|:---:|:--------:|---------|
| **项目经理** | Contractor's Representative | 项目整体管理，承包商唯一代表 |
| **设计负责人** | Design Manager | 设计全过程管理，确保符合合同与技术规范 |
| **施工经理** | Construction/Site Manager | 现场组织、协调各施工队伍 |
| **采购经理** | Procurement Manager | 设备、材料、服务采购 |
| **QA/QC经理** | QA/QC Manager | 质量管理体系实施与监督 |
| **HSE经理** | HSE Manager | 职业健康、安全、环保、风险控制 |
| **合同/商务经理** | Contract/Commercial Manager | 合同执行、变更索赔、成本控制 |
| **技术负责人** | Technical Director/Chief Engineer | 技术方向决策，应急处置 |
| **调试经理** | Commissioning Manager | 设备调试、试运行管理 |
| **财务经理** | Finance Manager | 财务预算、资金管理、工程保险 |

**关键条款要点：**
- 所有关键人员须全职在现场工作
- 须流利使用合同约定的交流语言
- 工程师有权要求撤换有不当行为的关键人员
- 替换须经工程师事先同意，否则视为违约

### 2.3 索赔管理的三大战略先机
- **当机立断**：知悉事件后28天内发出索赔通知（FIDIC 20.2.1）
- **尽早解决**：索赔解决越早越好
- **多元化手段**：掌握索赔主动权，建立全流程争议管理体系
- **典型翻车案例**：某央企45天通知→丧失3000万索赔权

### 2.4 总包-分包合同风险条款（L&T案例研究）

**四大核心风险规避逻辑：**
1. **责任绝对化与单向化**：通过"视为包含"、"批准不免责"等条款将所有风险转移给分包商
2. **程序控制与时效陷阱**：索赔14天通知期、变更必须书面指令，程序不合规即失权
3. **财务担保闭环**：履约保函(开工前) + 进度款保留金(施工中) + 保函换保留金(缺陷期)
4. **法律地理优势**：选择新加坡法律+SIAC仲裁，国际商事中心替代项目所在地司法

**13类条款类别：** 定义与解释→工作范围→工期与延期→价格与支付→设计与技术→现场条件→变更与索赔→知识产权→检验测试→缺陷责任→保险保障→终止暂停→法律与争议

**Carl培训价值：** 此案例分析是"国际工程合同风险识别"课程的黄金素材，可用于合同管理2日专项课。

## 三、三大金融组织投标规则深度对比

### 3.1 核心定位

| 维度 | 世界银行(WB) | 非洲开发银行(AfDB) | 伊斯兰开发银行(IsDB) |
|:---:|:----------:|:---------------:|:----------------:|
| 总部 | 美国 | 科特迪瓦 | 沙特吉达 |
| 覆盖 | 180+国 | 非洲大陆 | 57伊斯兰国 |
| 核心逻辑 | 规则至上 | 非洲优先 | 宗教+成员国优先 |
| 规则特征 | 铁面、零弹性、强透明 | 规则同源但灵活 | 规则为基、关系加权 |

### 3.2 投标策略差异

| 维度 | 世行标 | 非开行标 | 伊斯兰开行标 |
|:---:|:-----:|:------:|:---------:|
| 评标方法 | QCBS/LCB，价格权重60-80% | 技术权重40-50%，兼顾本地 | 价格权重50-60%，重过往业绩 |
| 谈判空间 | 绝对禁止 | 可合理澄清 | 允许有限谈判 |
| 本地化要求 | 鼓励，无强制 | 本地含量权重极高，须组JV | 成员国企业天然优势 |
| 关系重要性 | 零 | 中 | 极高 |
| 适配策略 | 做"完美合规者" | 做"本地融合者" | 做"圈层参与者" |

### 3.3 实战打法

**世行标 — "完美合规者"**：标书零缺陷、技术保守可靠、精准低价、放弃关系幻想

**非开行标 — "本地融合者"**：必组本地JV（部分项目本地含量≥30%）、突出属地团队与供应链、灵活利用澄清

**伊斯兰开行标 — "圈层参与者"**：提前对接成员国企业（如沙特Alfanar、土耳其Enka）、强化伊斯兰国家业绩背书、遵循伊斯兰金融原则

**共性红线：** 三大机构均执行联合制裁机制，反腐/欺诈/串标终身禁入多边市场；现汇项目须独立承担政治/汇率/法律/通胀风险

## 四、建筑业出海 Dos and Don'ts（毕马威框架）

### 4.1 Dos

**深耕属地化经营（三维推进）：**
- 人才本地化：引入当地中高层，发挥语言、人脉、法律优势
- 供应链区域化：培育周边供应体系，降低远距离跨境依赖
- 社会责任常态化：参与公益、环保、民生项目，树立负责任形象

**构建高标准合规体系：**
- 建立专项合规体制，对标FIDIC、英美法系、当地法规
- 从被动应对转向主动合规

**打造产业生态共同体：**
- 产业链协同出海 > 单一施工出海
- 联盟化出海：湖北建筑产业生态集群出海联盟（2026.4）、东莞建筑产业出海服务联盟（2025.11）

### 4.2 Don'ts

**不简单复制国内管理模式：** 各国建材标准、技术标准、环保要求、工会体系、监管尺度差异显著。粗放式管理易触发合规处罚、停工、索赔。

**不过度依赖单一供应链：** 建立多元化、区域化、有备份的供应体系，前瞻性布局备选资源。

**不相信"低价中标、高价索赔"：** 海外合同严苛的索赔时限（7-28天）和程序要求使"后期找补"策略失效。

## 五、能建国际战略框架（提质增效参考）

"四敢精神" + "八字方针"：敢于拼闯、敢于突破、敢于奉献、敢于胜利 + 守正、创新、实干、担当

**核心举措：**
- **履约+承接双中心**：提升在建履约和"三化发展"（属地化、国际化、市场化）两个竞争力
- **精益化管理**：将精益化理念融入项目管理全过程，推动规范化、精益化、高效化转型
- **联营体模式**："五大主动"管理理念+"五大融合"管理要求
- **"一局一院一平台"模式**：总结深化沙特光伏、乌兹风电等标杆项目
- **大风控体系**：安全生产+资金管控+风险防控+变更索赔

- 基建EPC：多国籍团队管理最关键
