# 摩根英语项目方案内部行文格式规范

## 通用格式
| 项目 | 要求 |
|------|------|
| 字体 | 汉字：仿宋；英文/数字：Times New Roman |
| 行间距 | 固定值24磅（目录除外） |
| 页面起始 | 居中标题前空1～2行（大号字体空1行，小号字体空2行） |

## 标题层级体系

| 层级 | 示例 | 字体 | 字号 | 加粗 | 对齐 | 空行要求 |
|------|------|------|------|------|------|---------|
| 大标题 | 中国中原对外工程有限公司 | 仿宋 | 2号 | 否 | 居中 | 前空2行，后空1行 |
| 方案名称 | "国际市场开发专业能力提升"三天特训营 | 仿宋 | 小2号 | 是 | 居中 | 前空1行，后空1行 |
| 一级标题 | 第一部分：项目设计概览 | 仿宋 | 2号 | 是 | 居中 | 前空2行，后空1行 |
| 二级标题 | （一）课程总览 | 仿宋 | 4号 | 是 | 左对齐 | 前空1行 |
| 三级标题 | 1. 模块1 | 仿宋 | 4号 | 否 | 左对齐 | 紧接上文 |
| 四级内容 | （1）（2）（3） | 仿宋 | 4号 | 否 | 左对齐 | 紧接上文 |

## 标题编号规范（⚠️ 易错点）
| 层级 | 正确 | 错误 |
|------|------|------|
| 一级标题 | 第一部分、第二部分... | 1. 2. 3. ❌ |
| 二级标题 | （一）（二）（三）...（中文括号） | 1. 2. 3. ❌ |
| 三级标题 | 1. 2. 3. ... | （一）（二）（三）❌ |
| 四级内容 | （1）（2）（3）... | 1. 2. 3. ❌ |

## 正文格式
- **字体**：仿宋4号
- **行间距**：固定值24磅
- **首行缩进**：2字符

## 目录格式
- **字体**：仿宋4号
- **行间距**：固定值28磅

## 表格格式
| 项目   | 要求                      |
| ---- | ----------------------- |
| 位置   | 居中，前后留0.5行段间距           |
| 字体   | 仿宋小4号                   |
| 表头背景 | 灰色底色                    |
| 表头字体 | 加粗，居中                   |
| 内容对齐 | 中间行对齐（词汇类居中，文段/句子类两端对齐） |
