# 方案设计指导框架

> 基于摩根世纪2025-2026约70份企业培训方案提炼的方法论体系
> 适用范围：国际化人才外语培训方案设计

---

## 一、五段式标准架构

所有方案均遵循同一宏观结构，不论客户行业、规模、预算：

```
[诊断] → [设计] → [实施] → [管理] → [考核]
```

### 1.1 诊断阶段
| 要素 | 说明 | 必/选 |
|------|------|-------|
| CEFR对标测评 | 在线自适应测评，确定学员基线等级 | ✅ 必 |
| 业务场景问卷 | 学员自评工作场景语言需求（商务/技术/跨文化） | ✅ 必 |
| 岗位分析 | 客户业务描述→拆解为具体沟通场景 | ⚠️ 高定制时才做 |

### 1.2 设计阶段 · 三轨融合模型

```
A轨（基础层）    B轨（核心层）       C轨（转化层）
平台自学         外教互动           线下/线上工作坊
7000+课件库      1对1母语外教约课   场景模拟/角色扮演/案例
随时做           高频做             定期做
```

**决策规则：**
- A轨：预算＞5万时标配
- B轨：客户明确要求"口语实战"时必配
- C轨：项目周期＞3个月时必配；纯线上项目用直播实战课替代

### 1.3 管理阶段 · 铁三角团队

```
项目总监（统筹协调）
    ↓
学术总监（课程+师资设计）
    ↓
班主任（日常跟盯+数据管理）
```

### 1.4 考核阶段 · 70/30双轨公式

```
总成绩 = 过程考核（70%）+ 结果考核（30%）
  过程 = 出勤 + 约课次数 + 学习时长 + 课堂表现
  结果 = 口试（80%权重）+ 笔试（20%权重）
       口试五维度：听力·发音·词汇·句式·流利度

结业门槛：总分≥70分 + CEFR提升至少一个子级
```

## 二、五大设计方法论

### 2.1 以终定始（Backward Design）
- **原则**：先锁定客户海外业务的核心场景和目标，再倒推课程体系
- **方案表达**："聚焦实战，直击工作场景""有任务、有路径、有考核、有反馈"
- **检查清单**：
  - [ ] 是否明确了客户"海外业务最核心的2-3个沟通场景"？
  - [ ] 课程内容是否直接对应这些场景？
  - [ ] 考核方式是否能够验证这些场景下的能力？

### 2.2 训战结合
- **原则**：课堂上70%时间用于学员练习，而非教师讲授
- **课堂形式**：场景模拟/角色扮演、主题演讲/辩论、案例分析（客户真实项目脱敏）、沙盘推演
- **检查清单**：
  - [ ] 课堂时间中，学员练习比例是否≥70%？
  - [ ] 练习素材是否来自客户的真实业务场景？

### 2.3 混元学习（OMO Blended Learning）
- **原则**：线上与线下形成闭环，而非简单叠加
- **闭环逻辑**：`线上夯基础 → 线下练应用 → 诊断反馈 → 线上补短板`
- **检查清单**：
  - [ ] A/B/C三轨之间是否有明确的承接关系？
  - [ ] 诊断结果是否能反哺学习路径调整？

### 2.4 管理有硬度更有温度
- **原则**：严格管理（考勤/排名/公示）←→ 温度支持（心理疏导/学习方法/激励方案）
- **管理工具**：微信群管理、排名公示、月度/季度优秀学员评选、学习之星表彰

### 2.5 过程+结果双轨评估
- **原则**：甲方既要看到"人在学"（过程），又要看到"进步了"（结果）
- **关键设计**：过程指标占70%权重保证活跃度数据好看，结果考核口试占80%保证实用能力

## 三、方案亮点工具箱

### 3.1 可复用的高端差异化元素

| 元素 | 出自 | 适用场景 | 落地难度 |
|------|------|---------|---------|
| **三年长周期·螺旋上升** | 中核二十五"扬帆·核建" | 预算100万+、长期合作客户 | 高（需持续交付） |
| **4M实训模型** | 中铝长沙院 | 强调"系统化实训"的客户 | 中 |
| **红蓝牌陪跑机制** | 海工（海油工程） | 重视项目过程质量的客户 | 中高（需团队配合） |
| **能力模型（GLOBAL/LAB）** | 建设银行 | 有体系化人才建设需求的客户 | 高（需研发） |
| **一课双得·融合培训** | 中石化 | 时间紧、要求"效率最大化"的客户 | 高（需交叉设计） |
| **Capstone顶点项目** | 中信科移动 | 项目周期≥6个月的客户 | 中 |
| **共同萃取机制** | 伟志集团 | 定制深度要求高的客户 | 中 |
| **VR应急模拟** | 中铝长沙院 | 仪式感/差异化提案需求 | 中高（需技术合作） |

## 四、2025→2026 演变趋势

| 维度 | 2025特征 | 2026特征 | 对设计方案的影响 |
|------|----------|----------|----------------|
| AI赋能 | 概念提及 | 正式落地（AI陪练、AI知识库） | 方案中应加入AI模块截图/数据 |
| 行业深度 | 通用英语+泛化定制 | 茶饮出海、绿地投资等极度细分 | 方案需展示行业理解深度 |
| 交付形态 | 培训方案 | 人才体系建设方案 | 结尾要增加"可沉淀资产"清单 |
| 课程融合 | 英语技能培训 | 管理语言一课双得 | 可设计"1+1>2"的融合课程包 |
| 交付模式 | 线下为主 | 纯线上方案出现 | 要有"纯线上版"模板备选 |
| 跨文化 | 课程模块 | 独立产品线（文化工作坊） | 跨文化可以单独成方案报价 |
| 考核 | 基本双轨 | 更精细化+业务成果关联 | 考核设计要加入"业务产出"维度 |

## 五、常见短板自查清单

提交方案前检查：

- [ ] 是否引用了过往项目的**真实效果数据**？（如"CEFR平均提升1.2级"）
- [ ] AI模块是否有**具体产品截图/落地路径**？
- [ ] 跨文化内容是否基于客户**实际出海国家**定制，而非通用礼仪清单？
- [ ] 方案中对客户行业的**痛点分析**是否真实具体，而非泛泛而谈？
- [ ] 结尾是否列出了**可沉淀的组织资产**（手册/SOP/能力模型等）？
- [ ] 方案是否**跳出了模板感**（师资介绍等通用段落是否做了定制化改写）？
