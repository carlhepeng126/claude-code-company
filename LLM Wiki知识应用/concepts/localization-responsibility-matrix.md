# 🌐 属地化权责矩阵模型

## 定义
麦当劳克洛克对持牌人的核心管理哲学：**经营自主权归你（本地持牌人），管理标准归我（总部）**。总部管QSCV标准、品牌、供应链、培训体系；本地管经营策略、员工招聘、社区关系、本地化产品开发。

## 结构

```
┌──────────────────────────────────────────────┐
│              总部管控（管什么）                  │
│  • QSCV操作标准                               │
│  • 品牌规范与营销策略                           │
│  • 供应链管理体系                              │
│  • 培训体系（汉堡大学）                         │
│  • A级检查评分系统                             │
└──────────────────────────────────────────────┘
          ↑ 标准化 ↓ 本地化
┌──────────────────────────────────────────────┐
│              本地自主（经营什么）                │
│  • 经营策略与定价决策                           │
│  • 本地员工招聘与管理                           │
│  • 社区关系与政府关系                           │
│  • 本地化产品创新                               │
│  • 日常运营执行                                │
└──────────────────────────────────────────────┘
```

## 核心原则
1. **标准不可妥协** — QSCV是全球统一的，不因市场不同而打折
2. **自主必须尊重** — 总部不干预持牌人的日常经营决策
3. **创新从本地来** — 麦香鱼/巨无霸等爆款都来自本地一线
4. **70:30比例** — 大部分持牌店由本地人经营，确保市场驱动力

## 应用场景
- **央企海外分支机构管理体系设计** — 总部管什么 vs 海外团队管什么
- **属地化人才培养方案** — 培训重心在哪（标准输出 vs 本地能力建设）
- **海外授权经营模式** — 合资/独资/合资+特许经营的不同权责分配

## 对Carl的价值
给央企客户做属地化管理培训时，直接用"经营在你，管理在我"八个字概括——比讲"总部管标准、本地管运营"说一百句都有效。这是最精炼的出海管理话术。
