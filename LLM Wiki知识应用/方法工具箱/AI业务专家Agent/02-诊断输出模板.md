# AI诊断输出模板

> AI诊断客户需求时必须按此格式输出。每次测试后将AI输出与标准答案对比，检查一致性。

## 诊断卡

```
## 客户诊断卡 — [案例编号]

### 基本信息
- 驱动类型：[业务倒逼 / 战略扩张 / 战略储备 / 能力退化 / 预算框死]
- 采购模式：[直接采购 / 招投标 / 试点→推广]
- 客户画像：[行业] × [出海阶段] × [人员规模] × [紧迫度：高/中/低]

### 需求判断
- 客户说的：[1-2句话概括客户原始表述]
- 实际需要的：[透过表面需求看到的核心问题]
- 需求分类：[能力缺口型 / 业务倒逼型 / 政策驱动型]

### 短板定位
- 四象限位置：[语言短板 / 专业短板 / 综合短板 / 双弱]
- 学员水平分化：[均匀 / 轻度分化 / 严重分化]
- 核心短板描述：[一句话]

### 方案建议
- 方案类型：[单次培训 / 系列课程 / 分层定制 / 体系项目 / 术语库/微课]
- 交付形态：[纯线下 / 纯线上 / 线上+线下混合]
- 建议周期：[ ]
- 建议规模：[ ]

### 师资配置
- 师资模式：[单人主讲 / 四角色拼盘 / 线上外教+线下专家 / 中教+外教]
- 核心角色：[ ]

### 报价
- 报价区间：[ ]
- 定价依据：[ ]

### 风险与动作
- 风险提示：[ ]
- 下一步动作：[ ]
```

## 评分标准

测试后将AI诊断与标准答案逐项对比：

| 评分项 | 权重 | 评分方法 |
|------|:------:|------|
| 驱动类型一致 | 20% | 5种类型精确匹配 |
| 方案类型一致 | 30% | 大类一致即得分（如"系列课程"vs"分层定制"算不一致） |
| 短板象限正确 | 20% | 四象限位置一致 |
| 师资模式合理 | 10% | 不要求完全一致，但不能方向相反 |
| 报价区间合理 | 10% | 在标准答案±30%范围内 |
| 无方向性错误 | 10% | 没有"该推持续服务的推了单次培训"这类根本性误判 |

**通过标准**：总分≥80分，且无方向性错误。
