# 中海油服：需求诊断萃取问答

> 本文为"你问我答"格式，用于从中海油服项目中提取判断逻辑、方案选型、师资配置与报价区间。

## 1. 客户需求与真实问题

1. 客户最初说的需求/问题是什么？

   人力资源联系我们，询问我们能否做钻井方面的专业英语培训。

2. 你当时判断的"真实问题"是什么？

   客户在乌干达有项目，但甲方业主对中海油服人员的英语不满意、不通过，影响了项目推进。这是典型的**业务倒逼型**需求——不是"我们想提升英语"，而是"甲方不认可、项目推不动"。真实问题不是英语能力，而是**甲方信任和项目准入**。

## 2. 方案类型判断

3. 你当时认为这更适合：
   - 单次培训 → 持续服务
   - 为什么？

   第一年从单次培训切入。客户关注的核心是"你们有没有石油钻井方面的老师"，我们提供了专业钻井老师视频，客户满意后直接约见。

   采购第一年后，客户连续三年采购了我们的产品——从单次培训自然演化到持续服务。这个演化的驱动力是**交付质量建立信任**，不是销售推动。

   客户后续自己建设了覆盖300人的内训体系，我们在这个过程中持续提供培训服务。

## 3. 短板定位

4. 你判断出的主要短板属于哪一象限？
   - 语言 / 专业 / 管理 / 文化

   核心短板是**专业 + 语言**的复合短板。学员英语水平不一：市场人员英语较好，但技术人员存在明显短板。培训方案设置为一个月的英语集训项目，课程涵盖商务英语和专业英语内容，不是通用口语培训。

## 4. 师资配置

5. 你给出的师资配置核心是什么？
   - 核心角色和人数

   师资是成交的第一关。客户的决定逻辑是"你有没有对的人"——不是先看方法论，是先看师资匹配度。我们提供了专业的钻井方面老师视频，匹配成功后才进入方案环节。

   核心配置：石油钻井行业英语专家主讲（如Ben Phone级别）。钻井场景不需要宏观嘉宾和区域专家拼盘，需要的是**懂钻井、能教英语**的复合型讲师。

## 5. 报价区间

6. 你给出的报价区间是什么？

   按客户预算约2000元/小时左右定价。

## 6. 额外备注

7. 这个项目的关键判断点还有哪些？

   - **师资匹配是第一关**：客户心里已经有需求，在找能落地的人。先证明"我有对的人"，再谈方案。
   - **驱动类型决定紧迫度**：业务倒逼型（甲方不认可→项目受阻）的紧迫度和决策速度远高于战略规划型。
   - **交付质量是复购引擎**：第一年做好了，客户连续三年采购——不是靠销售推动的。
   - **客户自己会建体系**：300人和内训体系是客户自己建设的，我们不需要在第一天就推"体系搭建"，做好培训交付，客户自然会扩展。

8. 你希望 AI 在这个案例里特别避免的错误是什么？

   - 不要把"甲方业主不满意英语"误判为"需要通用英语培训"。这是专业场景+业务倒逼，方案必须紧扣客户的作业场景（钻井指令、安全沟通、甲方汇报）。
   - 不要把师资当成"方案的一个配置项"——在这个案例里，师资匹配是**成交的先决条件**，先于方案设计。
   - 不要一上来就推"体系搭建"或"长期合作"。客户是被倒逼的，先解决他的燃眉之急，交付好，复购自然发生。
   - 不要忽略客户内部英语水平的分化——市场人员和技术人员的短板不同，方案需要有针对性。
