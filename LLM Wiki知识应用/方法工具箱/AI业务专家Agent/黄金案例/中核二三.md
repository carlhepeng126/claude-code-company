# 黄金案例：中核二三国际化工程英语人才培训

## 一、客户说了什么

| 维度 | 信息 |
|------|------|
| 客户 | 中核二三建设有限公司（中核集团子公司） |
| 对接部门 | HR部门 + 国际部门联合发起 |
| 背景 | 海外业务从2012年开始，覆盖格鲁吉亚/马来西亚/越南/伊拉克/纳米比亚，海外221人（伊拉克120人，50中方+60外方） |
| 需求表述 | "解决实际的涉外沟通交流挑战" |
| 核心矛盾 | 人与英语能力不匹配，业主也在反映语言问题 |
| 学员 | 多层次人员：有基础的可3个月外派，基础偏弱的需6-8个月 |
| 约束条件 | 基础人员集中培训不现实，工作会受影响 |

## 二、客户画像速写

- **行业**：核电工程建设
- **出海阶段**：已在海外深耕10+年，持续扩张中
- **驱动类型**：业务倒逼型（业主反映语言问题 + 人与英语能力不匹配）
- **紧迫度**：高（当下50人缺口，后续100人需求）
- **人员规模**：221人海外，多层次（管理/操作/基础）
- **关键约束**：不能大规模脱产——工作会受影响

## 三、诊断过程

### 第一轮沟通，客户原话是什么？
HR部门与国际部门联合找上来，要"解决实际的涉外沟通交流挑战"。不是泛泛的"提升英语水平"，而是指向了具体的业务问题。

### 追问了什么？
- 不同层级的人英语基础差异多大？（发现有基础的3个月可外派，基础弱的需6-8个月）
- 人员能不能集中脱产？（不能——基础人员集中培训不现实，工作会受影响）
- 长期规划是什么？（当下50人，后续100人，需要持续培养机制）

### 发现了什么客户没意识到的问题？
- 客户最初可能想"一个班解决"，但实际上学员水平分化严重，从"有基础能开口"到"水平很低"都有，必须分层
- "工作会受影响"这个约束意味着传统脱产模式行不通，需要混合式
- 客户不仅需要培训，还需要**在培训过程中建立体系**——这不是一次性的项目

### 核心判断
高度的定制——培训必须是分层的，设计更加个性化的解决方案。线上外教约课提基础，线下实战专家脱产授课促转化。多层次人员的培训过程中同步建立体系。

## 四、方案决策理由

| 决策点 | 你选了什么 | 为什么没选替代方案 |
|--------|-----------|-------------------|
| 方案类型 | 分层定制 + 体系化 | 不能一刀切——学员水平从"很低"到"有基础"都有；不能纯脱产——工作会受影响 |
| 培训模式 | 线上外教约课 + 线下实战专家脱产 | 线上解决基础提升（不脱产、不影响工作），线下解决能力转化（集中突破） |
| 师资配置 | 外教（线上）+ 实战专家（线下）拼盘 | 不同层级需要不同类型的老师——外教拉基础，实战专家做场景转化 |
| 分层逻辑 | 有基础→3个月外派；基础弱→6-8个月 | 不按统一周期，按实际水平分路径 |
| 体系化 | 培训过程中同步建立体系 | 客户有长期需求（50→100人），不能每次从头来 |

## 五、最终交付方案

- **类型**：分层定制培训 + 体系化建设
- **模式**：线上外教约课（基础提升）+ 线下实战专家脱产（能力转化）
- **师资**：外教 + 实战专家，拼盘式配置
- **分层**：
  - 有基础人员：3个月集训 → 外派
  - 基础偏弱人员：6-8个月 → 外派
- **报价**：约2000元/小时（与中海油服类似）
- **长期**：培训过程中同步搭建培养体系，覆盖当下50人+后续100人

## 七、可复用的模式

> 这个案例中的哪些判断逻辑可以迁移到其他央企工程类客户？

1. **分层是刚需，不是加分项**：当学员水平从"很低"到"有基础"都有时，不分层等于浪费所有人的时间
2. **脱产约束是央企工程类客户的普遍特征**：一线人员在项目上，抽不出来。混合式（线上+线下）是务实方案
3. **业务倒逼型客户需要"边培训边建体系"**：因为他们有持续的人才缺口（50→100→更多），不能每次从头采购
4. **HR+业务部门联合发起**是信号：意味着需求不是"完成培训指标"，而是真的有业务痛点
