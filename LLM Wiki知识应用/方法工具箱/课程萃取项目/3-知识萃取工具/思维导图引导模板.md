# 思维导图引导模板

> 用思维导图辅助专家梳理知识结构，适合在访谈中或工作坊中使用。

## 引导话术

### 开场
> "接下来我们一起画一张图，把您在这个领域的知识地图画出来。不用写太多字，关键词就行。"

### 核心引导
- "这个领域的核心是什么？用一个词放在中间。"
- "从核心出发，可以分成哪几个大的板块？"
- "每个板块里，最关键的知识点是什么？"
- "这些板块之间有什么关系？"

---

## 推荐导图结构

### 结构一：中心辐射型（适合知识梳理）

```
             ┌── 子知识点1
             │
    ┌── 板块A ── 子知识点2
    │          │
    │          └── 子知识点3
    │
    │          ┌── 子知识点1
──● 核心主题 ── 板块B ── 子知识点2
    │          │
    │          └── 子知识点3
    │
    │          ┌── 子知识点1
    └── 板块C ── 子知识点2
               │
               └── 子知识点3
```

**适用场景**：梳理某个领域的知识体系

### 结构二：流程型（适合流程/步骤梳理）

```
┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐
│步骤1│→│步骤2│→│步骤3│→│步骤4│→│步骤5│
└──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘
   │       │       │       │       │
   ▼       ▼       ▼       ▼       ▼
 关键点   关键点   关键点   关键点   关键点
  输入     判断     决策     输出     检查
```

**适用场景**：梳理操作流程或业务流程

### 结构三：对比型（适合方法论/方案选择）

```
                    ┌── 优点
          ┌── 方案A ── 缺点
          │          └── 适用场景
          │
核心问题 ── 方案B ── 类似结构
          │
          └── 如何选择
               │
               └── 判断条件1 → 条件2 → 条件3
```

**适用场景**：比较不同方法/方案

### 结构四：问题解决型（适合经验案例）

```
          ┌── 背景
          │
          ├── 挑战 ── 难点1 / 难点2 / 难点3
          │
核心案例 ── 方案 ── 步骤1 / 步骤2 / 步骤3
          │
          ├── 结果 ── 量化数据
          │
          └── 经验 ── 可复制方法 / 避坑 / 原则
```

**适用场景**：深入复盘一个具体案例

## 追问话术

在专家画图过程中，适时追问以下问题：

### 知识结构类
- "这个板块还能再往下分吗？"
- "这两个知识点之间是什么关系？是并列还是递进？"
- "如果新人要先学什么后学什么，顺序怎么排？"

### 流程步骤类
- "第 3 步到第 4 步之间，有没有什么关键的判断？"
- "每一步做完，怎么确认做到位了？有什么检查标准？"
- "如果某一步出错了，通常是因为什么？怎么补救？"

### 经验判断类
- "什么情况下选择走这条路？什么情况下走另一条？"
- "判断的依据是什么？有什么可观察的信号？"
- "这个经验有没有什么前提条件？换了环境还适用吗？"
