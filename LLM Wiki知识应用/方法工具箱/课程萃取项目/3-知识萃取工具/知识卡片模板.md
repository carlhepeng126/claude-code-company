# 知识卡片模板

> 知识卡片是萃取的最终输出单元——每个卡片封装一个独立、完整、可传递的知识点。

---

## 知识卡片质量检查清单

- [ ] **独立性**：一张卡片只讲一个知识点
- [ ] **可操作性**：看完知道怎么做（有具体步骤或判断标准）
- [ ] **案例支撑**：至少有一个实际案例
- [ ] **边界明确**：清楚什么情况下适用、什么情况下不适用
- [ ] **专家确认**：内容已请专家审核
- [ ] **可记忆**：核心要点不超过 5 条，最好有口诀或模型
