# 经验复盘工作坊设计

> 当知识需要通过多人协作萃取时（如一个团队的经验汇总），采用工作坊形式会比一对一访谈更高效。

## 标准流程设计（半天版）

### 破冰：20分钟
**活动**：经验拍卖会 — 每人写3个经验关键，小组"拍卖"最有价值的

### 第一环节：经验 Map：30分钟
**活动**：业务流程图上标注 🟢（做得好）/ 🟡（关键但难教）/ 🔴（新人易错），投票选3个深挖点

### 第二环节：深度复盘：60分钟
**活动**：世界咖啡屋 — 第一轮成功案例复盘，第二轮踩坑经验复盘

### 第三环节：模型共创：40分钟
**活动**：各组将经验整合为结构化框架，命名并配口诀，海报展示

### 收尾：10分钟
**产出确认**：拍照存档、记录员整理、确定补充计划

---

## 引导技巧卡片

| 情境 | 引导话术 |
|------|---------|
| 参与者沉默 | "如果给刚入行时的自己一句建议，您会说什么？" |
| 回答太笼统 | "能举一个您亲身经历的例子吗？当时发生了什么？" |
| 有争议观点 | "很有意思的视角，其他人在这方面有什么不同的经验吗？" |
| 信息杂乱 | "我们来归归类，这些可以分成哪几类？" |
| 缺少步骤 | "这个过程的先后顺序是怎样的？第一步做什么？" |

---

## 产出物要求

工作坊结束后 48 小时内需产出：
1. **核心成果摘要**（1页）
2. **素材详录**
3. **可视化海报**电子版
4. **行动清单**
