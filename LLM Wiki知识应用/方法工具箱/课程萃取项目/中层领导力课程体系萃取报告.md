## 一、核心模型：中层领导力三维发展体系

中层管理者领导力发展的核心逻辑是 **"由知到行，由行到体系"**，底层支撑是三大维度：

```
                 ┌──────────────────────────┐
                 │     管理能力              │
                 │  (带人·管事·建体系)        │
                 └──────────┬───────────────┘
                            │
      ┌─────────────────────┼─────────────────────┐
      │                     │                     │
      ▼                     ▼                     ▼
┌──────────┐         ┌──────────┐          ┌──────────┐
│ 综合素养 │◄────────┤ 中层干部 ├────────►│ 经营能力 │
│ (思维·   │         │ 领导力   │          │ (财务·   │
│  沟通·   │         │ 发展模型 │          │  战略·   │
│  文化)   │         └──────────┘          │  业务)   │
└──────────┘                               └──────────┘
                            │
                     ┌──────┴──────┐
                     │             │
                     ▼             ▼
              ┌──────────┐  ┌──────────┐
              │ 国际化    │  │ 实战落地  │
              │ 视野      │  │ 复盘闭环  │
              └──────────┘  └──────────┘
```

### 核心主张

> **中层干部培训的关键不是「教什么」，而是「体系怎么长」——让领导力伴随业务扩张同步进化。**

- **管理能力** — 带团队、定目标、追过程、拿结果
- **综合素养** — 沟通表达、跨文化协作、思维升级
- **经营能力** — 财务思维、成本意识、业务洞察
- **国际化视野** — 出海背景下的跨文化领导力（摩根世纪核心差异化）
- **实战落地** — 培训→实践→复盘闭环，而非一次性课程

│  └──────────────────────────────────────────────────────────────┘  │
│                              ▼                                      │
│  阶段二：攻坚（第5-8月）                                            │
│  ┌──────────────────────────────────────────────────────────────┐  │
│  │ 绩效突破 · 协同增效 · 业务应用                                 │  │

│  └──────────────────────────────────────────────────────────────┘  │

│  └──────────────────────────────────────────────────────────────┘  │

### 三阶段能力递进矩阵

| 阶段 | 管理能力 | 综合素养 | 经营能力 | 国际化 | 关键产出 |
|------|---------|---------|---------|--------|---------|
| **破局** | 角色认知、目标计划、非人、情景领导 | 高效沟通、团队协同 | — | 国际化视野导入 | 开题报告、个人转型计划 |
| **攻坚** | 绩效辅导、执行力、复盘初阶 | 冲突化解、跨部门协作 | 成本意识、报表解读 | 涉外谈判导入 | 中期成果、季度业绩回顾 |
| **收官** | 复盘萃取、领导力进阶 | 跨文化沟通、表达呈现 | 经营沙盘模拟 | 跨文化实战应用 | 结题成果、年度述职 |

## 三、四层领导力定位（D1-D4）

中层干部（D2层）不是孤立的，它处于企业领导力发展体系中的关键位置：

| 层级 | 对象 | 核心目标 | 周期 | 关键差异点 |
|------|------|---------|------|-----------|
| **D1** | 基层主管 | 从个人贡献者→管理者转型 | ~半年 | 基础管理技能、任务分配、反馈技巧 |
| **D2** | 🌟 **中层经理** | **跨部门协作与管理体系搭建** | **~1年** | **目标分解、团队建设、跨部门协同、国际化视野** |
| **D3** | 高层总监 | 战略思维与跨文化领导力 | ~1.5年 | 战略制定、变革管理、组织设计 |
| **D4** | 决策层 | 全球化视野与组织设计 | ~2年 | 全球运营、资本运作、文化融合 |

> 中层（D2）的核心定位是：**上接战略、下接执行、左右协同、内外兼修**。

## 四、中层干部核心能力模型

基于典型中国出海企业的中层管理场景，中层干部需具备以下 **8项核心能力**：

### MANAGER 素质模型

```
M ─ Mission-driven  使命驱动 —— 理解公司战略，层层分解为目标
A ─ Align & Execute  对齐执行 —— 跨部门资源整合，推动落地
N ─ Nurture talents  培育人才 —— 选育用留，搭建人才梯队
A ─ Agile adapt      敏捷适应 —— 面对不确定性快速调整
G ─ Global mindset   全球视野 —— 跨文化理解与国际化经营
E ─ Empower others   赋能团队 —— 授权、激励、辅导
R ─ Reflect & learn  反思迭代 —— 复盘萃取，持续改进
```

### 八项能力对应的课程映射

| 能力 | 核心课程 | 教学方式 | 评估方式 |
|------|---------|---------|---------|
| M-使命驱动 | MTP综合管理、战略目标分解 | 讲授+演练 | 部门目标分解表 |
| A-对齐执行 | 执行力提升、跨部门协作 | 案例+工作坊 | 跨部门协作改进案例 |
| N-培育人才 | 非人力资源的人力资源管理 | 讲授+演练 | IDP辅导记录 |
| A-敏捷适应 | 创新思维与问题解决 | 案例研讨 | 问题分析报告 |
| G-全球视野 | 国际化经营、跨文化谈判 | 讲授+模拟 | 国际化课题 |
| E-赋能团队 | 情景领导力、绩效辅导 | 测评+演练 | 团队管理改进计划 |
| R-反思迭代 | 复盘方法论、经验萃取 | 工作坊 | 复盘报告 |

## 六、教学设计方法论

### 6.1 教学四步法

```
预习探疑 ──→ 互动答疑 ──→ 行动实践 ──→ 总结复盘
    │            │            │            │
    ▼            ▼            ▼            ▼
  提前阅读    案例/情景/    课后任务     ORID复盘
  需求调研    角色扮演      上级确认    分享会
  痛点收集    讲师点评      应用落地    案例沉淀
```

### 6.2 线上线下融合机制

```
┌───────────────────────────────────────────────────────┐
│                    12个月成长闭环                        │
│                                                         │
│  线下                            线上                    │
│  ┌────────┐              ┌─────────────────────────┐   │
│  │ 集中    │              │ 每月主题工作坊（1.5h）    │   │
│  │ 培训    │────交互────→│ 月度工具推送              │   │
│  │        │              │ 名师诊断室（在线答疑）     │   │
│  └────────┘              │ 社群运营与积分管理        │   │
│       │                  │ 实践案例跟踪与汇编         │   │
│       │                  └─────────────────────────┘   │
│       ▼                           │                    │
│  ┌────────┐                       │                    │
│  │ 复盘    │◄────反馈────────────┘                    │
│  │ 实践    │                                          │
│  └────────┘                                          │
└───────────────────────────────────────────────────────┘
```

## 七、项目交付框架（12个月标准版）

### 7.1 标准交付流程

| 阶段 | 时间 | 核心任务 | 主要交付物 |
|------|------|---------|-----------|
| 项目诊断 | 第1-2周 | 需求调研、学员画像、干系人访谈 | 需求报告、画像分析 |
| 方案设计 | 第3-4周 | 课程定制、讲师匹配、时间确认 | 定制方案、课程大纲 |
| 阶段一：破局 | 第2-5月 | 7天集训 + 线上辅导 + 开题 | 培训实施、开题报告 |
| 阶段二：攻坚 | 第6-9月 | 4天课程+3天复盘 + 线上 | 中期汇报、季度回顾 |
| 阶段三：收官 | 第10-12月 | 4天课程+3天复盘 + 结业 | 结题、述职、案例集 |
| 评估总结 | 结项后1月 | 四级评估、结项报告 | 评估报告、归档 |

### 7.2 关键成功要素

1. **✅ 双班主任制** — 客户方+摩根世纪各一名班主任，分工明确
2. **✅ 课题绑定业务** — 学员课题必须源于真实业务痛点
3. **✅ 上级卷入** — 课后实践任务需直线上级签字确认
4. **✅ 评估挂钩绩效** — 培训评估占综合评分的40%
5. **✅ 案例沉淀** — 优秀案例汇编为《管理实战案例集》

## 十、面向不同行业的定制指南

| 行业 | 需调整内容 | 需保留的通用模块 |
|------|-----------|----------------|
| **新能源/材料**（如当升科技） | 国际化经营→出海合规；财务→供应链金融 | 管理能力、综合素养全套 |

| **制造/工业** | 创新思维→精益管理；财务→成本控制 | 角色转型、非人、复盘全套 |
