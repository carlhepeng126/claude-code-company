# 课程萃取项目

## 项目概述

> 面向客户交付的课程开发与知识萃取项目。本框架旨在系统化地将专家经验转化为结构化、可传承的课程内容，适用于企业内训师培养、岗位经验沉淀、专家知识传承等客户场景。

## 核心流程

课程萃取采用 **五步法**：

1. **需求分析** → 明确客户期望、学员画像、业务场景
2. **知识采集** → 运用访谈、观察、复盘等方式获取原始素材
3. **知识萃取** → 结构化整理，提炼核心模型与框架
4. **课程设计** → 围绕萃取成果进行教学设计
5. **交付迭代** → 课程实施并持续优化

## 使用指南

- 按编号顺序推进各阶段工作
- 每阶段产出物需经客户确认后再进入下一阶段
- `3-知识萃取工具` 是本项目的核心工具集，可灵活组合使用
- 最终交付物统一放在 `交付成果` 目录
