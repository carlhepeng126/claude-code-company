# 交付物检查清单

> 项目各阶段的交付物质量检查标准。

## 需求分析阶段

**需求调研报告**：客户背景清晰 / 痛点有数据支撑 / 目标行为化表述 / 客户确认

**学员画像**：数量与分布 / 经验分层 / 学习动机 / 知识缺口

## 知识萃取阶段

**访谈记录**：录音完整 / 知识点标注清晰 / 24h内整理 / 专家确认

**知识卡片**：独立 / 可操作 / 有案例 / 边界清晰 / 专家确认

**核心模型**：可视化 / 一页纸讲清楚 / 准确性 / 可记忆性

## 课程设计阶段

**课程大纲**：目标对齐 / 结构完整 / 逻辑递进 / 时间合理 / 客户确认

**教学设计**：教学方法多样化 / 学员参与度＞50% / 练习匹配目标

## 课程开发阶段

**讲师手册**：逐页讲解指引 / 话术示例 / 时间提示 / 突发应对

**课件PPT**：每页≤1核心观点 / 图表呈现 / 讲师备注 / 风格统一 / 无错字

## 交付阶段

**试讲反馈**：覆盖学员/专家/客户三方 / 问题分类 / 改进闭环

**正式交付**：交付物齐全 / 格式规范 / 打包清晰 / 客户签收
