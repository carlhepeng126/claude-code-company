# 课程萃取全流程模板

> 项目级操作手册，涵盖从启动到交付的完整流程。

---

## 流程总览

```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│  启动    │→│  需求    │→│  采集    │→│  萃取    │→│  设计    │→│  交付    │
│  对接    │ │  调研    │ │  知识    │ │  提炼    │ │  开发    │ │  迭代    │
│ Day1-3   │ │ Day4-7   │ │ Day8-14  │ │ Day15-21 │ │ Day22-35 │ │ Day36+   │
└──────────┘ └──────────┘ └──────────┘ └──────────┘ └──────────┘ └──────────┘
```

## 各阶段交付物

| 阶段 | 关键活动 | 输出物 |
|------|---------|--------|
| 启动对接 | 项目启动会、确认范围与专家名单 | 项目立项书、项目计划 |
| 需求调研 | 高层访谈、学员问卷、场景观察 | 需求报告、学员画像、场景分析 |
| 知识采集 | 专家访谈2-3轮、现场观摩、案例采集 | 访谈记录、案例素材、原始素材库 |
| 知识萃取 | 素材编码、模型构建、知识卡片、专家审核 | 卡片集、核心模型、案例集 |
| 设计与开发 | 大纲设计、课件开发、手册编写 | 讲师手册、学员手册、课件PPT |
| 交付迭代 | 试讲、正式交付、四级评估 | 试讲报告、全部交付物、评估报告 |

## 常见问题处理

| 问题 | 应对方案 |
|------|---------|
| 专家时间难以协调 | 提前锁定3个可选时间段；每次访谈≤90分钟 |
| 专家"会做不会讲" | 用隐性知识显性化清单引导 |
| 客户需求中途变化 | 书面变更管理流程 |
| 知识量过大 | 按重要紧急排序，优先"高价值易传递" |
