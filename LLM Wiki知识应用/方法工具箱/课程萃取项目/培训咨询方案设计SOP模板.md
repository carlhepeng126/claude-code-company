## 方案提交前检查清单

### 启动前
- [ ] 客户有没有成文的战略/领导力框架？引用了吗？
- [ ] 客户在比价吗？方案里有拍板理由吗？
- [ ] 目标人群管理层级确认了吗？
- [ ] 客户已有的培训/咨询经历了解了吗？

### 设计中期
- [ ] 胜任标准在课程设计之前就定义了吗？
- [ ] 每个模块都能回答"练什么、产出什么、谁检验"吗？
- [ ] 分班策略和报价一致吗？
- [ ] 有没有不适合目标人群层级的模块？
- [ ] 所有天数加总一致吗？
- [ ] 有没有"待定"的讲师或内容？

### 交付前
- [ ] 有"秀"的表达吗？（删）
- [ ] 客户的语言出现在方案中了吗？
- [ ] HR拿着方案能直接向领导汇报吗？
- [ ] 报价逻辑清晰吗？
