# 小红书内容生产流水线

一站式串联：选题(去重)→热度预检→情绪分析→标题设计→文案(去AI味)→违禁词扫描→标签策略→质检→卡片→图片→输出→发布运营。触发后全流程自动跑，质检不通过自动打回重写。

## 素材源优先级

按顺序扫描，同源内优先取日期最近的文件：

1. `ACT笔记-深度思考/` — 项目方案、客户纪要、行动卡、B卡
2. `收件箱：同步助手/` — 每日行业动态、操作日志
3. `收件箱：需求模拟/` — 每日客户分析报告
4. `LLM Wiki知识应用/` — 知识实体、概念、报告

**日常模式**：用户未指定素材时，扫近期文件出3个选题让用户选。**指定模式**：用户给出了主题或文件路径，直接进流水线。

## 流水线

### STEP 0 — 素材→选题提取 + 去重检查

从素材中抓取爆点角度：
- 最反常识的数据点
- 最 relatable 的人物处境
- 最出人意料的结果

日常模式输出 3 个选题（含标题方向+素材来源），等待用户选择。

#### 选题去重规则

输出选题前，扫描 `收件箱：同步助手/小红书生产/` 下所有已发布帖子，按以下 6 类归类：

| 类型 | 特征 | 示例 |
|------|------|------|
| 框架/工具 | 提供可复用的方法论、决策框架、清单 | 六维自建vs外包决策框架 |
| 痛点/场景 | 聚焦一个具体职场处境，制造共鸣 | "在场但不在局"悖论 |
| 数据/趋势 | 以行业数据或趋势洞察为核心 | 央企出海东南亚趋势 |
| 案例复盘 | 脱敏后的客户项目经验 | 申菱4000人英语盘点 |
| 观点/反常识 | 挑战常规认知，引发讨论 | "AI时代领导力过时了吗" |
| 清单/模板 | 可直接保存使用的资源 | 培训体系搭建清单 |

**规则：近 4 天内不得发布相同类型的帖子。** 当前选题类型在 4 天内已出现过，自动换类型推荐。同一类型至少间隔 4 天。

### STEP 1 — 选题热度预检

用 `web-access` 搜小红书同类话题近期高赞帖，判断：
- 竞争热度（是否已卷烂）
- 差异化角度建议
- 如选题偏冷，给切入建议

### STEP 2 — 情绪触发点分析

调用 `marketing-psychology`，分析目标受众情绪杠杆，输出3个情绪钩子按冲击力排序。

### STEP 3 — 文案生产（标题→正文→去AI味→违禁词→标签）

调用 `copywriting` + `social`，按以下顺序逐项产出。

#### 3A. 标题设计（双标题体系 — 优先独立打磨）

标题决定点击率（质检权重 50%），必须在正文之前独立设计，不跟随正文随手写。

**一篇笔记有两个标题，各自承担不同功能：**

| 标题类型 | 写给谁 | 核心作用 | 设计逻辑 |
|----------|--------|---------|---------|
| 封面标题 | 用户 | 吸引点击 | 热点/痛点/爽点钩子 |
| 正文标题 | 系统 | 识别+搜索 | 主题关键词+长尾关键词 |

##### 封面标题（点击钩子 → 放在首图）

在信息流中用户第一眼看到封面标题，必须是够强的钩子。三大钩子按场景选用：

| 钩子类型 | 原理 | 本领域示例 |
|----------|------|-----------|
| 热点 | 借势当前关注焦点 | "出海企业英语，2026年变了" |
| 痛点 | 喊出读者真实处境 | "外贸团队英语差，丢了多少单" |
| 爽点 | 给出可触及的好结果 | "3个月从不敢开口到全英提案" |

**封面标题设计原则：**
- 喊出目标读者的真实处境，不是描述文章内容
- 数字锚点优先（"8大伤号操作" > "伤号操作汇总"）
- 对比/反差制造冲击（"在场但不在局"）
- 避免标题党，承诺必须能在正文中兑现

##### 正文标题（关键词引擎 → 放在小红书发布标题栏）

正文标题两个作用：帮系统识别内容主题精准推送 + 抢占搜索流量获得长期曝光。

**规则：**
- 必须嵌入核心关键词和长尾关键词，不能模糊
- 错误写法："我们公司最近做了个培训项目"——系统完全不知道推给谁
- 正确写法：把主题关键词直接写进标题，让算法秒懂

**本领域核心关键词库（正文标题优先嵌入）：**

| 层级 | 关键词 |
|------|--------|
| 大词 | 企业英语培训、商务英语、行业英语、跨文化沟通、国际化人才培养 |
| 场景词 | 海外参展英语、商务谈判英语、英文邮件、跨国会议、外派英语 |
| 人群词 | 外贸团队、出海企业HR、海外项目经理、国际化团队 |

**输出格式：** 每次提供 3 个标题方案（每个方案含封面标题+正文标题配对），标注主推，等待确认后进入正文。

**正文标题强制验证：** 定稿后必须列出嵌入的关键词清单，格式：`嵌入关键词（N个）：企业英语培训、商务英语、行业场景英语…`。N<3 则打回重写——正文标题的核心作用就是关键词引擎，关键词密度不够等于白写。

#### 3B. 内容规则
- 情绪钩子开头，不要直接给数据
- 客户原话制造共鸣
- 人设一句带过
- 三步解法各≤2行
- 效果闭环必须
- 金句收尾
- 最后一张必是清单卡
- 脱敏：不出现客户真实名称、行业细节、上市代码
- 一篇只讲一件事，用四层结构讲透（观察→对比→方向→承诺）
- 每篇带一个可抄走的工具（模板/清单/自测表/检查表）

#### 3C. 去AI味规则

朱雀等检测抓两个点：**句长太均匀** + **结构太机械**。分两阶段执行——先注入再自查，不混在一起。

##### 阶段一：写作时注入（边写边用，不是事后修补）

**句长节奏 → 最关键的一条**

极短句(2-4字)与正常句(15-25字)交替。节奏参考：
> "下雨了。大的。哗啦啦直往下泼。我站在公交站台，望着漆黑的马路，心里直犯嘀咕。这都快十一点了，末班车到底来不来？"

三个硬要求：每段至少2-3个极短句 / 连续3句不差不多长 / 段落长短不一，一句话成段可以

**结构跳跃**

不用过渡词直接换话题。自然插入回忆（"记得去年有个客户…"）。不写"首先其次最后""另外""顺便说"。段落转换不做铺垫

**段落温差**

不同段风格有波动。前段严肃后段松弛，叙事段口语、说理段短句。让"写累了"的感觉自然透出。防的是：整篇像同一个人用同样情绪一口气写的

**四维注入**

| 维度 | 写法 |
|------|------|
| 思维 | 犹豫、转折、突然想到、自问自答、分享小故事 |
| 表达 | 口语化、不规范句子正常出现、段落长短不一 |
| 情感 | 主观看法、互动提问、该感叹就感叹但不夸张 |
| 视角 | "我"讲经历 / "我们"讲共鸣 / "你"给建议 / 第三人称讨论，自然混用 |

##### 阶段二：写完后自查（逐条举证，不打勾了事）

自查不是打勾。每条后面必须摘一句正文作为证据，没证据 = 没通过。

- [ ] 极短句(2-4字)有没有？摘一句：`"钱花了。证考了。"` 
- [ ] 连续3句长短交替？摘一组：`"四周后回访。12个人里9个说：真敢开口了。"`（短+长+短）
- [ ] 有没有"首先其次最后"或有序序号？
- [ ] 有没有禁止词：还有啊、其实啊、然后、然而、所以啊、随着、总之、总而言之？
- [ ] 有没有括号、破折号说明性标记？
- [ ] 每个段落结尾去掉了句号感叹号问号？
- [ ] 字数在1000以内？
- [ ] 禁止机构名称、产品名称、电话、地址、网址
- [ ] 如果你是朱雀，你会判这是AI吗？——这一条必须诚实回答，不确定就是有风险

核心原则：**过度的完美 = 不自然。让文章不那么完美，但内容质量不打折。**

#### 3D. 违禁词预检清单

文案完成后立即逐条扫描以下三类违禁词，命中则替换或删除：

**极限词（广告法禁止）：**
最、第一、唯一、首个、首选、顶级、极致、独家、全网、国家级、世界级、史上最、永久、万能、100%、百分百、彻底、绝对

**导流词（小红书风控）：**
微信、微信号、QQ、手机号、电话、扫码、加好友、私聊、私信我、点链接、公众号、小程序

**营销敏感词（限流/屏蔽）：**
推荐、强烈推荐、权威推荐、专家推荐、无效退款、治愈率、骗局、欺骗、骗子、退款、下架、差评、别信、价格、多少钱、优惠、折扣、购买、下单、抢购

#### 3E. 标签策略

正文定稿后，按三层结构设计 5-6 个标签：

| 层级 | 作用 | 示例 |
|------|------|------|
| 大品类词 | 兜底覆盖，进入大流量池 | #中国企业出海 #品牌出海 #培训 |
| 场景词 | 精准搜索，锁定需求意图 | #海外参展 #英语培训方案 #跨文化沟通 |
| 受众词 | 框定客户，吸引目标人群 | #B2B出海 #工厂出海 #HR培训经理 |

**规则：**
- 总数 5-6 个，不多于 8 个
- 三层至少各有 1 个
- 不用与正文无关的热门标签蹭流量
- 标签写入正文末尾，格式 `#标签1 #标签2`

### STEP 3.5 — 违禁词扫描 ⚠️

文案初稿完成后，独立执行违禁词扫描。逐条检索三类违禁词，命中则**必须替换或删除**后方可进入 STEP 4。

#### 扫描流程

1. 逐行扫描文案全文（含标题、正文、标签）
2. 对三类违禁词逐一检索匹配
3. 命中 > 记录位置 + 给出替换建议
4. 批量替换后复扫确认清零
5. 输出扫描报告（扫描词数 / 命中数 / 处理方式）

#### 违禁词库

**A类 — 极限词（广告法禁止，触发违规通知）：**
```
最 第一 唯一 首个 首选 顶级 极致 独家 全网
国家级 世界级 史上最 永久 万能 100% 百分百
彻底 绝对 领先 冠军 巅峰 无敌
```

**B类 — 导流词（小红书风控，触发限流/禁言）：**
```
微信 微信号 QQ 手机号 电话 扫码 加好友 私聊
私信我 点链接 公众号 小程序 二维码 联系我
网址 http https://  @
```

**C类 — 营销敏感词（影响推荐，限流/屏蔽）：**
```
推荐 强烈推荐 权威推荐 专家推荐 无效退款 治愈率
骗局 欺骗 骗子 退款 下架 差评 别信 价格 多少钱
优惠 折扣 购买 下单 抢购 免费领取 点击领取
保证 有效 成功率 疗效 根治 疗程 保证治愈
```

**D类 — 行业敏感词（医疗/金融/教育特别管控）：**
```
治愈 治疗 疗效 诊断 患者 病例 临床 医保
收益 回报率 保本 无风险 理财 投资 利率
通过率 包过 代考 替考
```

#### 处理规则

| 命中类型 | 处理 |
|----------|------|
| 极限词 | 删除或降级（"最有效"→"有效"，"第一"→删除） |
| 导流词 | 删除，用站内自然表达替代 |
| 营销敏感词 | 替换为中性表达（"推荐"→"分享"，"购买"→"了解"） |
| 行业敏感词 | 删除整句，改写为安全表述 |

命中后修改文案，复扫确认清零。扫描报告附在 STEP 7 汇总输出中。

### STEP 4 — 质检关 ⚠️

调用 `marketing-reviewer` Agent 评分。

评分模型：Q1看(50%) + Q2赞(30%) + Q3留(20%)

| 总分 | 动作 |
|------|------|
| ≥ 85 | 放行 |
| 80-84 | 放行，微调建议 |
| 70-79 | 打回 STEP 3 重写，附失分原因 |
| < 70 | 打回 STEP 3 重写，附改写范例 |

打回后最多重写**2次**，仍不通过则保留当前最佳版并标注"需人工介入"。

### STEP 5 — 卡片框架 + 视觉质检

调用 `baoyu-image-cards` 分析文案 → 出卡片框架（张数+风格+布局+配色）。

**卡片设计铁律：每张卡是独立知识资产，不是文字的配图。** 读者刷到任何一张都应该有理由停下来——要么截屏保存，要么产生认知收益。除了封面（①），其余卡片（②-⑥）必须至少满足以下一条：

| 类型 | 判据 | 示例 |
|------|------|------|
| 对比表 | 可截屏当参考资料 | 通用英语 vs 行业英语 6维对比 |
| 数据图 | 一张图讲清一个认知 | 预算分配85% vs 15% → 应该30% vs 70% |
| 框架卡 | 可直接套用到自己公司 | 两层体系实操版（换成自己的岗位就能用） |
| 清单卡 | 可截屏即用 | 5条自查清单，打勾就能出结论 |

**禁止的卡片类型：** 纯情绪配图（HR看着电脑困惑）、纯场景插画（餐厅vs工地）、单个数字撑一张卡（"9/12"）。这些是"秀"，不是资产。

视觉质检：封面1秒测试 + 风格一致性。不合格标记建议但不阻断生成。

### STEP 6 — 图片生成

调用 `baoyu-imagine` + Seedream 5.0 逐张生成：
- 人物默认女性
- 配色：珊瑚橙(#FF6B35)+深海军蓝(#1A1A2E)+奶油白
- 扁平矢量插画，社论杂志风
- 中文文字在安全边距内

### STEP 7 — 汇总输出

- 最终文案（含评分）
- 配图文件列表
- 发布说明（顺序、每张图对应段落）
- 违禁词扫描报告
- 数据回填入口

### STEP 7.5 — 发布运营策略

内容发布不等于结束。评论区是第二流量引擎，运营好坏直接决定帖子能走多远。

#### 7.5A. 置顶评论

发布后立即在评论区置顶一条，选项（按场景选一）：

| 场景 | 置顶话术方向 |
|------|-------------|
| 引流咨询 | "需要【XX模板/清单】的姐妹扣1，我发你" |
| 建立专业感 | 补充正文没展开的一个关键细节，展示深度 |
| 冷启动互动 | 抛一个轻度问题引导讨论（"你们公司有没有这种情况？"） |

**注意：** 置顶评论不能出现导流词（微信、加好友等），用小红书站内交互承接。

#### 7.5B. 评论区运营节奏

| 时间窗口 | 动作 |
|----------|------|
| 发布后 30 分钟内 | 回复前 3 条评论，引导讨论方向 |
| 发布后 2 小时内 | 逐条回复所有评论，保持评论区活跃度 |
| 发布后 24 小时内 | 持续回复新评论，重点关注有追问意图的用户 |
| 发布 48 小时后 | 选择性回复高价值评论，不再逐条回 |

**回复原则：**
- 每条回复都是一次"微内容"——值得被其他人看到
- 有人问具体问题 → 先答再反问，把对话拉长
- 有人表达共鸣 → 追问具体场景（"你们公司多少人？什么行业？"）
- 有人反对/质疑 → 感谢 + 补充视角，不辩论
- 单纯"学习了""收藏了" → 回复一句轻量互动即可

#### 7.5C. 私信承接（合规）

当评论中出现明确咨询意图时，引导至私信。**私信内同样遵守违禁词规则，不发微信号/手机号。**

首条私信模板：
> 感谢关注～关于【话题】，你可以先描述一下你的具体情况，我帮你看看从哪里入手比较好。

先了解需求再推进，不直接发联系方式。建立信任后再用站内工具引导后续沟通。
