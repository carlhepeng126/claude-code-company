# 小红书内容生产流水线

一站式串联：选题→热度预检→情绪分析→文案→质检→卡片→配图→输出。触发后全流程自动跑，质检不通过自动打回重写。

## 素材源优先级

按顺序扫描，同源内优先取日期最近的文件：

1. `ACT笔记-深度思考/` — 项目方案、客户纪要、行动卡、B卡
2. `收件箱：同步助手/` — 每日行业动态、操作日志
3. `收件箱：需求模拟/` — 每日客户分析报告
4. `LLM Wiki知识应用/` — 知识实体、概念、报告

**日常模式**：用户未指定素材时，扫近期文件出3个选题让用户选。**指定模式**：用户给出了主题或文件路径，直接进流水线。

## 流水线

### STEP 0 — 素材→选题提取

从素材中抓取爆点角度：
- 最反常识的数据点
- 最 relatable 的人物处境
- 最出人意料的结果

日常模式输出3个选题（含标题方向+素材来源），等待用户选择。

### STEP 1 — 选题热度预检

用 `web-access` 搜小红书同类话题近期高赞帖，判断：
- 竞争热度（是否已卷烂）
- 差异化角度建议
- 如选题偏冷，给切入建议

### STEP 2 — 情绪触发点分析

调用 `marketing-psychology`，分析目标受众情绪杠杆，输出3个情绪钩子按冲击力排序。

### STEP 3 — 文案初稿

调用 `copywriting` + `social`，产出千字以内小红书文案。遵循：
- 情绪钩子开头，不要直接给数据
- 客户原话制造共鸣
- 人设一句带过
- 三步解法各≤2行
- 效果闭环必须
- 金句收尾
- 最后一张必是清单卡
- 脱敏：不出现客户真实名称、行业细节、上市代码

### STEP 4 — 质检关 ⚠️

调用 `marketing-reviewer` Agent 评分。

评分模型：Q1看(50%) + Q2赞(30%) + Q3留(20%)

| 总分 | 动作 |
|------|------|
| ≥ 85 | 放行 |
| 80-84 | 放行，微调建议 |
| 70-79 | 打回 STEP 3 重写，附失分原因 |
| < 70 | 打回 STEP 3 重写，附改写范例 |

打回后最多重写**2次**，仍不通过则保留当前最佳版并标注"需人工介入"。

### STEP 5 — 卡片框架 + 视觉质检

调用 `baoyu-image-cards` 分析文案 → 出卡片框架（张数+风格+布局+配色）。

视觉质检：封面1秒测试 + 风格一致性。不合格标记建议但不阻断生成。

### STEP 6 — 图片生成

调用 `baoyu-imagine` + Seedream 5.0 逐张生成：
- 人物默认女性
- 配色：珊瑚橙(#FF6B35)+深海军蓝(#1A1A2E)+奶油白
- 扁平矢量插画，社论杂志风
- 中文文字在安全边距内

### STEP 7 — 汇总输出

- 最终文案（含评分）
- 配图文件列表
- 发布说明（顺序、每张图对应段落）
- 预留数据回填入口
