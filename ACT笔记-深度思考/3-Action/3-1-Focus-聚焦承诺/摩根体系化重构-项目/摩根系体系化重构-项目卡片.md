# 摩根系体系化重构 — 项目卡片

> 从"手工方案作坊"转型为"产品体系驱动"的知识服务商。
> AI 深度融入每个环节，弥补团队能力与人效缺口。
> 项目别名：「摩根世纪体系工作坊」

## 项目背景

当前困局：
- 一客户一方案，Carl 时间被方案设计吃掉 80%
- 产品点状，无体系，线上平台与竞品差距大
- 团队3人能力缺口大，工具应用/系统化能力弱
- 业务下滑，成本压力，团队幸福指数低

核心矛盾：Carl的能力无法被复制和规模化。

## 关键成果（完成标准）

- [ ] **成果1**：完成五大体系（产品/市场/教学/团队/交付）的深度拆解与打法定义
- [ ] **成果2**：AI 融入方案落地（至少 3 个提效场景可用）
- [ ] **成果3**：方案自动化达到"新需求→AI 出骨架→Carl 30分钟调装配"
- [ ] **成果4**：团队三张清单（能力/成长/卡点）建立，每人有清晰定位
- [ ] **成果5**：工作坊共创结果产出 + 具体实施举措文档
