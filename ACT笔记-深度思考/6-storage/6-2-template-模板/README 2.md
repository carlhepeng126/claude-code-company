# 如何使用模板

Obsidian **不会自动识别模板文件夹**，需要手动配置一次（只要配一次，永久的）。

## 配置步骤

1. **打开设置** → `Ctrl/Cmd + ,`
2. **左边栏** → 核心插件 → 找到 **模板** → 点击启用
3. **启用后** → 点击模板右边的 ⚙️ 图标
4. **模板文件夹路径** → 填入：`40-storage/42-template-模板/`
5. **日期格式**（可选）→ 填入：`YYYY-MM-DD`
6. **时间格式**（可选）→ 填入：`HH:mm`
7. 关闭设置

## 使用模板

配置好后，在任意笔记中：
- `Ctrl/Cmd + P` → 输入"模板" → 选"插入模板" → 选要用的模板
- 或者设置快捷键 `Alt+T` 一键插入

## 已有模板列表

| 模板文件 | 用途 | 适用场景 |
|---------|------|---------|
| `日计划模板` | 今日待办 + 流水 | 每天开始工作时新建 |
| `周计划模板` | 本周目标 + 项目进度 | 每周一早晨 |
| `月度计划模板` | 月度目标 + 关联项目 | 每月第一周 |
| `月度回顾模板` | 三层联动回顾 | 每月末 |
| `任务笔记模板` | 单个任务/行动 | 确定要做的任务 |
| `项目卡片模板` | 完整项目 | 新项目启动时 |
| `活跃跟进模板` | 潜在项目跟进 | 持续沟通中但未立项 |
| `将来也许模板` | 想法储备 | 暂时不做但不想忘 |
| `快速捕捉模板` | 碎片想法快速记录 | 任何闪念 |
| `核心卡模板` | 关键知识点/概念 | 深度理解一个概念 |
| `阅读卡模板` | 读书/文章笔记 | 读完一本书/文章 |
| `知识卡片模板` | 常规知识点 | 记录碎片知识 |

## 最佳实践

- 新建日计划时，先插入模板再写内容
- 快速捕捉直接用收件箱文件夹 + 快速捕捉模板
- 任务笔记关联到项目：在 frontmatter 的 `project` 字段填项目名
- 核心卡关联行动：在 `trigger_action` 字段注明引发了什么行动
