## 二、整体思路

### 2.1 设计理念：场景即课程

本方案不做"词汇表式"的英语培训。**以中海油服海外钻井项目的真实作业场景为课程主线**，把学员每天在井场遇到的工作情境搬进教室：从班前会到交接班、从钻进指令到井控应急、从设备故障沟通到甲方监督汇报。

### 2.2 四层实战递进

```
┌─────────────────────────────────────────────────────────────┐
│                    四层实战递进模型                            │
│                                                             │
│   Layer 1        Layer 2        Layer 3        Layer 4      │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐    │
│  │ 术语激活  │→│ 指令听说  │→│ 场景对话  │→│ 综合模拟  │    │
│  │          │  │          │  │          │  │          │    │
│  │ 唤醒已有  │  │ 听懂指令  │  │ 场景化   │  │ 全流程   │    │
│  │ 词汇储备  │  │ 准确回应  │  │ 角色扮演  │  │ 压力测试  │    │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘    │
│                                                             │
│  输入为主         输入+输出       输出为主         综合输出    │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 课程设计原则

1. **场景聚焦** — 只教钻井现场用得上的，不教"教科书英语"
2. **指令驱动** — 以外方监督对司钻/带班的真实指令为训练主线
3. **安全前置** — 所有场景中安全沟通（PTW/JSA/Toolbox Talk）贯穿始终
4. **开口即练** — 每个知识点后立即角色扮演，不讲"回去再练"
5. **每日通关** — 每天末尾设置核心句复述+抽查，当堂验收不过夜
6. **工具带走** — 每人带走《钻井现场英语速查手册》（中英对照+缩写+常用指令模板）

## 三、课程安排

### Day 1：核心作业场景（6小时）

| 时段          | 模块      | 主题      | 内容要点                                                                                                                                                                 | 实战形式                        |
| ----------- | ------- | ------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------- |
| 09:00-10:30 | **模块一** | 井场人与物+钻前检查 | ① **带甲方监督首次巡视井场**：从人员介绍到设备指认的完整英文导览对话（Driller/Toolpusher/Floorman / Derrick/Draworks/Top Drive/BOP）；② **开钻前设备检查确认（Pre-spud Check）**：司钻向平台经理逐项确认——"All clear / System normal / Pump ready / Circulation good"；③ **20in导管&表层钻进准备沟通**：工具确认、指令对齐——"Drill 24in hole / Run 20in conductor / Start pump / Slow down"；④ **工具借还与紧急报修对话**：从"Pass me the pipe wrench"到"The tong is jammed, need maintenance"的完整沟通链 | 设备图片速认 → 钻前检查角色对练（司钻vs平台经理）→ 工具借还情景对话 |
| 10:45-12:00 | **模块二** | 钻进作业指令  | ① 钻进参数英文对话（WOB/RPM/ROP/Flow Rate/Pump Pressure）——监督下指令 → 司钻复述确认 → 执行后回报；② 指令变更的完整对话链："Make up BHA" / "Circulate bottoms up" / "Stop drilling at 2000m" 及司钻的标准应答；③ **钻进异常主动汇报**：钻头磨损/泵压异常/钻速变慢——不仅是"报告问题"更是"描述现象+提出推测+请求指令"的三段式沟通 | 指令听写 → 司钻-监督角色对练（含指令变更压力测试）→ 异常汇报三段式练习 |
| 13:30-15:00 | **模块三** | 起下钻与接单根 | ① 起下钻标准指令序列（POOH/RIH/Make a Connection/Break Out/Set Slips）——司钻边操作边英文回报；② 管柱处理英文（Pick up stands/Lay down DP/Set back BHA）——与井架工、场地工的英文配合喊话；③ 灌浆与循环指令（Fill up every 5 stands/Circulate hole clean）；④ **监督催促下的应对**："Speed up, we're behind schedule"——如何用英文解释当前节奏、预估完成时间 | 指令序列排序 → 操作+英文同步模拟 → 速度压力练习（加入监督催促角色） |
| 15:15-16:30 | **模块四** | 下套管与固井  | ① 套管作业英文（Run Casing/Casing Size/Casing Shoe/Casing Centralizer）；② 固井流程英文（Cementing Pump/Cementing Head/Squeeze Cement/WOC）；③ 与固井分包商协作沟通——"How's the slurry mix?" "Need 10 more minutes" "Pressure holding steady"；④ **固井后压力测试确认**："Pressure test OK / No leaks / Job done" ——与分包商逐项签字确认的英文对话 | 流程图英文标注 → 双组协作对话（钻井队vs固井队）→ **每日核心句通关复述（抽查）** |

### Day 2：安全、汇报与综合实战（6小时）

| 时段 | 模块 | 主题 | 内容要点 | 实战形式 |
|------|------|------|---------|---------|
| 09:00-10:30 | **模块五** | 井控与应急 | ① 井控核心概念英文（Well Kick/Blowout/Well Killing/Hard Shut-in）——不只是记词，是在紧急状态下用最短英文句子传递最关键信息；② H2S警报与应急响应英文——听到警报后30秒内完成英文广播；③ 三种演习英文广播（Abandon Drill/Fire Drill/BOP Drill）——每人上台实操广播；④ **向甲方电话通报紧急情况**：3句话模板——"This is COSL rig. We have [situation]. Actions taken: [措施]." | 应急广播听写 → 紧急电话汇报压力测试（限时30秒）→ 应急场景角色模拟 |
| 10:45-12:00 | **模块六** | 复杂情况沟通 | ① 卡钻类型与英文描述（Differential Sticking/Mechanical Sticking/Balling Sticking）——"What's stuck? Where? How bad?" 三问即答；② 井漏/井斜英文报告（Lost Circulation/Hole Deviation/Tight Hole）——描述现象+初步判断+请求甲方指令的对话模板；③ **向甲方监督写英文短信汇报异常**：50词以内，现象→原因→措施→所需支持，四段式短信模板；④ **电话请示处置方案**："We recommend [方案A]. Alternative is [方案B]. Which one do you approve?" | 案例阅读 → 英文短信限时写作（10分钟）→ 监督电话汇报压力测试 |
| 13:30-15:00 | **模块七** | 安全沟通与日常汇报 | ① 班前安全会英文主持（Toolbox Talk）；② **吊装作业英文口令**：Sling/Shackle/Lift slowly/Stand clear/All secure——指挥吊车司机的标准喊话与手语配合；③ PTW/JSA英文讨论；④ 钻井日报（DDR）英文填写要点；⑤ 交接班英文沟通（Shift Handover）——上一班对下一班的口头+书面英文交接 | TBT主持练习 → 吊装口令指挥练习 → DDR填写 → 交接班角色对练 |
| 15:15-16:30 | **模块八** | 两日通关+综合实战 | **前半段（30min）——两日核心句通关复述**：Day 1 + Day 2 核心场景句抽查，每人随机抽5句，不过关的现场补练；**后半段（1h）——全流程压力模拟**：学员分组，每组分配角色（司钻、带班、监督、甲方代表）。模拟一个完整班次：① 班前TBT → ② 接收甲方当日作业指令 → ③ 执行钻进/起下钻 → ④ 突发井涌应急 → ⑤ 向甲方汇报 → ⑥ 填写DDR交接班 | 核心句通关抽查 → 分组综合演练（每组25分钟）+ Ben Phone逐组点评反馈 |
