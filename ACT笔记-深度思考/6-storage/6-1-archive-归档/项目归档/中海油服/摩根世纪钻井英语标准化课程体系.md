# 摩根世纪钻井英语标准化课程体系

## 一、体系定位

### 1.1 产品定位

本课程体系是摩根世纪**行业英语产品线**的核心模块之一，定位为"可配置、可复制、可迭代"的标准化培训产品。面向中国油气钻井行业出海企业，解决海外钻井项目一线人员"听得懂、说得出、写得对"的现场英语沟通能力问题。

### 1.2 设计哲学：核心60% + 可变40%

```
┌───────────────────────────────────────────────────────────┐
│                    标准化课程架构                           │
│                                                           │
│   ┌─────────────────────────────────────────────────┐    │
│   │              固定框架 60%                          │    │
│   │  钻井通用场景 · 标准指令体系 · 安全沟通 · 井控术语    │    │
│   └─────────────────────────────────────────────────┘    │
│                         +                                  │
│   ┌─────────────────────────────────────────────────┐    │
│   │              可变内容 40%                          │    │
│   │  客户特定区域 · 甲方公司文化 · 项目类型 · 设备品牌    │    │
│   └─────────────────────────────────────────────────┘    │
```

类比 B016 区域可替换框架：固定部分覆盖钻井行业通用的场景和术语体系，可变部分针对不同客户（中海油服/中石油钻井/中石化工程等）的特定市场区域、甲方类型、设备配置进行适配。

### 1.3 目标客户群

| 客户类型 | 典型场景 | 核心需求 |
|---------|---------|---------|
| 油田服务公司（如中海油服） | 海外钻井项目投标与执行 | 司钻/带班与甲方监督沟通 |
| 国有钻井公司（如中石油钻井） | 海外区块作业 | 全井队英文作业能力 |
| 钻井设备制造商 | 海外设备安装调试培训 | 设备英文技术沟通 |
| 钻井项目承包商 | 国际项目人员外派 | 面试准备+现场生存英语 |

## 二、能力分级模型

采用**三级能力进阶**，覆盖钻井人员英语能力从基础到精通的全路径：

```
Level 1                Level 2                Level 3
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  基础生存层    │     │  岗位操作层    │     │  管理沟通层    │
│              │     │              │     │              │
│  听懂日常指令  │  →  │  独立执行岗位  │  →  │  主持安全会议  │
│  指认设备工具  │     │  英文沟通      │     │  英文撰写报告  │
│  简单安全用语  │     │  应对常见异常  │     │  与甲方深度沟通 │
│              │     │              │     │  带队应对检查  │
└──────────────┘     └──────────────┘     └──────────────┘
     ↓                    ↓                    ↓
  钻工/场地工           司钻/副司钻          带班队长/平台经理
  新入职海外人员         有经验的一线操作      项目管理与监督层
```

| 层级 | 对象 | 词汇量目标 | 核心能力 | 建议课时 |
|:----:|------|:--------:|---------|:------:|
| L1 | 钻工、场地工、新入职人员 | 300-500词 | 听懂指令、指认设备、简单安全沟通 | 3天（18h） |
| L2 | 司钻、副司钻、钻井工程师 | 500-800词 | 执行英文操作指令、异常汇报、填写DDR | 2天（12h） |
| L3 | 带班队长、平台经理、项目经理 | 800-1200词 | 主持TBT、撰写英文报告、与甲方深度沟通 | 2天（12h） |

## 四、各模块详细设计

### M1 井场人与物（Personnel & Equipment）

| 项目 | 内容 |
|------|------|
| **目标层级** | L1/L2 |
| **标准课时** | 1.5h |
| **核心词汇** | 岗位名称15个、设备名称40个、工具名称25个 |
| **关键场景** | ① **带新来甲方监督巡视井场**：从人员介绍到设备指认的完整英文导览对话；② **开钻前设备检查确认（Pre-spud Check）**：司钻向平台经理逐项确认——"All clear / System normal / Pump ready / Circulation good"；③ **20in导管&表层钻进准备沟通**："Drill 24in hole / Run 20in conductor / Start pump / Slow down"；④ 工具借还与紧急报修——从"Pass me the pipe wrench"到"The tong is jammed, need maintenance"的完整沟通链 |
| **实战形式** | 设备图速认 → 钻前检查角色对练（司钻vs平台经理）→ 工具借还情景对话 → "找设备"团队竞赛 |

### M2 钻进作业英文指令（Drilling Operations）

| 项目 | 内容 |
|------|------|
| **目标层级** | L2/L3 |
| **标准课时** | 1.5h |
| **核心词汇** | 钻进参数12项、指令动词25个、工况缩写15个 |
| **关键场景** | ① 钻进参数英文对话——监督下指令 → 司钻复述确认 → 执行后回报；② 指令变更的完整对话链："Make up BHA" / "Circulate bottoms up" / "Stop drilling at 2000m" 及司钻标准应答；③ **钻进异常主动汇报**："描述现象+提出推测+请求指令"三段式沟通（钻头磨损/泵压异常/钻速变慢/水眼堵塞）；④ 取芯作业英文 |
| **实战形式** | 指令听写 → 司钻-监督角色对练（含指令变更压力测试）→ 异常汇报三段式练习 |

### M3 起下钻与接单根（Tripping & Connections）

| 项目 | 内容 |
|------|------|
| **目标层级** | L2/L3 |
| **标准课时** | 1.5h |
| **核心词汇** | 起下钻指令20个、管柱术语15个、井口工具10个 |
| **关键场景** | ① 标准起下钻指令序列（POOH/RIH/Make a Connection/Break Out/Set Slips）——边操作边英文回报；② 接单根全流程英文——与井架工、场地工英文配合喊话；③ 甩钻杆/甩单根英文 + 通井/划眼英文指令；④ **监督催促下的应对**："Speed up, we're behind schedule"——如何用英文解释节奏、预估完成时间 |
| **实战形式** | 指令序列排序 → 操作+英文同步模拟 → 速度压力练习（加入监督催促角色） |

### M4 下套管与固井英文（Casing & Cementing）

| 项目 | 内容 |
|------|------|
| **目标层级** | L2/L3 |
| **标准课时** | 1.5h（可压缩至1h） |
| **核心词汇** | 套管术语20个、固井术语15个、附件名称15个 |
| **关键场景** | ① 下套管流程英文；② 固井作业英文指令；③ 与固井分包商协作沟通——"How's the slurry mix?" "Need 10 more minutes" "Pressure holding steady"；④ **固井后压力测试确认**："Pressure test OK / No leaks / Job done"——与分包商逐项确认的英文对话；⑤ 候凝期间沟通 |
| **实战形式** | 流程图英文标注 → 双组协作对话（钻井队vs固井队） |

### M5 井控与应急英文（Well Control & Emergency）

| 项目 | 内容 |
|------|------|
| **目标层级** | L1基础篇（1h）/ L2-L3全篇（1.5h） |
| **标准课时** | 1.5h |
| **核心词汇** | 井控术语25个、应急广播15个、警报类型8个 |
| **关键场景** | ① 井涌/井喷识别与报告——紧急状态下用最短英文句子传递关键信息；② H2S警报响应英文——听到警报后30秒内完成英文广播；③ 三种演习英文广播（Fire/BOP/Abandon Drill）——每人上台实操广播；④ **向甲方电话通报紧急情况**：3句话模板——"This is [Company] rig. We have [situation]. Actions taken: [措施]." |
| **实战形式** | 应急广播听写 → 紧急电话汇报压力测试（限时30秒）→ 应急场景角色模拟 |

### M6 复杂情况英文沟通（Problem Handling）

| 项目 | 内容 |
|------|------|
| **目标层级** | L2（选）/ L3（必） |
| **标准课时** | 1.5h |
| **核心词汇** | 卡钻类型6种、井漏/井斜术语、处置方案英文 |
| **关键场景** | ① 卡钻类型与英文描述（Differential/Mechanical/Balling Sticking）——"What's stuck? Where? How bad?" 三问即答；② 井漏/井斜英文报告——描述现象+初步判断+请求甲方指令的对话模板；③ **向甲方写英文短信汇报异常**：50词以内，现象→原因→措施→所需支持，四段式模板；④ **电话请示处置方案**："We recommend [方案A]. Alternative is [方案B]. Which one do you approve?" |
| **实战形式** | 案例阅读 → 英文短信限时写作 → 监督电话汇报压力测试 |

### M7 安全沟通体系（Safety Communication）

| 项目 | 内容 |
|------|------|
| **目标层级** | L1/L2/L3（三层递进） |
| **标准课时** | 1.5h |
| **核心词汇** | HSE术语30个、PTW/JSA术语20个、PPE术语15个 |
| **关键场景** | ① 主持英文Toolbox Talk；② **吊装作业英文口令**：Sling/Shackle/Lift slowly/Stand clear/All secure——指挥吊车司机的标准喊话与手语配合；③ PTW（工作许可）英文讨论；④ JSA（工作安全分析）英文表述；⑤ 红区管理与安全提示；⑥ L1层级侧重"听懂安全指令"，L2侧重"执行安全沟通"，L3侧重"主持安全会议" |
| **实战形式** | TBT主持练习 → 吊装口令指挥练习 → PTW批准对话 → 险兆事件(Miss)英文汇报 |

### M8 日常汇报与邮件（Daily Reporting & Email）

| 项目 | 内容 |
|------|------|
| **目标层级** | L2（选）/ L3（必） |
| **标准课时** | 1.5h |
| **核心场景** | ① DDR（钻井日报）英文填写要点；② 英文邮件汇报作业进展；③ 交接班英文沟通（Shift Handover）；④ 与甲方驻井监督日常交流 |
| **实战形式** | DDR填写练习 → 邮件模板写作 → 交接班角色对练 |

### M9 面试英语专项（Interview Preparation）

| 项目 | 内容 |

| **目标层级** | L1/L2 |

| **核心场景** | ① 自我介绍（按岗位模板）；② 技术问题英文回答；③ 安全理念英文表述；④ 行为面试英文应对（STAR法则） |
| **实战形式** | 模板填空 → 一对一模拟面试 → 面试官反馈 |

### M10 技术演讲与汇报（Technical Presentation）

| 项目 | 内容 |

### M11 合同与商务英语（Contract & Business English）

| **实战形式** | 合同条款阅读 → 变更请求英文对话 → 与"甲方"价格沟通模拟 |

### M12 综合实战模拟（Integrated Simulation）

## 五、师资配置建议

### 5.1 讲师配置标准

| 课程类型 | 主讲要求 | 助教配置 |
|---------|---------|:------:|
| 三日基础营（L1） | 行业英语讲师 + 1名钻井技术顾问 | 1名 |

| 两日提升营（L3） | 资深行业英语专家 + 有过海外钻井管理经验 | 1名 |

### 5.2 主讲讲师池

| 讲师 | 定位 | 擅长领域 |
|------|------|---------|
| **Ben Phone** | 首席行业英语专家 | 钻井/跨文化/演讲谈判 |
| 待扩充 | 行业英语讲师 | 油气行业场景英语 |
| 待扩充 | 钻井技术顾问（配合） | 提供技术场景把关 |

### 5.3 讲师培养路径

依托标准化课程体系，后续可培养2-3名经过认证的钻井英语讲师，形成交付梯队：

1. **观摩期**：跟课2次，做助教
2. **辅助期**：独立承担1-2个模块教学
3. **独立期**：全程独立交付，首席讲师远程督导

### 6.2 定制化适配清单

在标准化模块基础上，针对每个客户完成以下适配项：

| 适配维度 | 内容 | 示例 |
|---------|------|------|
| **区域市场** | 客户项目所在国的甲方文化、英语口音特点 | 中东项目：沙特阿美甲方沟通风格；非洲项目：法语区英语特点 |
| **甲方类型** | 客户主要服务的甲方公司文化 | Shell/BP/Chevron不同监督风格 |
| **设备品牌** | 客户使用的主要设备品牌英文 | NOV顶驱 vs Varco顶驱 |
| **项目类型** | 日费/大包/合资等不同模式 | 大包井 vs 日费井的沟通差别 |
| **人员现状** | 学员现有英语基础摸底 | 词汇量前测 + 口语能力评估 |
| **紧急场景** | 客户近期遇到或担心的特定场景 | 某项目近期频繁卡钻，则M6加强卡钻场景 |

### 6.3 质量保障

| 保障环节 | 措施 |
|---------|------|
| **课前** | 学员英语摸底 + 岗位场景调研 → 模块精准配置 |
| **课中** | 每模块后学员自评（1-5分）→ 当日调整教学节奏 |
| **课后** | 学员满意度问卷 + 岗位英语应用跟踪（训后1月回访） |
| **迭代** | 每次交付后更新场景案例库，保持案例时效性 |
