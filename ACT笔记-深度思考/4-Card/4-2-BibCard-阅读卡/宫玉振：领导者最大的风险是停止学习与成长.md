# 阅读内容

## ① 阅读目的
提升自我领导能力

## ② 核心观点

来源：宫玉振译《领导的品格》（彼得·舒梅克等著，2026年3月出版）

**领导者的三项核心职责：实施领导、发展组织、完成使命。**

### 一、实施领导（四项能力）

1. **引领他人**：提供清晰目标（员工应能一句话说清当前最重要目标）、提供持久动机（来自认同而非KPI）、强化执行标准（什么是合格、什么不可越界）。
2. **横向领导力**：在指挥链之外扩展影响力。矩阵组织中最大的痛点是"没有直接权力如何推动业务"——只在有权时才会领导的领导者不是真正的领导者。
3. **以身作则**：下属听你怎么说，更看你怎么做。一把手若成为企业规则的最大破坏者，就是在用行为摧毁自己口头上宣讲的价值观。
4. **沟通交流**：管理就是沟通、沟通、沟通。三个关键环节——倾听（了解真实情况）→ 陈述（清晰传递目标）→ 确认（确保共识形成）。

### 二、发展组织（三项能力）

1. **营造积极环境**：领导者是组织环境的第一责任人。积极环境的特质：公正包容、坦诚沟通、终身学习、尊重关怀。报喜不报忧、双标不公、办公室政治等，最终导致竞争力衰退。
2. **自我准备**：面向任务持续进行能力提升与自我更新。包含认知准备（稳定决策框架+复盘机制）、状态准备（精力情绪管理）、能力准备（跨界视角+品格风度才智提升）。
   > ⚠️ **核心命题：领导者最大的风险是停止学习与成长。** 各种事务吞噬时间，成功经验固化路径依赖，学习退化为形式培训，导致用数年前的经验解决当下的问题，能力与需求缺口越来越大。
3. **培养他人**：培养他人是领导者的直接责任，不是HR的事。三个原则——把人才培养写进岗位职责与考核、提供有挑战的任务+反馈+成长路径、用面向未来的标准培养人才（今天的领导要应对明天的挑战）。

### 三、完成使命（三项能力）

1. **提供方向、指导，明确优先次序**：方向解决"去哪里"，指导解决"理解一致"，优先次序解决"什么最重要"。
2. **制订与执行计划**：将模糊的"做什么"转化为"什么时间、什么地点、什么人完成什么任务"。好的计划提供路径、资源配置、执行节奏、评估标准、风险预案。
3. **运用监控纠偏与改进**：识别风险及时纠偏、发现优点进行激励、形成评估→反思→调整→优化的闭环。

> 灵魂三问：你的组织环境是否健康？你自己是否在不断升级？你的下属是否在不断进步？


## ③ 我的思考

今天（2026-05-13）围绕这个阅读内容，结合自己的实际处境（中信建设考试直播项目、Andy的培养、公司最忙的人），做了一次深度的自我对照，输出了四张核心卡：

### 对照一：引领他人 → 我的真实死锁

宫玉振说「引领他人」是领导者的核心能力，但我的现实是：**中信建设考试直播Andy完全可以做，但培训他比我自己做更花时间。** 所以我每次都选了自己做。结果就是：不培养人→自己最忙→更没时间培养人→永远没合适的人可用。

这不是能力问题，是**投入产出曲线的选择问题**。自己做每次3小时，没有复利。带人做第一次你花2小时，第三次只花20分钟。我每次都选第一次，却期待得到第三次的结果。

→ [[B018-技术型专家的时间陷阱]]

### 对照二：沟通交流 → 带人的操作系统

宫玉振讲「沟通三环：倾听→陈述→确认」，我在实际中三个环都有断裂：
- **倾听**断在「我以为大家知道」——不倾听就不知道对方真实状态和盲区
- **陈述**断在「目标不确定」——小公司跟着市场走，领导自己的目标也在变。破局是用「过程确定性」替代「结果确定性」，给框架而非给答案
- **确认**断在「从不复述」——讲完觉得自己讲了，对方觉得听了，两边都没确认

这三环是B018落地的操作系统。没有三环，交出去的任务理解偏了，做出来不对，收回来改，比自己做更花时间——不是带人行不通，是跳过了沟通三环。

→ [[B019-引领他人的操作系统-倾听陈述确认]]

### 对照三：自我准备 → 领导者最大的风险

宫玉振说的「领导者最大的风险是停止学习与成长」对我触动最深。我现在大部分时间在算账——花费多少、人效如何——困在存量管理里，没有战略思考。

这和B018是同构的死锁：向内抠成本→没时间向外学→没有战略视野→只能向内卷→越卷越穷→更多算账。破局就是**走出去**：接触客户、接触同行、接触更优秀的人，跳出自我，空杯心态。没有空杯心态，走出去只是换个地方内耗。

→ [[B020-领导者的学习陷阱-向内算账vs向外学战略]]

### 对照四：提供方向、指导、优先次序 → 层次关系

宫玉振说的「方向→指导→优先次序」我今天的体悟是：**方向是最重要的，优先次序是下游能力。** 员工能力没成长、辅导没到位，给了优先级他们也判断不了轻重缓急。所以领导者的重心顺序是：先把方向定清楚→再通过辅导让团队「知道」→等能力成长了，优先级才成为可传递的判断力。

→ [[B021-领导者的核心产出-方向指导优先次序]]

### 四卡形成的链路

```
B020（领导者向外学习：走出去+空杯心态）
  → B021（产出方向→指导→优先次序，按层次来）
    → B019（沟通三环：倾听→陈述→确认，传递下去）
      → B018（轻试行+带人，团队接住）
        → 领导者被释放 → 回到B020
```
