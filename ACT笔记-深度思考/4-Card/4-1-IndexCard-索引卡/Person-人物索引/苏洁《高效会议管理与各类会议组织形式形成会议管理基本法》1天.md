# 苏洁《高效会议管理与各类会议组织形式形成会议管理基本法》1天

**高效会议管理基本法**

**培训背景**

随着社会的发展，竞争越来越激烈，跨部门开会已经成为取得成功的关键，越来越多的企业开始重视跨部门协作。跨部门会议管理是企业成功发展的基础，不仅可以提升企业整体竞争力，还可以提高企业效能，从而实现更大的利润增长。

会议的主要目的就是充分整合参会人的智力资源以达成共识和解决问题，但由于开会技巧不佳或过于频繁，不但无益于解决问题，反而使问题愈趋复杂。员工花太多时间在无效率的会议上，不仅浪费公司成本，也造成工作效率低下。如何能在会议中高效率且有效果地解决问题，主持人如何合理地设计会议流程并游刃有余的控制住场面，以及处理各种突发事件，如何做好会议管理，提升效率与产出，是每位管理者的必修课。

**课程收益**

掌握会议管理的八个要点，学会实时的控场和有效引导技巧；

掌握会前准备、会中服务及会后执行的内容，大幅提升会议效率；

团队行动学习，进行研讨竞赛、形成公司会议管理基本法，落地实施，共创佳绩。

**三、课程时长：**1天，6小时/天

**四、授课形式**

本课程运用角色演练、案例研讨、问题分析、团队学习等方式，让受众在学习方法论的同时，能够与日常管理实践紧密结合，解决工作中的问题。

本课程通过场景设计与管理工具相结合的方式，使受众进行场景化学习，在实际工作现场下可以快速运用工具，可复制管理。

**各类型会议如何组织召开**

**定期部门****会议：**

一、会前准备

1. 确定会议目的：回顾上周工作进展、解决问题、明确本周任务。

2. 确定会议时间和地点：选择小组成员都方便的时间，地点

3. 通知参会人员：提前发送会议通知，明确会议主题、时间、地点和议程

4. 收集相关资料：上周工作总结、问题反馈、重要数据

二、会议进行

1. 开场：主持人简要说明会议目的和议程

2. 上周工作总结：小组成员依次汇报上周工作完成情况、遇到的问题及解决方案

3. 问题讨论：针对上周出现的问题进行深入讨论，集思广益寻求最佳解决方案

4. 本周任务安排：明确本周工作重点和目标，分配具体任务给小组成员

5. 其他事项：如团队协作需求、资源需求

三、会议结束

1. 总结发言：主持人对会议进行总结，强调重点事项和行动方向

2. 整理会议记录：记录员整理会议内容，包括工作进展、问题及解决方案、本周任务

3. 发送会议记录：将会议记录发送给小组成员，确保大家清楚了解会议决策和任务安排

**复盘会议****：**

一、会前准备

1. 明确复盘目的

2. 确定参会人员

3. 选择会议时间和地点

4. 收集资料

5. 发送会议通知

二、会议开场

1. 主持人开场

2. 回顾目标

三、回顾过程

1. 按时间顺序或工作流程回顾事件过程

2. 展示相关资料和数据

四、评估结果

1. 对比实际结果与目标

2. 分析关键事件和决策的影响

五、分析原因

1. 成功原因分析

2. 失败原因分析

六、总结经验教训

2. 制定改进措施

七、会议结束

2. 确定责任人与时间节点

3. 整理会议记录


**创意****会议****：**

一、会前准备

1. 明确会议目的

2. 确定参会人员

3. 选择会议时间和地点：地点可以选择较为轻松、舒适且能激发灵感的环境

4. 发送邀请和准备材料

二、会议开场

1. 主持人开场

2. 暖场活动：进行一些简单的破冰游戏或创意小练习，帮助大家放松心情，进入创意状态

三、创意激发阶段

1. 头脑风暴

2. 创意工具运用：使用创意工具如思维导图、六顶思考帽等，帮助大家从不同角度思考问题，拓展思维广度和深度

3. 案例分享和启发

四、创意筛选阶段

1. 整理想法

2. 初步筛选

3. 深入讨论

五、会议结束

1. 总结成果

2. 确定责任人


**Work Shop****会议****：**

一、会前准备

1. 明确主题和目标：确定 Workshop 的具体主题，例如解决特定问题、进行团队建设、学习新技能

2. 确定参会人员

3. 选择会议时间和地点：会议地点应具备良好的环境和设施，以支持各种活动的开展。


4. 准备材料和工具

5. 发送邀请和议程

二、会议开场

1. 主持人开场

2. 团队建设活动：可以进行一些简单的团队建设活动，帮助大家熟悉彼此，营造良好的合作氛围

三、知识分享和问题阐述

1. 主题介绍

2. 问题阐述

四、小组活动和讨论

1. 分组

2. 小组任务

3. 小组讨论

4. 小组汇报

五、总结和反馈

1. 总结成果

2. 反馈和评价
